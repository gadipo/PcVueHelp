Using the Log Viewer's Filter

The filter for a Log Viewer may be changed by the User if this option was enabled when the display was configured. A User may change the Log List to which the display is attached, select the Domain or Nature and select the type of log data to be displayed. Many of these options are also available from the toolbar. Show picture



Changing the log list at run-time is only a temporary operation, when the window containing the Log Viewer is closed, the changes to the filter are lost.

The log list is displayed from the small command button under the scroll bar.
