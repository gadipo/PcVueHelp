Using the Map Control at Run-time

When a mimic containing a Map Control is opened at run-time, the map appears centered at the configured location and zoomed at the configured zoom level. The center of the map is indicated with a small red cross. Show Picture

The Map Control supports the following run-time actions.

  • For the map itself
    • Click and drag - Move the map by changing the map center within the panning bounds.
    • Mouse wheel - Zoom the map within the configured zoom bounds.
    • Zoom control (lower right corner) - Click the + and - tools, or enter a number in the text box, to zoom the map within the configured zoom bounds.
    • Keyboard shortcuts
      • Ctrl and + - Zoom in.
      • Ctrl and - - Zoom out.
      • Ctrl and 0 - Zoom out to the minimum allowable zoom level.
  • For a map marker
    • Mouse over - Displays the marker's tooltip (if configured).
    • Click - Opens the mimic (window) associated with the marker (if configured).
    • Right click - Opens the User Data pop-up associated with the marker (if configured).

Why are some of the map tiles not displayed?

If the Map Control was configured so that it allows zooming and/or panning for which map tiles do not exist, or the connection to the map tiles location has been lost, it is possible that part or all of the map will not be displayed. Instead, the unavailable map tiles are replaced by blanks.
