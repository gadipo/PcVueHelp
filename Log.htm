How to Add a Log Page

Each Log Page can include data from one Log List. A single Log Page can generate two outputs.

  • Data output - Basic table of the logged data as extracted from a Log List in an archive unit.
  • Statistics output - Statistical information about the extracted log data.
  1. Open the Application Explorer and expand the configuration tree to select a previously configured Data Export.
  2. From the task list, select Add a log page. The Log Page Creation dialog opens. Show picture
  3. Configure the main properties.
    1. Enter the Name for the log page. The name is the unique identifier of the log page within PcVue and can also be used in the expression used to generate the output name.
    2. Enter the optional Description.
    3. Select the output to be created by ticking the Export data and Statistics output properties as appropriate.
    4. Enter the Output name. This property is used to generate the sheet name when using an Excel output or the file name when using a Csv output. It can either be an expression or plain text. See the topic Using Expressions in the Folder, Workbook, Template or Sheet Names. Note that the name for the statistics output is configured in the Statistics tab.
    5. Click Apply to create the log page.
  4. Select the Period tab to configure the default period. The properties in the Period tab are only used if you generate the export using SCADA Basic and do not specify the period in the EXPORT instruction. See the topic Understanding the time period used by Data Export for information about how the period properties are used.
  5. Select the Columns tab to configure the columns that will appear in the output. See the topic Configuring the log page columns.
  6. If you have chosen to generate the statistics output, select the Statistic tab to configure what will be generated. See the topic Configuring the Log Statistics Output.
  7. Click the OK button to close the dialog. The final configuration step is to add a Log List to the log page. See the topic How to Add a Log List filter to a Log Page.
