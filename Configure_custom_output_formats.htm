How to Configure Custom Output Formats

Custom output formats may be used when displaying the value of register. Use the following procedure to enter a custom output format.

  1. Apply the required animation to the drawing element. The drawing element dialog is displayed with a new tab corresponding to the chosen animation.
  2. Select or enter the name of the register to be displayed.
  3. Select the down arrow button next to the Format field.
  4. Select the Custom option. The Custom Display Format dialog is displayed.
  5. Enter the format in the Output Format field. You can either type in the format characters or select one of the examples using the button next to the field.
  6. If you are displaying a time and date, select the received input units.
  7. Click the OK button to confirm the format.
