Special Days Behavior Page

The Special Days Behavior page is displayed using the Special Days Behavior tab on the Calendar page. It displays the Special Days collections that can be selected for the schedule. Show picture

The Special Days Behavior page displays the time intervals to be used for all days of each selected Special Days calendar, for a particular schedule. Users with configuration rights can edit the intervals. See the topic - Editing a schedule.

Special Days Behavior page main features

  1. The identity of the schedule to which the Special Days Behavior applies.
  2. The default output of the schedule - that is the output when there is no active interval BACnet only).
  3. Edit toolbar. Edit tools for the Schedule Control. See the topic - Editing a schedule.
  4. Schedule Control. Provides the main display and edit features for the schedule. See the topics - Schedule Control main display features and Editing a schedule.
  5. Edit mode tools. See the topic - Editing a schedule.
  6. Schedule Control zoom tools. See the topic - Schedule Control main display features.
  7. Selection of the Special Days calendars used in the schedule.

Selecting Special Days Calendars

All configured Special Days Calendars are available to each BACnet Schedule. They are not available to PcVue Native Schedules. To enable a Special Days Calendar tick the adjacent box and then the OK button. Selected calendars appear in the Schedule control in a similar manner to days in a Standard Week.

The Special Days Calendars select which are the special days. The intervals for those special days are configured in the Special Days Behavior page and therefore can be different for each schedule.
