Language Record Format

Lines starting with the keyword LANG identify the languages used.

The fields of a Language record are as follows:

Field

Description

Data type

Value

1

Record ID

Text

LANG.

2

Rank

Numeric

1 to n.

3

Language code name

String

The Microsoft identifier for the language and locale, separated by a hyphen.

4

Language locale identifier

Numeric

The Microsoft locale ID in decimal.

5

Role ID

Numeric

Always 0.

Example

This is an example of a record indicating the English (language) and USA (locale).

LANG,1,ENU-USA,1033,0
