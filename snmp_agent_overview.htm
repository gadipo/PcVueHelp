Overview of the SNMP Agent Add-on

The SNMP Agent Add-on allows you to make PcVue data available to third party SNMP Managers. It is designed as an agent extension for the Microsoft Windows SNMP service and supports SNMP v1, v2 and v2c.

The SNMP Agent Add-on can:

  • Expose the value of I/O and internal variables (scope Shared and Local station) as SNMP data.
  • Send traps.
  • Create and maintain an up-to-date MIB file comprising the list of exposed data.

The Add-on fulfills the following two requirements:

  • Expose system variables via SNMP so that standard IT monitoring tools can obtain information about how PcVue main components are behaving and report any major change: Start, stop, network connection states, database connection states…,
  • Use PcVue as an SNMP gateway between automation devices and SNMP-capable enterprise systems.

There are no configuration dialog's specific to the SNMP Agent Add-on. At the time of writing, it uses a combination of dedicated text files (.dat) and regular configuration features. The Add-on automatically generates a MIB file containing the OID's of variables that are configured as exposed to SNMP.
