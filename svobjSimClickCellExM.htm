SimClickCellEx Method

 See also Applies to 

Simulates a click in the currently selected cell - extended version.

Syntax

Sub SimClickCellEx(Row As Long, Column As Long, mkSimulation As ModifierKey)

Part

Description

Row

Identifies the row.

 Column

Identifies the column.

mkSimulation

Identifies a modifier key.

The cell is then given focus and selected if it is selectable.
This is especially useful for working without a mouse.
This method does not fire a ClickCell event.
