Data Type Conversion Instructions

Instruction

Action

BIN

Return a string representing, in binary, a number passed in base 10.

CONVERT

Convert alphanumeric string to & from Hexadecimal, Octal, Binary and BCD.

DVAL

Return the numeric value of a character string as a DOUBLE.

HEX

Return a string representing, in hex, a number passed in base 10.

IVAL

Return the numeric value of a character string as an INTEGER.

LVAL

Return the numeric value of a character string as a LONG.

OCT

Return a string representing, in octal, a number passed in base 10.

SVAL

Return the numeric value of a character string as a SINGLE.

TOC

Convert a number to a STR (character string).

TOD

Convert a number to a DOUBLE.

TODOUBLE

Convert a variable to a DOUBLE value.

TOI

Convert a number to an INTEGER by truncation.

TOL

Convert a number to a LONG.

TOLL

Convert a number to a LONGLONG.

TOS

Convert a number to a SINGLE.
