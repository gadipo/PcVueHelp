Using the Trend Viewer's Toolbar

The toolbar is the main source of the tools used to manipulate the Trend Viewer at run-time.

Select real-time mode.

Select pause mode.

Explode time axis. The time axis is doubled. The maximum period for the time axis is limited by its original configuration.

Implode time axis. The time axis is halved. The minimum period for the time axis is one second.

Restore the time axis to its original configuration.

Historic. The Historic Request dialog box is displayed. See below for a full explanation of its operation.

Print the trend display on the default printer.

Select cursor mode. The cursor appears as a single vertical line.

Zoom in area. See the topic Using the cursor for further information.

Zoom in time axis only. See the topic Using the cursor for further information.

Return to previous zoom level.

Cancel all zoom. The Trend Viewer's X and Y axis return to their original configuration.

Select the default cursor.

Restore the entire Trend Viewer to its original (design time) configuration. This will reset any changes made using the Trend Viewer Legend at run-time even if they have been previously saved.

Change the period of the time axis. The Change Period dialog is opened. See below for a full explanation of its operation.

Select cross-hair cursor mode. The cursor appears as a horizontal and vertical line.
Cancel the pending historic data request. Useful if you have made an error and it is taking too much time to retrieve the data.
Run the Data Export Wizard.

Using the Historical Request dialog

The Historical Request dialog is opened from the Historic tool. The dialog will appear in one of two forms depending on what was last used - either with a start and end time or with a start time and period. You can change modes by clicking the large vertical bar. Show picture



To make a historical request you enter a new start and end time, or new start time and period as appropriate. Selecting the Apply button will start the historical request and leave the dialog open. Selecting the OK button will start the historical request and close the dialog. If the Trend Viewer is currently in real-time mode, it will automatically switch to historical mode.

The colored circles denote the status of processing of each variable (not the color codes of traces of the variables) as follows:
– in progress
– completed.

Using the Change Period dialog

The Change period dialog is opened from the Change period tool. It can be used in both real-time and historical modes. Using it, you can change the period for the X axis and optionally the X-axis grid. If the Trend was configured with a fixed rather than automatic grid, it may be necessary to reconfigure it to reduce the number of grid lines, particularly if you make a big increase in the period. Show picture



The maximum period that you can enter using the Change Period dialog is limited to that of the Time capacity property found in the Display tab.
