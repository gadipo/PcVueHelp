Using the Alarm Viewer Toolbar

The toolbar contains the tools used to select the Alarm Viewer mode, filter the alarms that are displayed and allow the user to act on alarms (acknowledgment, masking...).

Mode selection - Select On-line mode. This button is only displayed when in List mode.
Mode selection - Select List mode. This button is only displayed when in On-line mode.
Filtering - Display alarms that are On and Not Acknowledged.
Filtering - Display alarms that are On and Acknowledged.
Filtering - Display alarms that are Off and Not Acknowledged.
Filtering - Display the alarms that are Off and Acknowledged.
Filtering - Display all alarms that are unavailable.
Filtering - Display alarms that are user masked.
User action - Acknowledge selected alarms.
User action - Acknowledge all displayed alarms.
User action - Mask selected alarms.
User action - Un-mask selected alarms.
Command - Print selected alarms.
Filtering - Display alarms that are masked by a program.
Filtering - Display alarms that are masked by dependency on another variable.
Filtering - Display alarms are inhibited.
Filtering - Display alarms that are not accessible.
Filtering - Display alarms that are masked by an expression.
Filtering - Display alarms that have maintenance mode set.
User action - Start maintenance mode on an alarm.
User action - Stop maintenance mode on an alarm.

The modes ENABLE and DISABLE of the SCADA Basic instruction VARIABLE are the only way to change the inhibition status of an alarm. As for any other type of variable, such changes done by script will be logged if the alarm is archived and the log list is configured appropriately.
