Importing Strings for a Single Mimic, Mimic Template or Symbol

This process imports strings for a single mimic, mimic template or symbol. Before you use Import/Export for a single item, you must first configure the Import/Export Tool to select the source import/export file and import language. See the topic Configuring the Import/Export tool.

If the Import/Export options are grayed out in the main menu or pop-up, ensure that the type of process (mimic, mimic template or symbol) is enabled and that the import/export file names are configured in the dialog Translation Import/Export Tool.

If the file from which you are importing contains no entries for the selected item, no warnings are given and the item remains unchanged.

How to import the strings to a single mimic or mimic template

  1. Open the mimic or mimic template and select Design mode.
  2. Right-click on the mimic and select Mimic Translations.Import from the pop-up menu.

The import process proceeds without further interaction with the user unless any errors are encountered.

You can also select the import command from the menu using Window.Mimic Translations.Import.

How to import the strings to a single symbol

  1. Open any mimic or mimic template that contains an instance of the symbol.
  2. Right-click on the symbol and select Symbol Translations.Import from the pop-up menu.

The import process runs without further interaction with the user unless any errors are encountered.

You can also select the Import command from the menu using Edit.Symbol Translations.Import.
