Zoom (Window) Method

 Applies to 

To zoom the window's contents by a zoom ratio.

Syntax

Sub object. Zoom(ZoomValue As Integer,[Horizontal As Integer],[Vertical As Integer], [Window As Integer])

Part

Description

ZoomValue

The zoom factor (as a percentage of the original window size) to be applied to the window's contents.

Horizontal

Defines the horizontal co-ordinate in pixels of point that the zoom operation has to keep visible and centred. Default: window.width / 2.

Vertical

Defines the vertical co-ordinate in pixels of point that the zoom operation has to keep visible and centred. Default: window.height / 2.  

Window

The index of the mimic window for which the zoom is executed.

A mimic can have several windows open (see the Mimic collection property: Windows).

The Zoom method in Microsoft VBA applies to the UserForm object.

The Zoom method for windows is subject to these Window properties: EnableZoom, MinimumZoom and MaximumZoom.
