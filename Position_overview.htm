Overview of Position Animations

The position animations allow you to move, rotate and change the size of drawing elements at run-time according to the value of one or more register variables. Some of the position animations also allow the user to set the value of the variables by clicking and dragging the drawing element. This is known as Cursor Behavior and is typically used for creating slider controls.

Specialized position animations

Included in the position animations are two specialized animations not covered by this book.
