Checking DirectX on your PC

Using 3Dmimics with PcVue requires a computer that supports graphic hardware acceleration and an appropriate runtime version of DirectX to be installed on the operating system. See the readme file for more information.

Performance

Lack of system performance can be caused by inadequate hardware or by running several applications at the same time. It often manifests itself as a corruption of the contents of a 3D mimic on the screen. This does not affect the mimic file on disk.

The corruption may take the form of a mimic's contents 'breaking up' or even of the Windows Desktop's contents appearing through the mimic. Should either of these effects be experienced, all applications should be shut down and the PC restarted. You should then consider increasing the PC's performance and/or reducing the number of tasks that are running.

DirectX diagnostics

You can check the version of DirectX on your PC, and its behavior when rendering 3D graphics, by using the DirectX Diagnostic Tool.

  1. Start the DirectX Diagnostic Tool, DXDIAG, from the Windows prompt.
  2. The diagnostics take a few seconds to check the PC. Progress is indicated by the green bar at the lower left of the dialog. When this finishes, check in the System tab that the version of DirectX is as specified in Prerequisites for 3D mimics. Show picture

  3. In the DirectX Diagnostic Tool dialog, select the Display tab (or Display1, Display 2 etc. if you have multiple displays). Show picture

DirectX Features

If available, use the buttons to run tests on the 2D and 3D graphical capabilities and performance.
  1. If you wish to save the results of the tests, click on the button Save All Information.
  2. In other tabs, you can check the PC's related device and driver files.
  3. Close the dialog when you are finished.
