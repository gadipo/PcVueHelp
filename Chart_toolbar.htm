Using the Chart Run-time Toolbar

The toolbar is main tool used to interact with the Chart at run-time. The same tools are also available from a pop-up menu displayed when right-clicking anywhere in the chart.

The position and contents of the toolbar can be changed at design time. The location of the tools within the toolbar can be changed and/or individual tools removed. The toolbar location can also be changed or the toolbar even removed altogether. This topic refers to the default configuration of the toolbar.

Changes that are made to the Chart using the Toolbar are temporary and are not saved when the mimic containing it is closed.

 Return the Chart to its default (as design time settings.

Cursor mode. A vertical line appears on the chart at the mouse position. The value of each data series at that position is displayed in the chart. Note that this only functions for data series configured with a line or area style.

Zoom X-axis. Click and drag the mouse to zoom the X-axis. If the scrollbar option was enabled when the Chart was configured a horizontal scrollbar is displayed.

Zoom area. Click and drag the mouse to zoom both X and Y axes. If the scrollbar option was enabled when the Chart was configured, horizontal and vertical scrollbars are displayed.

Zoom in. The chart X-axis is zoomed in by a pre-configured amount (20%). If the scrollbar option was enabled when the Chart was configured a horizontal scrollbar is displayed.

Zoom out. The chart X-axis is zoomed out by a pre-configured amount (20%). When 100% is reached, the scrollbar is hidden.

Reset zoom. Reset any zoom so the chart is displayed at 100%. Any scrollbars are hidden.
Historical request. See below.

Refresh. The chart is refreshed by making a new historic request using the properties configured at design time.

Print. Print a hard copy of the chart on the default printer.

Making a historical request

The Historical Request tool displays a dialog from where you can change the historical request used to get the data displayed in the Chart. The dialog appears in one of two formats - either with a Begin Time and End Time or with a Begin Time and Period. You can change between the two the formats by clicking the large vertical bar. Show picture

To produce the data that forms the Chart's series, the Chart samples PcVue's historical data. The number of samples is determined by the Period and Sample properties. Or the Begin Time, End Time and Sample properties if using the alternative format of the Historic Request dialog. For example, a period of 2 minutes with a sample rate of 5 seconds would generate 25 samples.

The maximum number of samples for one series is 20,000. If you enter a Period and Sample that exceeds this an error will be displayed and the historic request will not be processed.
