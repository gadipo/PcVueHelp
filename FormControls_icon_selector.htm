The Icons Selector

The Icons Selector is used to generate and select icons that can be used within the Form Controls. Currently only the Tree Control supports icons. The Icons Selector uses images from the libraries to generate the corresponding icons. The images are scaled by the Icons Selector to generate icons of 16x16 pixels.

As for colors in the color palette, icons can be used in 2 ways:

  • Non-indexed icons - When a non-index icon is used, a copy of the icon is saved in the control. Changing the icon in the Icon Selector does not affect previously configured form controls. You can configure up to 120 non-indexed icons.
  • Indexed icons - When an indexed icon is used, the icon index is used and saved in the control. Changing the icon in the Icon Selector affect all form controls using it. You can configure up to 120 indexed items.

The Icon Selector can be opened either from the Main Menu (Display.Icon Selector) or from with from controls where it is supported.

How to add an icon to the Icons Selector

  1. Open the Icons Selector.
  2. Choose the position of the icon to generate and click the exclamation mark to open the Image Browser dialog.
  3. Select the library.
  4. Select the image by double clicking it. Show picture
