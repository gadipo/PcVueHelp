Drawing Layers and De-clutter Templates

 PcVue supports a CAD-like layer system for drawing. When you create any drawing element it is allocated to one or more layers. In order for the drawing to be visible, one or more of the corresponding layers must be enabled.

In the window configuration you can select the enabled layers in two ways.

  • By using the Layer buttons.
  • By specifying a de-clutter template. De-clutter templates enable layers according to zoom level. If you select a de-clutter template then the layer select buttons are no longer operational. See the Creating and Using De-clutter Templates topic for more information.
