MultiTech gateway configuration

To connect your MultiTech Gateway using this Add-on, you need to log in on the interface of its Node-Red application.

  1. Type the IP address of the gateway in a web browser and enter your username and password. Show picture

  2. Click on the Apps tab and launch Node-Red. Show picture

  3. Drag a function node from the left hand side and fill it with the following code: Show picture

    var ret = {}

    ret.payload = JSON.stringify(msg);

    return ret;

  4. Finally, pick a TCP Request node and fill it as in the picture below. Connect the nodes and click Deploy. Show picture
