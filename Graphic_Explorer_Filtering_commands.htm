Filtering Commands in the Graphic Explorer

The top three rows of buttons in the Graphic Explorer turn filter commands on or off.

When a filter is turned on it allows all objects of a particular type to be displayed in the Graphic Explorer. When a filter is turned off, it prevents all objects of a particular type from being displayed in the Graphic Explorer.

  • When a filter is turned on, its button has a gray background
  • When a filter is turned off, its button has a white background

When all of the filters are turned off, only the mimics are listed in the Graphics Explorer.
