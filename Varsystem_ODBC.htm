ODBC Status Variables

SYSTEM.DB.UnitName.STATUS

R

Bit

Manual

State of the connection with the ODBC data source.

0 OK
1 Error

The system variable is set to 1 (Error) when PcVue receives one of the following ODBC error code as part of the response to a request:

  • 08001 - ODBC client unable to establish connection
  • 08002 - Connection name in use
  • 08003 - Connection does not exist
  • 08004 - The ODBC server rejected the connection
  • 08007 - Connection failure during transaction
  • 08501 - Communication link failure

Any other error code is ignored and does not affect the value of the system variable.
