VBA Tools for the Graphic Explorer

The Graphic Explorer provides tools to:

  • Mark each object configured as a VBA Control.
  • Filter the list of objects according to whether there is an associated event.
  • Directly access an object's VBA event script.

VBA control marker command

You can mark the entries for objects configured as VBA controls:

  • In the Graphic Explorer click on the button, Highlight VBA Control Objects. Show picture

When the VBA Control Marker is turned on, each object listed in the Graphic Explorer and configured as a VBA Control is displayed with the informational text (position, identity etc.) in BLUE.

Complex filtering

The VBA event filtering commands work in the same way as the animation filtering commands:

  • Any filtering by object type (i.e. Rectangle, Ellipse, Group etc.) takes place first.
  • The sub-filters for Events are applied to its result.
  • The filtering process applies a logical 'AND' between the object type filters and the animation and VBA event filters (e.g. display objects [Rectangle AND scripted with VBA Event]).
  • The VBA event and animation filtering commands are combined by using a logical OR (e.g. display objects [Rectangle AND [without animation OR with scripted with VBA event]]).
  • An object VBA event is regarded as scripted as soon as the object event procedure is declared in the VBA scripting editor (with or without code).

Displaying Event details

When displayed, each event is represented by a sub-item that is placed directly below the VBA Control object or VBA Project to which it belongs. The sub-items display an Event icon with the event name next to it.

There are two kinds of event:

  • Associated with a VBA project, i.e. for a Mimic or Symbol (if defined as a VBA Project).
  • Associated with a VBA Control object.

VBA scripting for access to an event

From the Graphic Explorer, you can directly access the code of a VBA Control object or VBA Project event as follows:

  • In the Graphic Explorer, double-click on the VBA event item.

The VBA Editor is automatically opened and the editor cursor positioned at the first line of code of the event: Show picture
