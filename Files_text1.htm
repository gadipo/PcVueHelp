File and File Item Properties Specific to the Text Format

The following properties are for File and File Item objects when using the text format.

File

Property Description Default value or example

Encoding

 Character encoding.
ANSI, UTF8 without BOM, UTF8, UTF16 BE, UTF16 LE

Default: ANSI

Header

 Text for the beginning of the file before any items. Can be fixed or contain expressions and / or parameters.

Example: “= Title : Building list of Area ” + me.Description

Footer

 Text for the end of the file after any items. Can be fixed or contain expressions and / or parameters.

Example: “This is the end of the file”

Separator

Field separator.

None, semicolon, comma, tab, space.

Default: semicolon

File item

Property Description Default value or example
Text String to store in the file. can contain multiple data (name + description for example) Example: Me.Name + « , » + Me.description
Line position Position of the line with the file.
Intermediate, start lie, end line.
