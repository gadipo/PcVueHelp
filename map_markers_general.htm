Managing Map Markers in General

This topic is for managing map markers in general. There are also topics for each type of marker (shape, text, image, symbol, track and polygon) which contain more specific information.

The Map Markers toolbar

The toolbar selects the mode in which the cursor operates.

Icon Description
Map - Select to zoom and/or pan the map. This is the default mode.

Select - Select and optionally move one or more markers on the map.

  • Click on a single marker to select it. Click again anywhere else in the map deselects it.
  • Click on the stroke (not a node) to select a track or polygon.
  • Ctrl and click selects multiple markers.
  • Click (not over a marker) and drag anywhere to select all markers within a frame.
  • Click and drag a marker to move it.
Shape marker - Click anywhere within the map to create a new shape marker.
Text marker - Click anywhere within the map to create a new text marker.
Image marker - Click anywhere within the map to create a new image marker.
Symbol marker - Click anywhere within the map to create a new symbol marker.
Track - Click anywhere within the map to create the start of a new track marker.
Polygon - Click anywhere within the map to create the start of a new polygon marker.
Delete - Delete all selected markers.

How to create a single marker

The process to create a marker is generally the same for all marker types.

  1. Zoom and pan the map so that it is at a convenient position to add the marker.
  2. Using the map objects tree, select the layer to which the marker is to be added.
  3. Select the tool corresponding to the marker to be created. The marker properties are displayed.
  4. Click in the map at the required position. A new marker with default properties is created. The toolbar automatically returns to the default Map Mode.
  5. Configure the properties as required.

Be careful not to click & drag. Click & drag pans the map without adding a marker.

How to create multiple markers with similar properties

  1. Zoom and pan the map so that it is at a convenient position to add the marker.
  2. Select the tool corresponding to the markers to be created. The default marker properties are displayed.
  3. In the marker properties, select Modify properties of a new marker.
  4. Select the layer under which the marker is to appear using the Layer drop down list box, or the map objects tree.
  5. Configure the marker properties. These properties are then used as default until you select a different tool.
  6. Click one or more times in the map. As many markers as you have clicked will be created, all with the properties that you entered in the previous step (except the position and the name that must be unique within the layer).
  7. Select the Select objects tool. Select and configure any markers that require different properties to the default that you entered.

The Map Editor also supports clipboard operations on the markers. You can copy and paste them using the Ctrl+C and Ctrl+V keyboard shortcuts.

How to delete one or more markers

  1. Zoom and pan the map so that it is at a convenient position to select the markers.
  2. Select the markers.
  3. Use the Delete tool (Layer or Markers toolbar) to delete the selected markers.

Stroke thickness         Y Y The width of the line when rendering a track or polygon.
Stroke Dash         Y Y The style of the line when rendering a track or polygon.
Latitude Y Y Y Y Y Y Used in conjunction with the longitude to define the location of the marker. Can be configured by dragging the marker, or by entering a numeric value.
Longitude Y Y Y Y Y Y Used in conjunction with the latitude to define the location of the marker. Can be configured by dragging the marker, or by entering a numeric value.
Width Y Y Y Y    

The width of the marker in pixels irrespective of the map zoom level. A value of zero has special meaning.

For text, it means automatically adjust the size according to the font and number of characters. All characters will be on a single line.

For symbols and images, it means display at their native size (no scaling).
Height Y Y Y Y     The height of the marker in pixels irrespective of the map zoom level. A value of zero has special meaning (see Width).
