Logical Operators

See also

The logical operators are:

Symbol

Operation

!=

Inequality

==

Equality

>

Greater than

>=

Greater than or equal to

<

Less than

<=

Less than or equal to

!

Negation

||

Logical Or

&&

Logical And

Example 1

DIM I1 As Integer, I2 As Integer;

If (I1 == I2) then ...

If (I1 >= I2) then ...

Example 2

DIM Bool1 As Integer, Bool2 As Integer;

Bool1 = 0;

Bool2 = 1;

If (Bool1 == Bool2) then ...

If (Bool1 || Bool2) then ...

String concatenations and comparisons may only be done using specialized functions.

Logical operations on register variables are made using the LOGICAL or LOGICAL64 instructions.
