OPC UA client status Variables

SYSTEM.OPCUA.XON

R

Bit

Manual

Flow regulation status.
0 Flow regulation activated. Loss of exchanges possible.
1 Flow regulation not activated.

 

SYSTEM.OPCUA.LOSTTRANS

R

Register

Automatic

Counter for the number of lost OPC UA exchanges from the when flow regulation was last activated.

 

SYSTEM.OPCUA.CURRENTDATACHANGES

R

Register

Automatic

Flow counter of OPC UA data changes being processed.

 

SYSTEM.OPCUA.<Network>.ON

R

Bit

Automatic

Network activation indication where <Network> is the network name.
1 The network is activated.
0 The network is not activated.

 

SYSTEM.OPCUA.<Network>.STATUS

R

Register

Automatic

Network status where <Network> is the network name.
0 Network is started and healthy
1 Network state unknown
2 Network is stopped
3 Network not started due to license, trial period elapsed or missing license option

 

SYSTEM.OPCUA.<Network>.<Server>.ON

R

Bit

Automatic

Server activation indication where <Network> is the network name and <Server> is the server name.
1 The device is activated.
0 The device is not activated.

 

SYSTEM.OPCUA.<Network>.<Server>.STATUS

R

Register

Automatic

Server status indication where <Network> is the network name and <Server> is the server name.
0 The server is connected with a live session
1 Connection in progress
2 Invalid server Url
3 Invalid session's security information
4 Configuration validation failed
5 The server is disconnected
6 Connection warning - Watchdog time-out
7 Connection error - Reconnection in progress
8 Remote server shutdown
9 Connection error - Failed to validate certificate
10 Connection error - Failed to open secure channel
11 Connection error - Failed to create session
12 Connection error - An error occurred upon session activation
13 Connection error - Server could not be reached
14 Session activation retry in progress
15 Secure channel closed due to a publish request time-out
16 Invalid client security information
17 Server certificate not found
18 Server certificate is invalid
19 Server certificate is not trusted
20 Server certificate mismatch
21 Server certificate verification failed
22 Client connection error not overridden
23 Server unavailable

 

SYSTEM.OPCUA.<Network>.<Server>.REDUNDANCYACTIVECLIENT

R

Text

Automatic

Returns the name of the active data acquisition server when the OPC UA Server is configured to be produced by an association of PcVue servers. Where <Network> is the network name and <Server> is the name of the main server.

 

SYSTEM.OPCUA.<Network>.<Server>.REDUNDANCYACTIVEON

R

Bit

Automatic

The redundancy status of the server.

0 Server is in standby (not active).
1 The server is active.

R

Automatic
Standby server activation indication where <Network> is the network name, <Server> is the server name and <StandbyServer> is the standby server name.
1 The server is not activated.
0 The server is activated.
