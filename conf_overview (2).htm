The Configuration Environments

The HMI workspace

The Workspace is the name given to the area that the PcVue desktop application occupies on the screen of your PC. The workspace provides the tools used to develop the windows that form the user interface and is also the container for those windows at run-time. It is also used to develop windows and symbols displayed on the non desktop clients including WebVue and SnapVue.

The workspace main menu provides direct access to the main configuration tools, that is the Application Explorer, Application Architect and Smart Generators. Show picture

The Application Explorer

The Application Explorer is an explorer style configuration tool that can be used for the majority of the non-graphical project configuration except for the selection and configuration of equipment communication protocols. Show picture

The Application Architect

The Application Architect is a tool to configure part or all of an application (project) by designing and instantiating templates. A template is a representation of a real world object in terms of PcVue configuration elements. Show picture

The Smart Generators

A Smart Generator is a software tool for generating the PcVue project configuration by importing it from files created by third party software such as PLC programming packages or generic XML files. It works by communicating directly with PcVue configuration management interfaces to generate the configuration. Show picture

In general, the Smart Generators are limited to generating variables and communication links. However, the Smart Generators are not just simple conversion tools. They allow you to filter, convert and modify the structure of the data so that it is applicable to PcVue.
