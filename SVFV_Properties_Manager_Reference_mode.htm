Using the Reference Mode of the Drawing Properties Manager

The reference drawing element is the last that has been selected along with other drawing elements. Its drawing handles are black whereas those of other objects in the selection are white. Show picture

The reference object is the one on the right.

The reference mode is used for copying the properties of the reference object to the other objects in a selection. In reference mode, the Drawing Properties Manager shows two additional buttons. Using these buttons you can apply all or part (from the active tab only), of the reference drawing elements properties to the other drawing elements in the selection.

Displaying and modifying properties

In Reference mode, the Drawing Properties Manager works similarly to Selection mode with a single object selection (i.e. the reference one):

  • The Drawing Properties Manager displays the appropriate tabs for the properties provided by the reference object only. It disables the property controls and names not provided by the reference object.
  • Any change to a property value is applied directly to the reference object only.

Using the Apply to selection buttons

The Apply To Selection command buttons can be used to apply all or part (from the active tab only) of the reference object properties to the other objects in the selection. Show picture

In this case, the Shape controls are grayed out because the reference object does not support them.

Command

Effects

Apply All

The Drawing Properties Manager applies all the available (i.e. not disabled) properties of the reference to the other drawing elements in the selection.

Apply tab

tab is the name of the currently active tab.

The Drawing Properties Manager only applies the available (i.e. not disabled) properties shown in the active tab.

The command is disabled if the property values in the reference are the same as those of the target.

The Apply To Selection buttons are not displayed if there is no property or selected drawing element available. The None tab is then displayed to say so.
