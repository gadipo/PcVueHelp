Adding Associated Labels to a Template

This topic explains how to add associated labels to a template using the Application Architect. There are three types of associated label

  • Bit - Used to associate text strings with a bit variable's values.
  • Alarm - Used to associate text strings with an alarm variable's values.
  • Enumerated - Used to associate text strings with pre-defined discrete numeric or text values.

For an explanation of their use, see the help for the Application Explorer and HMI.

How to add a bit or alarm associated label to a template

  1. Navigate to a template's Configuration elements folder.
    1. In the right pane of the Application Architect select the Templates tab.
    2. Using the templates configuration tree select the template to which the associated label will be added.
    3. Using the secondary navigation tree select the Configuration elements folder.
  2. Add and configure the Associated label.
    1. Select Add associated label and then the label type (Bit or Alarm). A new associated label with a default name is created.
    2. Edit the name to one appropriate to your application. You can only edit the name directly by clicking on it in the configuration tree.
    3. Configure the associated label's properties, in particular the series of custom labels you require. Show picture

How to add an enumerated associated label to a template

  1. Navigate to a template's Configuration elements folder.
  2. Add and configure the Associated Label.
    1. Select Add associated label and then the Enum label type. A new enumerated associated label with a default name is created.
    2. Edit the name to one appropriate to your application. You can only edit the name directly by clicking on it in the configuration tree.
  3. Populate the list of discrete values and their labels.
    1. Right click the Enumerated Label and, from the context menu, select Add a label. A new label is created.
    2. Configure the label. You must at least configure the Enum value and Value (Value is a text string).
    3. Repeat a. and b. for all the enumerations that are required. Show picture

After they have been created, the order in which the labels are listed can be changed by right clicking and selecting Up or Down from the context menu.

Configuring an associated label's advanced properties

There are no advanced properties for an associated label.
