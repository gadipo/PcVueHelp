Using the Log Viewer to Add a Comment

A user comment of up to 40 characters may be added to any line in the display list. To enter a comment, click on the clipboard icon beside the relevant line. You can then type the comment directly into the line.

Using the soft keypad when entering a comment

On touch screens, you can use PcVue's soft keypad when entering a comment. To enable, use the option Soft keypad management.Touch screen.Use soft keypad when modifying the comment using the Log Viewer in the HMI Options. See the topic Soft Keypad Management.
