Using the Color Picker and Dropper Tools

The color picker and dropper tools allow you to pick a color from one drawing element and use it to color another. The tools are found both on the Color Palette and on the Draw toolbar. Show picture

Both color picker and dropper tools only work when a mimic is in Draw mode.

From where can I pick a color?

You can pick a color from any of these sources:

  • The border or fill of a drawing element or text.
  • A graphic object or shape in a group or a symbol, or derived from the mimic template (if used).
  • An image (bitmap, GIF, WMF etc.).
  • An ActiveX control.
  • An OLE object.
  • A color in the Color Palette.
  • A color swatch in the Properties dialog for a drawing element graphic object or animation. Show picture

You cannot apply a color directly from the Color Palette's own Color Picker dialog. Instead, use it to add the color to the swatch for Line and/or Fill and then click on OK to apply it.

How to pick a color

  1. Select the Color Picker tool. The icons and color swatch at the top of the color palette change to indicate that it is selected. Show picture

    Note the two 'empty pipette' icons.

  2. Position the cursor as follows. The shape of the cursor changes to appear as an empty pipette when it is over a suitable drawing:
    • To select a color in a drawing element, hold the cursor over it.
    • To select a color from a properties dialog, open it. You cannot do so first and then select the Color Picker tool.
    • To select a color from a picture, ActiveX or OLE object, select a pixel in it.
  3. Left click to pick the color.

You can use Shift + left click to pick the color from a drawing element's border.

Memorizing a color

The Color picker 'remembers' the color most recently picked in the project, even when the Color Picker and Color Dropper tools are not selected. This enables you to re-apply that color until you pick a different color.

Where can I drop a color?

The range of items onto which you can drop a color is more limited:

  • The border or fill of a drawing element or text.
  • A color in the Color Palette.
  • A color swatch in a drawing elements Properties dialog.
  • A drawing element in a group (not to multiple elements).

You cannot drop a color onto a graphic element in a symbol (also see Restrictions below).

How to drop a color

  1. Pick a color using the Color Picker tool. The icons and color swatch at the top of the color palette change to indicate that it is selected. Show picture

    Note the two 'full pipette' icons.

  2. Select the color dropper tool and position the cursor over the item that you want to color. The shape of the cursor changes to appear as a full pipette when it is over a suitable drawing.
  3. To drop a color into a Properties dialog, open it. You cannot do so first and then select the Color Dropper tool.
  4. Left-click to drop the color.

To drop a color onto a drawing element border, use Shift + left-click.

Restrictions

  • To modify the indexed colors in the Colors Palette, a user must have the Preferences right enabled. (Configure.project.Rights.Users and profiles.Profiles.profile. Properties.Access.Preferences).
  • The type of color must be suitable for the context into which it is 'dropped'. For example, you cannot apply a transparent, blinking or gradient color to a mimic's background color setting in its Properties dialog.
