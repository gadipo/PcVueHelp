List Of Image Animations and Behavior

Animation

Behavior

Bit

Selection of 2 images according to the status of a bit. A third image may be selected to represent the invalid state. Show picture

Register value

Selection of images according to up to 128 thresholds applied to a register. Further images can be selected for the variable being invalid or out of range. See the section on changing color for an explanation of threshold operation. Show picture
Text value Selection of images according to the equality of a text variable to 128 entered strings. Further images can be selected for the variable being empty (null), not equal to any of the entered strings, or invalid. The animation is case sensitive or not depending on the value of the following Mimic Options: Enable case sensitivity for animations that compare text strings case sensitive. For example, if set for being case insensitive, HELLO, Hello, and hELLO would be treated as the same text value. Show picture
