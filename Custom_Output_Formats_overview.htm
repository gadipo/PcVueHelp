Overview of Custom Display Formats

Custom display formats may be used to format register values for both output (when displaying a value) and input (when the user is entering a value). The following animation types support this feature.

  • Legend.
  • Register Display and Register Display with Color on Bits.
  • Send Register.

To enter a custom display format you select the down arrow button beside the Format field in the Animation tab and select the Custom option. The Custom Format dialog is then displayed into which you can enter the format characters.

You can also use the multi-language facility to enter different formats for different languages.

The content of the dialog depends on the type of animation. For Register Send, it has both Output Format and Input Format sections.
