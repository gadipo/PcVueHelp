Using the Global Mode of the Drawing Properties Manager

When the Default mode is selected, the Drawing Properties Manager is used to define the initial property settings of all subsequently drawn drawing elements. Show picture

When are the global values ignored?

The global property values are ignored in the following situations.

Properties

Ignored when drawing

Reason

Open Closed

Polygon, Polyline, Bezier shape, Bezier curve.

The open and closed properties are part of the shapes configuration, i.e. a polyline is a polygon with the open property selected.

Zoom bounds

Any drawing element that provides these properties.

When a drawing element is created in a window with a zoom level out of the limits as defined by the Zoom Bounds global properties values, the object's Zoom Bounds properties are not set to the global properties values to avoid creating an invisible drawing.

Layers

Any object that provides these properties.

When a drawing element is created in a window whose layers do not match any of the layers global properties values, the object layers are set to the window's layer values to avoid creating an invisible drawing.
