General Formatting Characters

  • I - Used for input formatting only.
  • O - Used for output formatting only.
  • I/O - Used for both input and output formatting.

I/O

Character

Meaning

O

""

 

Display the string inside the double quotation marks.

O

\

Display the following character literally. Do not interpret it as a formatting character.

O

[=n]

Condition expression to use a format when the value of the variable is n.Example [=15]#.#

O

[#n]

Condition expression to use a format when the value of the variable is not n. Example [#15]h:n:s

O

[<n]

Condition expression to use a format when the value of the variable is less than n. Example [<15]#.#

O

[<=n] or [=<n]

Condition expression to use a format when the value of the variable is less than or equal to n. Example [<=15]#.#

O

[>n]

Condition expression to use a format when the value of the variable is greater than n. Example [>15]#.#

O

[>=n] or [=>n]

Condition expression to use a format when the value of the variable is greater than or equal to n. Example [>=15]#.#

O

;

 

Condition separator - syntax 1. The characters on the left of the semicolon are used to format the value if it is positive and those to the right are used if it is negative. Example "Positive Value";"Negative Value"

O

;

Condition separator - syntax 2. [Condition1]Format1;[Condtion2]Format2;Format3 Format1 is used when Condition1 is met, Format2 is used when Condition2 is met, else Format3 is used.

O

0

Digit placeholder. Display a digit or a zero. If the expression has a digit in the position where the 0 appears in the format string, display it; otherwise, display a zero in that position. If the number has fewer digits than there are zeros (on either side of the decimal) in the format expression, display leading or trailing zeros.

O

#

Digit placeholder. Display a digit or nothing. If the expression has a digit in the position where the # appears in the format string, display it; otherwise, display nothing in that position.

O

.

Decimal placeholder. The decimal placeholder determines how many digits are displayed to the left and right of the decimal separator.

O

%

Percentage placeholder. The expression is multiplied by 100. The percent character % is inserted in the position where it appears in the format string.

O

~

Display the absolute value (unsigned).

O

,

Thousand separator. The thousand separator separates thousands from hundreds within a number that has four or more places to the left of the decimal separator.
