Configuring Multilingual Strings

Many of the text properties configured in the Map Markers Editor are multilingual. That is you can configure different strings for different languages. When a text property is multilingual, an Ellipsis button is displayed adjacent to the property. Clicking the Ellipsis button opens the Manage Translations dialog where you can enter the strings for the various languages. Show picture

At run time, the strings that are displayed are selected by matching PcVue's Project Language, with the language in the Manage Translations dialog. If there is no match, the string for the default language is used.

How to add a language string

  1. Open the Manage Translations dialog by clicking the Ellipsis button adjacent to the text field.
  2. Click the Add Text tool, select the required language and enter the corresponding string. The languages that are available are the same as in the Project Languages in Languages settings.
  3. Click OK to confirm the configuration and close the dialog.
