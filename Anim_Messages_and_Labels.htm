Displaying Texts and Labels

The Text Animation

The Text animation displays the contents of a text variable. The most common use of this is to display the time, date and user name. Other uses include operator prompts, warning messages and status information.

When using the Text animation it is normal to select the property Do not auto size of the support text field. If unselected, the length of the field will dynamically change at run-time according to the number of characters displayed. This can give an odd effect if you are using a style, such as inverse or button, that displays a background color.

If you need to display very long messages, you can select the Multiline property in the support text field. The text may then be displayed on a number of lines, automatically broken at the most convenient white space.

The Label Animation

The Label animation displays text associated with a Bit variable. You can display:

  • The variable’s Associated Label.
  • A variable’s item Description.
  • Context sensitive information such as the name of the window in which the animation resides, the branch of an object etc.

Displaying Associated Labels

An associated label is a sequence of strings associated to variable values. For example, for bit variables, one string is displayed when the bit is off, another when the bit on and a 3rd one when the bit is invalid. If the bit has not been attached to an associated label then the default strings of Animation 0 and Animation 1 will be displayed.

The Label display animation with Label type set to Associated label allows you to display the label corresponding to the current value of a variable whatever its type: Bit, Alarm, Register and Text. For Register and Text variables, you need to use labels associated to enumerations.

Displaying a variable's title

If you choose the option to display the variable's title, and the variable does not have a title, then the variable name will be displayed instead.

A variable's title is bilingual, so if you change the language for the project you must ensure that you have a title in the appropriate language.

Displaying context sensitive information.

To display context sensitive information, you select the Format option button and then enter a substitution string in the field provided. The substitution string determines what is displayed in the label. See the topic Using Substitution Strings or Variable Attributes in an Animation.

You can combine substitution strings and use other characters that are displayed literally.

In the animation toolbox and menu you will find, in addition to the standard Text and Label animations, Text - Bit and Label - Bit. These create a Color on Bit animation at the same time as the Text or Label. See the section on Changing Color for details. They are provided for compatibility with earlier versions of the software where it was only possible to attach one animation to each drawing element.

Using these animations is exactly the same as applying a Text (or Label) and a Color - Bit animations to the same drawing element.
