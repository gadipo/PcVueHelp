About the Translation Import/Export Tool

The Translation Import/Export Tool allows you to export, modify and import the text strings used in mimics, mimic templates and symbols. The tool is especially useful for large projects and for projects that use PcVue's language handling capability. For example, you could export all strings used in a project's mimics, translate them to another language using a third party tool and then import them back into the project as the strings for that other language. Without the Translation Import/Export tool, this would have to be done piecemeal which could be very time consuming.

The use of the Import/Export Tool is not limited to translation. For example, you could export the strings in a project so as to check the spelling by using a word processor such as the one included with Microsoft Office.

Changes you make to all or part of a project using the Import/Export Tool are irreversible, so it is essential that you back the project up before you apply the tool.
