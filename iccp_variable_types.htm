ICCP Variable Types

Supported ICCP data types

There are 15 data types for monitoring and 3 types of commands available with the PcVue ICCP add-on.

ICCP data types Description
Discrete Contains information about a Discrete Value.
DiscreteExtended Contains information about a Discrete Value, Quality Flags, Timestamp (up to second resolution) and COV counter (Change of Value).
DiscreteQ Contains information about a Discrete Value and Quality Flags.
DiscreteQTimetag Contains information about a Discrete Value, Quality Flags and Timestamp (up to second resolution).
DiscreteQTimetagExtended Contains information about a Discrete Value, Quality Flags and Timestamp (up to millisecond resolution).
Real Contains information about a Real Value.
RealExtended Contains information about a Real Value, Quality Flags, Timestamp (up to second resolution) and COV counter (Change of Value).
RealQ Contains information about a Real Value and Quality Flags.
RealQTimetag Contains information about a Real Value, Quality Flags and Timestamp (up to second resolution).
RealQTimetagExtended Contains information about a Real Value, Quality Flags and Timestamp (up to millisecond resolution).
State Contains information about a State Value.
StateExtended Contains information about a State Value, Quality Flags, Timestamp (up to second resolution) and COV counter (Change of Value).
StateQ Contains information about a State Value and Quality Flags.
StateQTimetag Contains information about a State Value, Quality Flags and Timestamp (up to second resolution).
StateQTimetagExtended Contains information about a Discrete Value, Quality Flags and Timestamp (up to millisecond resolution).
ICCP command types Description
ControlCommand A discrete variable that can be modified by a client.
DiscreteSetPoint A discrete variable that can be modified by a client.
RealSetPoint A real variable that can be modified by a client.

Mapping of ICCP types to PcVue variable types

There are a few rules when creating a variable in PcVue and assigning an ICCP type to it:

  • ICCP data types for monitoring
    • All Discrete types - Register variable
    • All Real types - Register variable
    • all State types - Register, bit or alarm variable
  • ICCP data types for commands
    • ControlCommand - Register, bit or alarm variable
    • DiscreteSetPoint - Register variable
    • RealSetPoint - Register variable

Any variable type that does not follow these rules or any PcVue text variable with ICCP information in its extended attribute will not be included in the ICCP model.

For the ControlCommand type, when it is created as a Bit or Alarm:

  • The ICCP value of 0 gives value 0 in PcVue.
  • An ICCP value different from 0 gives value 1 in PcVue.
