List of Color Animations and Behavior

The number of colors that you can select depends on the animation chosen. You do not have to use all of the colors. For example, the animation Color - Bit Group allows you to select up to 16 colors using 4 bits, but you could use it to select just 4 colors using 2 bits. Whichever animation you choose, you must select at least 2 of the available colors since transition to or from an unselected state is not recognized.

Animation

Behavior

Bit

Selection of 2 colors according to the status of a single bit. Show picture

Bit group

Selection of 16 colors according to the binary combination of up to 4 bits. Show picture

Register bit

Selection of 2 colors using a bit masked from a register with a binary rank. For example, if you use the 3rd bit, when the binary representation of the register is xxxxxxxxxxxx1xx the color selected for 1 will be displayed. When the binary representation of the register is xxxxxxxxxxxx0xx the color selected for 0 will be displayed (x = either 1 or 0). Show picture

Alarm

Selection of 5 colors according to the 5 states of an alarm (Off, On - Nack, On - Ack, Off - Nack, Inhibited). If you select the Static option, the colors used are those you specify in the animation. If you select the Dynamic option the colors are taken from the preferences for the appropriate alarm level. Changing the preferences will change the colors displayed in the animation. Show picture

Bit & Alarm

Selection of 8 colors according to the binary combination of the 2 states of a bit and 4 states of an alarm. Show picture

Weighted bits

Similar to above except that the value is calculated by adding the values of 8 weighted bits. Bit 1 = 1, Bit 2 = 2, Bit 3 = 4, Bit 4 = 8 etc. For example, 00000111 = 7. This animation is generally used with binary encoded transducers. Show picture

Register value

Selection of up to 8 colors according to value of a single register. Show picture

See the topic Understanding the Thresholds Used in Color Register Value for an explanation of threshold operation.

Gradient register

You can vary the element's color with gradations between two selected colors, according to value of a single register. Show picture

See the topic Understanding the Gradient Register Animation for an explanation of how the Gradient Register animation works.
Bargraph

Fill a shape according to the value of a register variable. Show picture

See the topic Displaying a register as a bargraph.
Legend

Create a numeric scale using a rectangle as the support element and a register variable to provide the range. Show picture

See the topic The Legend Animation.

Text value

Selection of up to 128 colors according to the value of a text variable. The animation is case sensitive or not depending on the value of the following Mimic Options: Enable case sensitivity for animations that compare text strings. For example, if set for being case insensitive, HELLO, Hello, and hELLO would be treated as the same text value. Show picture
