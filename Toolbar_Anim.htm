Animation Toolbar

The Animation toolbar gives you quick access to the tools used for animation. The toolbar contains one tab for each class of animation.

The Color tab

Color using one bit.

Color on bit group, using a binary combination of up to 4 bits.

Color using a bit masked from a register.

Color on Alarm.

Color on Bit and Alarm.

Color using a value calculated from weighted bits.

Color using a register value.

Gradient color on Register.

Color shape fill using the value of a register (Bargraph).

Display a scale using a minimum and maximum value.
Color using a text value.

The Text tab

Display a text string

Display a string in a color according to the value of a bit.

Display Text with Color on Alarm: applies both of these animations.

Display a label.

Display a label in a color according to the value of a bit.

Display Label with Color on Alarm: applies both of these animations.

Display the value of a register numerically.

Display the value of a register using two bits to change the color.

Display Text on Bit.

Display Text on Bit Group.

Display Text on Register Bit.

Display Text on Alarm.

Display Text on Register Value.

The Image tab

Display an image using a bit value.

Display an image using a register value.

Display an image using a text value.

 

The Symbol tab

Select Symbols using a single bit.

Select Symbols using a binary combination of up to 4 bits.

Select Symbols using a bit masked from a register.

Select Symbols on Alarm.

Select Symbols using the value of a register.
Select Symbols using a register value.

The Position tab

Display the value of a register using movement in one axis.

Display the value of two registers using movement in two axes.

Free positioning - display the value of a register using displacement.

Path positioning - display the value of a register using movement along a path.

Display the value of a register using rotation.

Scaling - display the value of a register using size.
Allow jumping to a position on a Map Control instance
Allow selecting a 3D camera object.

Make a drawing element visible/invisible on Register Value.
Make a drawing element visible/invisible on Text.
Make a drawing element visible/invisible on the existence of a variable.
