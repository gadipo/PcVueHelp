Ordering 3D Models

When you use the ordering tools, models are moved along the 3D world axis that is perpendicular to the window. That is, the axis that is parallel to the direction in which the camera is aimed. For example, when using a Top view, if you select one model and apply Move To Front, it will move to a position above the rest.

The sides of a model

Because they are displayed in a window using the camera in an orthographic mode, the front and rear sides of a 3D model are defined as follows:

Side

Position

Front side

Nearest to the camera.

Rear side

Farthest from the camera.

The model's sides used for ordering are those of a virtual rectangle (which is not displayed) just big enough to encompass the 2D projection of the model's bounding box onto the screen.

It can be difficult to distinguish the front side from the back side of a model. Hence, it is easier to visualize a scene using a second window (onto the same mimic) with a different orthographic camera mode. For example, here are two windows of the same mimic, using the Top View and Front View orthographic camera modes respectively: Show picture

While you use the ordering tools in the first window (Mimic1:1), the front and back sides of a model can be visualized in the second window (Mimic1:2). In the first window (Mimic1:1), the 3D world Y axis is the one perpendicular to the window.

The ordering tools

The four ordering tools are as follows.

Tool

Icon

Effect of clicking

Move To Front

Move the selected model in front of all other models.

Move To Back

Move the selected model behind all other models.

Move Forward

Move the selected model in front of the model immediately in front of it.

If several models are selected, the tool moves the other models in front of the reference model.

Move Backward

Move the selected model behind the model immediately behind it.

If several models are selected, the tool moves the other models behind the reference model.

To order models using these tools:

  1. Holding down the Shift key, select the models to be ordered. A green bounding box is shown around each one. The last one you select is the reference model - its Gizmo is visible.
  2. Click on the button for the kind of ordering you require. The models are then arranged into that order.

When a model is moved in front of another model, it is moved closer to the camera. Conversely when a model moved behind another model, it is moved farther from the camera.

'Move To Front' and 'Move To Back' take all models into account whether selected or not.
