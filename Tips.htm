Drawing Tips

Interrupting the drawing process

When using the following drawing tools you can use the Esc key at any time to return to the selector (pointer) tool.

Line

Rectangle

Rounded rectangle

Ellipse

Half ellipse

Quarter ellipse

Arc

Pie

Polygon

Polyline

Bezier shape

Bezier curve

Text

Color picker

Color Dropper

The current drawing process (if any) will be completed before returning to the selector tool. For example, if you were drawing a polygon the final node would appear at the position of the cursor when the Esc key was pressed. Show picture

Automatic scrolling

A mimic window shows horizontal and vertical scrollbars when it is zoomed in and/or when a mimic is configured to be larger than the window. If you scale or anchor a drawing element outside of the current area of the mimic, the window will track the movement so as to keep visible the object drawing handle that you are manipulating.

For example, as you draw a line you can move its end point toward the right-hand edge of the window area (a). Show picture
