Using the Trigger Pop-up Menu

The trigger pop-up menu is displayed by right clicking on a trigger icon in calendar pane of the Task Scheduler dialog. The following commands are available.

  • Edit - Displays the Trigger Properties dialog. This can also be displayed by double clicking on the trigger icon.
  • Repeat - Shortcut to the Repeat tab of the Trigger properties dialog.
  • Show next - Scroll the calendar to display the next instance of the trigger to be executed.
  • Cut - Copy the selected trigger to the Windows Clipboard and delete it from the schedule.
  • Copy - Copy the selected trigger to the Clipboard.
  • Delete - Delete the current Trigger from the schedule.
