Grid Overview

The Grid control provides a tabular display of text cells. It can be used in one of three modes.

  • General use mode - The cells must be populated programmatically using a VBA script or a SCADA Basic script (SELECTOR instruction). Show picture
  • Variable tracking mode - The cells are populated with a list of variables and real-time values - A filter can be applied to select the variables that are displayed. Show picture
  • Historical mode - The cells are populated with a list of variables and their historical or statistical values. The variables are selected, and the historical or statistical behavior configured, at design time. Show picture

Feature summary

  • Overall size only limited by the size of the container mimic.
  • Configurable number of rows and columns limited only by system resources.
  • Optional row and column headers.
  • Configurable cell color.
  • Configurable font.
  • Optional vertical and horizontal scrollbars.
  • Optional run-time features including cell selection, sort on header click etc. See Configuring the run-time behavior for a comprehensive list. The available run-time features depend on the mode in which the grid control is used.

The Grid Display supplied with version 8.0 of PcVue and earlier, is not fully compatible with the current version and is no longer supplied.

If you are upgrading a project from version 8.0 or earlier, that uses the old style Grid Display, please contact technical support before proceeding with the upgrade.
