Time and Date Formatting Characters

You can display or enter a register as a date and time by configuring a date and time format and specifying the received input unit. The date and time display is calculated as the number of received input units since January 1st 1970 GMT0. For example, dd/mm/yyy hh:nn:ss@s will display 1037620800 seconds (representing 12:00 on the 18th of November 2002) as 18/11/2002 12:00:00.

Formatting time information as a duration

You can also display time information as a duration by enclosing the most significant format characters with square brackets. For example, [d]" days "h" hours"@s would display a raw input of 1234567 (seconds) as 14 days 6 hours.

Table of format characters

  • I - Used for input formatting only.
  • O - Used for output formatting only.
  • I/O - Used for both input and output formatting.
