SendEmail Method

 Applies to 

Composes an email for sending.

Syntax

Function SendEmail(
    EmailProfile As String,
    Recipients As String,
    CopyRecipients As String,
    BlindCopyRecipients As String,
    Subject As String,
    Body As String,
    [FileAttachments As String],
    [Priority As fvEmailPriority],
    [BodyFormat As fvEmailFormat],
    [MessageEncoding As fvEmailEncoding]
) As Boolean

Return value

True if the e-mail is sent successfully, otherwise False.

 

Part

Description

EmailProfile

The name of the email profile used to send the message. If no profile name is specified, the default profile is used.

Recipients

A list of email addresses to send the message to. Semicolon-delimited.

CopyRecipients

A list of email addresses to carbon-copy (CC) the message to. Semicolon-delimited.

BlindCopyRecipients

A list of email addresses to blind-carbon-copy (BCC) the message to. Semicolon-delimited.

Subject

The subject line of the message.

Body

The body of the message.

FileAttachments

A list of names of files to attach to the message. They must be specified as absolute paths. Semicolon-delimited. Default: "".

Priority

The priority (importance) of the message (fvEmailPriorityNormal, fvEmailPriorityLow or fvEmailPriorityHigh). Default: fvEmailPriorityNormal.

BodyFormat

The content type of the body of the message (fvEmailFormatText or fvEmailFormatHtml). Default: fvEmailFormatText.

MessageEncoding

The encoding used for the subject and the body of the email message. Default: fvEmailEncodingDefault.

Example
