3D Axis Positioning Animation

This animation enables you to translate (move) a model or frame in one, two or three axes, using one, two or three variables. Translation is a linear interpolation between the minimum and maximum values of the variables from the design position to the design position plus the Axis Moving (delta) values (see below). The axis or axes in which translation takes place is selected by tick boxes adjacent to the fields containing the variable names.

  • From the Graphic Explorer, select the object's Properties menu: Animate.Position.Single Axis Positioning, regardless of which mode you intend to apply.

Single variable mode

The Drawing Properties dialog for the 3D object opens with the 3D Axis Positioning tab selected.

Single variable mode is equivalent to 2D Single Axis Positioning providing one degree of freedom. The animation may be configured so that the variable provides movement in one, two or three axes.

The 3D Axis Positioning tab, when using one variable to move in three axes, might look like this. Show picture

In this example, a change in the variable MVX from 0 to 100 will move the model 1 unit in the X axis, 2 units in the Y axis and 3 units in the Z axis.

Two variable mode

Two variable mode is equivalent to 2D 2 Axis Positioning providing two degree of freedom. The animation may be configured so that each variable provides movement in one axis only or one variable provides movement in one axis and the other in two axes.

The 3D Axis Positioning tab, when using one variable to move in the X and Y axes and a second variable to move in the Z axis, might look like this. Show picture

In this example, a change in the variable MVX from 0 to 100 will move the model 1 unit in both the X and Y axes and a change in the variable MVZ from 0 to 100 will move the model 1 unit in the Z axis.

Three variable mode

Three variable mode provides three degrees of movement. There is no equivalent 2D animation.

The 3D Axis Positioning tab, when using three variables one for each of the axes, might look like this. Show picture

In this example, a change in the variable MVX from 0 to 100 will move the model 1 unit in the X axis, a change in the variable MVY from 0 to 100 will move the model 1 unit in the Y axis and a change in the variable MVZ from 0 to 100 will move the model 1 unit in the Z axis.
