Pie Chart Overview

The Pie chart displays one or more values as pie, doughnut or pyramid style chart. The values can be entered as part of the configuration, loaded from a file or set by scripting. Show picture



Scripting

The Pie Chart Control can be manipulated programmatically by both SCADA Basic and VBA.

A specific instruction, PIE, is provided in SCADA Basic. It contains many functions not available from the Pie Chart configuration dialog such as exploding a segment and combining segments depending on their value. See the help for SCADA Basic for information on its use.
