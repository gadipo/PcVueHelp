The General Tab

Show picture

The General tab is used to configure the first time a trigger is executed and to select either a single trigger or an interval begin/end trigger.

How to configure a single trigger

A single trigger executes just the Begin Trigger action from the task to which it is attached.

  1. Open the Trigger Properties dialog and select the General tab.
  2. Select the Single Trigger option button.
  3. Enter a date and time in the Begin On Date and Time fields.
  4. Click OK to confirm the configuration.

How the Data and Time field determines the trigger repeats

The trigger is executed for the first time at the date and time entered in the Date and Time fields. These fields also determine at exactly what time the repeat triggers occur.

  • Example 1 - The start date and time is set to 01/09/2003 00:30:00. A trigger with a repeat of 2 hours would next be executed at 01/09/2003 02:30:00. The minutes and seconds past the hour at which the first trigger was executed is used for all subsequent executions.
  • Example 2 - The start date and time is set to 01/01/2004 01:00:00. A trigger with a repeat of 1 day would next be executed at 02/01/2004 01:00:00. The hours, minutes and seconds in the day at which the first trigger was executed is used for all subsequent executions.

There is no relationship between the End On Date and Time in the General tab and the End On Date and Time in the Repeat tab.

  • The End On Date and Time in the General tab specifies when a task's end trigger action is to take place.

  • The End On Date and Time in the Repeat tab specifies when a trigger is to be no longer repeated.
