Resizing a symbol
 

 
 
 
 
 
 
 
 
 
 
 
 
 
Resizing a Symbol
 
You can re-size a symbol in the same way as a single drawing 
 element by selecting it, and pressing and dragging with the mouse on one 
 of the handles surrounding it. Show picture
 

 

 
 
 
 
The symbol may be resized in one or both dimensions depending 
 on where the mouse is positioned. Holding down the Ctrl key while re-sizing 
 will ensure that the X and Y dimensions retain their original ratio.

Resizing: details and limitations
 
To resize a 3D model properly, without skewing it, the 
 HMI must resize the model along one, two or all of its axes according 
 to the resizing tool and the direction of the 3D model's axes (depending 
 on the model's rotation settings).
 
The following tables provide information on what to expect 
 when resizing a 3D model. The examples show the model to be resized - 
 but not the reference model - in a window using the Top View orthographic 
 camera mode.

Resizing 3D models
 

 
 
 
 
 
 
 
 
 
 
 
 
 
Resizing 3D models
 
When you use the resizing tools, models are resized along
 the 3D world axes that are parallel to the window borders. When they are
 displayed in a window using an orthographic camera mode, a 3D model's
 height and width are defined as follows.
 
 
 
 

 
 
Dimension
 
 
 
 
Meaning
 
 
 

 
 
Height
 
 
 
Size of the right (or left) side.
 
 
 

 
 
Width
 
 
 
Size of the top (or bottom) side.
 
 
 
 
The model's sides that are used are those of a virtual
 rectangle (not displayed) that is just big enough to encompass the 2D
 projection of the model's bounding box.
 
For example, here is a window using the Top View orthographic
 camera mode. Therefore, the 3D world's X and Z axes are the ones parallel
 to the window borders. Show picture
 

 

 
 
 
 
Resizing tools
 
 
 
 
 

 
 
Tool
 
 
 
 
Icon
 
 
 
 
Resize the selected
 models so that:
 
 
 
 

 
 
Same Width
 
 
 

 
 
 
 
 
Their width matches the reference model's width.
 
 
 
 

 
 
Same Height
 
 
 

 
 
 
 
 
Their height matches the reference model's height.
 
 
 

 
 
Same Size
 
 
 

 
 
 
 
 
Their width and height match the reference model's width
 and height.
 
 
 
 

 
The tools may not always work as expected. For more information,
 please read the topic Resizing 3D models: Details and Limitations.
