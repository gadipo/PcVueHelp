GetArray Method

 See also Applies to 

Retrieves the contents of a block of cells in a string.

Syntax

Function GetArray(TopRow As Long, LeftCol As Long, BottomRow As Long, RightCol As Long, ColSeparator As String, RowSeparator As String) As String

 

Part

Description

TopRow

Identifies the top row.

 LeftCol

Identifies the left-hand column.

BottomRow

Identifies the bottom row.

RightCol

Identifies the right-hand column.

ColSeparator

The column separator.

RowSeparator

The row separator.
