The chats list

When a mimic containing a Messaging Control is opened, by default, the chats list is displayed. The chats list displays all the chats, for which the current user has permissions, plus a tool in the top left corner for adding a new chat. Show picture

Clicking a chat displays the chat view from where messages can be read and written and, if the user has the correct permissions, the chat settings can be changed.

How to add a chat

  1. Click the tool at the top left of the chats list. The Add chat dialog is displayed. Show picture


  2. Enter the Chat name. The chat name is only visible in the configuration and cannot be changed once the chat has been created.
  3. Enter the Chat title. The chat title appears as the identifier in the chats list and can be edited at run-time by a user having Owner privileges.
  4. Select the Storage time for messages. Messages are automatically discarded after this period.
  5. Select the users that will have access to the chat, and their privileges.
  6. Click the Tick button to confirm the creation of the chat. The chat view is displayed.

Once a chat has been created it can never be deleted.
