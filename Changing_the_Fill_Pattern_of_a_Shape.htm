Changing the Fill Pattern of a Shape

The fill style of a shape may be changed from the Base tab in the Drawing Properties dialog. Show picture

The following styles are available.

Solid

Diagonal –45

Transparent

Vertical

Diagonal + 45

Horizontal

Diagonal 45 & -45

Vertical & horizontal

With styles other than Solid or Transparent, the pattern color is selected using the Color property and the fill color using the Background property. With the Solid style, the fill color is selected using the Color property.

It is also possible to change the border (line) style of a rectangle or ellipse. For information on changing the line style, see the topic Drawing a line.
