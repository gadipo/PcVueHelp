Locking and Unlocking

A drawing element that has been locked cannot be moved or changed in any way while it is in Edit mode. When a locked drawing element is selected, its handles have a different appearance to a normal drawing element. In the picture below the drawing element on the right is locked whilst the one on the left is unlocked.

To lock a drawing element

  1. Display the drawing element's Properties dialog.
  2. Click on the down arrow button to display the Advanced properties.
  3. Tick the Locked tick box
  4. Click OK to confirm the change.

Or

  1. Select the drawing element.
  2. Click the Lock button on the Draw toolbar.

Or

  • Select the Lock command on the Drawing context menu.
