Data connection settings

The Data connections settings are displayed from the Settings task when the Data connections folder is selected in the Application Explorer's configuration tree. Show picture

  • Flow regulation

    • Read query

      • Suspension and Continuation- If the number of queued Sql read queries exceeds the Suspension threshold, new queries are temporarily suspended until, at a subsequent test, the number of queued Sql read queries is at or below the Continuation threshold.

    • Write query - Similar to the equivalents settings for the Sql read queries
