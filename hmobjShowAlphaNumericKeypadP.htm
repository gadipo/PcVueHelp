ShowAlphaNumericKeypad Property

              

Description

Displays the alphanumeric keypad and accepts an input value.

Syntax

Function ShowAlphaNumericKeypad(Value As String, [Mode As fvAlphanumKeypadOpeningMode = fvAutoLayout], [Caption As String], [Password As Boolean = False], [DisplayInitialValue As Boolean = True]) As Boolean

Part

Description

Value

Input value.

Mode

Format of the input value (see enumeration below).

Caption

Title for the input dialog box.

Password

Whether to display the input value in clear or as '*' characters.

DisplayInitialValue

Whether to display the previous value of the variable.

Enumeration of fvAlphanumKeypadOpeningMode

Value

Description

fvAutoLayout

Auto layout

fvFrenchLayout

French layout

fvGermanLayout

German layout

fvItalianLayout

Italian layout

fvRussianLayout

Russian layout.

fvSpanishLayout

Spanish layout.

fvStandardAzerty

Standard Azerty layout.

fvStandardQwerty

Standard Qwerty layout.

fvUSLayout

US layout.
