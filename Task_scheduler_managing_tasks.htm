Managing Tasks

A list of all configured tasks and their triggers is displayed in the lower pane of the Task Scheduler.

  • The schedule of triggers for a task is displayed in the calendar when the task is selected with a single click.
  • The Task Properties dialog, from where a task's properties may be displayed and edited, is displayed by double clicking the task.
  • A task may be temporarily disabled by un-ticking the box adjacent to the task name.

The task pop-up menu

The task pop-up menu is displayed by right clicking on a task in lower pane of the Task Scheduler. The following commands are supported.

  • Edit task - Displays the Task Properties dialog from where the task properties can be displayed and edited.
  • Add trigger - Displays the Trigger Properties dialog from where a new trigger can be created.
  • Paste trigger - Attaches a previously copied trigger to the task.
  • Show next trigger - Scrolls the calendar to show the text occurrence of a trigger for that task
  • Delete task - Deletes the task from the Task Scheduler.
