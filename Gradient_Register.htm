Understanding the Gradient Register Animation

You can configure a Gradient Register animation as follows:

  1. Select the drawing element whose color is to be made to vary.
  2. In the main menu, select Animate.Color.Gradient Register to open the Drawing Properties dialog with a new tab for Color Gradient Register. Show picture

  3. In the Register box, select the variable whose values are to be used to determine the colors.
  4. In the Line (or Text) section, with the Range box ticked, click on the Start color to select a color to represent the lowest value of the variable for gradation. See note on 'Value Limits' below.
  5. Click on the End color to select a color to represent the highest value of the variable for gradation. See the note on value limits below.
  6. If required, set the other properties as described in the table below.
  7. Click on OK to close the dialog and apply the settings.

To apply the values in the Line section only, un-tick the boxes in the Pattern section. To apply the values in the Pattern section only, un-tick the boxes in the Line section.

If you apply Value Limits, the values outside of those limits are represented by the static colors you specify for them. Otherwise, the graded colors apply to the whole range of values of the register.

Ranges of colors

The color for a value of the variable is determined by the value's position in relation to the lower and higher value limits that you configure. That proportion of the scale of values is applied separately to the red, green and blue (RGB) color differences between the Start and End colors. The result of those three components gives the color to display.

A simple use of this tool is to show shades of brightness for a particular color. In the example above, the gradation covers shades between the lightest and darkest on the row of green colors on the Color Palette. Show picture

If you choose different colors for the Start and End limits, the effect of gradation will not seem linear. The intermediate colors may look quite different to either end of the range, or (for a range of values) too similar to one of them. Show picture

The dark gray is the middle color between the yellow and the blue.

You can only use plain colors for the Start and End colors of a Gradient Register animation, not gradient or blinking colors.

You can configure the animation with a quality sequence from the Sequence Manager.
