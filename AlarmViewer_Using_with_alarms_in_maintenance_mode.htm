Using the Alarm Viewer with Alarms in Maintenance Mode

Maintenance Mode is used to modify the behavior of an alarm so that equipment maintenance can be taken into account. A typical scenario where maintenance mode might be used follows.

  1. An alarm occurs due to equipment requiring maintenance.
  2. The User acknowledges the alarm and calls the maintenance department.
  3. The User puts the alarm in maintenance mode. This action can be logged showing the condition has been recognized by the User.
  4. The maintenance department carries out the required action and resets the cause of the alarm so that the alarm is no longer present (Alarm - off).
  5. The User removes the alarm from maintenance mode.

The new alarm behavior may be summarized as follows.

  • The alarm now has only two possible states, Alarm on - Ack or Alarm off.
  • The alarm is only visible in an Alarm Viewer with the Maintenance option selected.
  • Changes of state of the alarm are no longer logged or printed.

How to set Maintenance Mode on an alarm

An alarm may be set to maintenance mode using an Alarm Viewer by first selecting the alarm and then clicking the Start Maintenance button on the tool bar. An alarm may also be set to maintenance mode using a SCADA BASIC program. An alarm may only be set to maintenance mode when it is in the Alarm On - Ack state.

How to clear Maintenance Mode on an alarm

An alarm may be cleared from maintenance mode using an Alarm Viewer by first selecting the alarm and then clicking the Stop Maintenance button on the tool bar. An alarm may also be cleared from maintenance mode using a SCADA BASIC program. An alarm may only be cleared from maintenance mode when it is in the Alarm Off state.

How to display a list of Alarms in maintenance mode

You can display a list of alarms in maintenance mode in an Alarm Viewer with the maintenance property selected. This is selected either at design time by selecting the Maintenance property in the Filter tab, or at run time using the Maintenance button on the toolbar or Maintenance property in the Alarm Filer dialog.

Alarms that have maintenance mode selected can be in one of two states, Alarm on - Ack or Alarm off.

Logging maintenance mode

Changes to alarms that are in maintenance mode are no longer logged. However, the action of selecting or clearing maintenance mode is logged and treated if a log list is configured to log such user actions in the same way as acknowledging or masking an alarm.
