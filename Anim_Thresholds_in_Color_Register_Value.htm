Understanding the Thresholds Used in Color Register Value

In the animation Color - Register Value you select a register and enter up to 8 threshold values and corresponding colors. The threshold values must be in ascending order. Each threshold determines the point at which a color change takes place. This is best explained by example.

Example

A register variable has a range of values from 0 to 29. If the option Reg.out of range is selected:

Range for color

Register value

Color

Threshold 1 = 10 Color = Blue

Less than 10

Blue.

Threshold 2 = 15 Color = Green

10 or more but less than 15

Green.

Threshold 3 = 20 Color = Yellow

15 or more but less than 20

Yellow.

Threshold 4 = 30 Color = Red

20 or more but less than 30

Red

Reg. out of range Color = White

30 or more

White

If the option Reg.out of range is not selected then values of 20 or more will be displayed in red.
