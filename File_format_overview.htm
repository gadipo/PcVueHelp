File Format Overview

The Import/Export Tool produces text files in CSV format (character separated value, where the separator is a comma or a semicolon throughout a file) or in TSV (tab separated value) format.

All files have the same basic structure with three groups of information. Each of these is made up of one or more lines, each line representing a specific item. The three groups of information always appear in the same order.

The tool saves an existing file using the original delimiter. However, it can only create a comma delimited text file.

  • Language information - An index of the languages used including standard language codes. Language information lines start with the identifier LANG.
  • Style information - An index of the font style used in the strings. Style information lines start with the identifier FONT.
  • String information - Details of each text string, in one or more languages. String information lines start with the identifier MIMIC, TEMPLATE or SYMBOL.

The following extract shows comma as the delimiter.

LANG,1,"ENG-GBR",2057,0
LANG,2,"FRA-",12,0

FONT,1,-21,0,400,0,0,"Arial",0,1,37
FONT,2,-21,0,400,0,0,"Arial",0,1,36

MIMIC,"mimic embedded",title,0,"Mimic2",1,0,0,"",1,0,0
MIMIC,"mimic embedded",link(1),0,"",1,0,0,"",1,0,0
MIMIC,"mimic embedded",link(2),0,"",1,0,0,"",1,0,0
