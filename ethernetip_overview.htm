Ethernet/IP Data Acquisition Driver Overview

The Ethernet/IP Add-on enables communication with PLCs supporting the Ethernet/IP standard. This driver supports communication with:

  • Rockwell Automation ControlLogix and CompactLogix PLCs.

  • Allen Bradley SLC500 PLCs.

  • Any other PLC supporting CIP explicit messaging (Common Industrial Protocol), with addressing based on symbolic elements.

There are no configuration dialog's specific to the Ethernet/IP Data Acquisition Driver. At the time of writing, it uses a combination of dedicated text files (.dat) and regular configuration features.

This driver supports access to atomic tags in:

  • Tags.

  • User Defined Data Types (UDT).

  • Arrays (atomic and UDT).

The Ethernet/IP Data Acquisition Driver Add-on is licensed and available on order. Contact your local reseller for more information.
