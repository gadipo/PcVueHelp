Using Animated GIF and AVI Files

Animated GIF or AVI files produce an animated image by playing a number of frames at a predetermined interval. The number of frames and the interval are fixed when the image is designed.

Images of this type have an extra tab called Settings in the Image Properties dialog to control their behavior at run time. Show picture

Section

Setting

Effect

Playback

Repeat continuously

Select to play the image repeatedly.

 

Play ... times

Set the number of times to play.

Content

All frames

Select this to play all frames.

 

From frame ... to ...

Play only this range of frames, inclusive.

Frame delay

Automatic

Play the frames at the default rate.

 

Manual

Display each frame for a specified interval before displaying the next one.

Others

Start playing automatically

Play as soon as the mimic opens.

 

Rewind When Stopped

When play is stopped, the first frame played is shown; otherwise, the last frame played is shown.

 

No sound

For AVI files: turn the audio off.

 

Hidden when not playing

Only show the image while it is playing.

 

Hide image only...

When the image is hidden, continue to display the surround (appearance, line and pattern).

 

Play on mouse hover

Only play the image while the cursor is held over it.

You can combine these options, e.g. always to show the surround, but make the image appear and play only when the cursor is over it.
