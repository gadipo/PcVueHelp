TouchVue Settings View

The Settings view is used to manage the app's general settings. The settings are divided into two categories, General Show picture and Alarm notifications Show picture. The settings are persistent.

General settings

  • Time filter properties
    • Log lists - The default time period for the List of logged events view.
    • Trends - The default time period for the Trend view.
  • Settings
    • Variables refresh interval - The period, in seconds, at which variable values are polled when the Watchlist view is open.
  • Theme
    • Theme color - The main app theme color, selected from a predefined palette. The default is red.
  • Audit
    • Diagnostic - Enable access to diagnostic to assist with troubleshooting, for example network name on the connection view.
  • Version info - The version of the installed TouchVue.

Alarm notification settings

  • Notifications

    • Mute alarm notifications - Mute, or not, alarm notifications. The default is Off (no muting).

    • Start the alarm notification service at boot time (Android app only) - Start the alarm notification service at device boot time. This means that, once logged in, alarm notifications will be displayed, even after a device restart, without the need to restart the app. The default is On.
  • Alarm properties

    • Change display order - Changes the order in which alarms are displayed - either the newest, or the oldest, at the top.

    • Polling interval for alarm list refresh - Period in seconds between requests for alarm list updates.
    • Ringtone (iOS only) - The ringtone associated to notifications.
