Selecting a Blinking Color
 
Some drawing elements support blinking colors. A blinking 
 color contains two color selections. At Run time, the affected part of 
 the drawing element will alternate between them at a pre-defined rate. 
 Show picture
 

 

 
 
 
 
How to select a blinking color
 
 
Select the Fill Mode 
 as Blinking.
 
Click the Primary 
 Color and use the Color Picker dialog to select it.
 
Click the Alternate 
 Color and use the Color Picker dialog to select it.
 
Select which of the 
 blinking periods to use. There are 16 configurable blink rates.
 
Click OK to apply 
 the color.
 
 
The blinking rates are configured from the main menu: 
 Show picture
 

 

 
 
 
 
 
Select Configure.Preferences.Blinking 
 to open the Blinking Color and Visibility dialog. The colored panel 
 blinks to show the rate of blinking.
 
Move each slider 
 to set the period for which its color is to be displayed.
 
Click OK to apply 
 the settings.
 
 

 
In the Blinking Color and Visibility dialog, the colors 
 that show the blink rates do not affect the colors used in an animation.

Selecting a Gradient Color
 
A gradient color contains two or more colors. The affected 
 part of a drawing element will be shaded starting with one color gradually 
 changing to the other colors across the drawing. The type of gradient 
 may be selected from either spot or linear. The following shows some examples 
 of the use of gradient colors.

How to select a color with a spot gradient
 
 
Select the Fill Mode 
 as gradient.
 
Select the Type as 
 spot.
 
Click the one of 
 the small color boxes on the ends of the gradient control bar above the 
 color circle and select its color using the Color Picker dialog in the 
 normal way.
 
Repeat for the other 
 color box.
 
Select the position

Select the center

 as Gradient.
 
Select the Type as 
 linear.

Repeat for the other

How to add additional colors when using a gradient
 
By default, a gradient uses two colors. However, if necessary, 
 it is possible to add additional colors up to a maximum of eight.
 
 
Select 
 the Fill mode as gradient.
 
Click 
 and drag the new color box icon to the left of the gradient control onto 
 the gradient control.
 
Select 
 the color for the new color box in the normal way.
 
Select

How to remove additional colors
 
 
Click the color box 
 to be removed and drag it over the new color box icon.
 
 
Performance Considerations
 
The use of gradient colors is very attractive but, when 
 using them, it is important to note that the speed at which they are rendered 
 will depend on the choice of graphic adapter and the color configuration.
 
 
Step -Make the Step 
 parameter are large as possible. The drawing performance depends directly 
 on the size of the Step. For example, a step of 4 will render twice as 
 fast as a step of 2.
 
Linear gradient - T the angles 0, 90, 180 or 270 are most efficient.
 
Radial gradient - M more time consuming than linear, particularly if the X and Y co-ordinates 
 are not close to the object's center (50%, 50%)
 
Number of colors - 
 This has no impact on the drawing performance.

Using the Palette to color a drawing
 

 
 
 
 
 
 
 
 
 
 
 
Using the Palette to Color a Drawing
 
The Color Palette may be used to color a drawing directly 
 or by changing the colors using the drawing's Properties dialog.
 
To change a drawing's color directly
 
 
Open the Color Palette.
 
Select a drawing 
 by clicking on it with the left mouse button. The Palette will display 
 the drawing's current colors in the color selection area.
 
To change the color 
 of the fill (or background of text), click on the new color with the left 
 mouse button.
 
To change the color 
 of the border (or text), click on the new color with the right mouse button.
 
 

 
If you select more than one drawing, the color changes 
 will be applied to all the selected drawings.
 
 
To change a color in a drawing's Properties dialog
 
 
Display the drawing's 
 Properties dialog.
 
Click in the color 
 to change - the Color Palette will be displayed.
 
Click on the color 
 in the Palette with the left mouse button.
 
If you want the color 
 to blink, select the second color with the right mouse button.
 
 
To copy the colors from an existing drawing
 
 
Open the Color Palette.
 
Click on the drawing 
 element from which the colors are to be copied.
 
Subsequent drawings 
 will be initially created using these colors.

Overview of the Color Palette
 

 
 
 
 
 
 
 
 
 
 
 
 
 
Overview of the Color Palette
 
The Color Palette is displayed each time you select 
 a color property from any drawing element's Properties dialog and may also 
 be displayed using the Display.Color Palette command. Show 
 picture
 

 

 
 
 
 
It is essential to understand the use of each area of 
 the Color Palette before starting any drawing.
 
 
Selected colors. 
 The colors currently in use.
 
Transparent color. 
 Click here to choose one of the selected colors as transparent. When you 
 use a transparent color for a drawing, the background will be seen through 
 it.
 
Non-indexed colors. When 
 you color a drawing using one of the colors from this area, the RGB code of the color is 
 saved in the drawing. There are 248 non-indexed colors in the palette. If you change the color in the palette, the color 
 in the drawing is not affected.
 
Indexed colors. When 
 you color a drawing using one of the colors from this area, the color index in the color palette is used as a reference. If you change the color in the palette, the color 
 of all drawings using its index will be affected. There are 248 indexed colors in the palette.
 
Scroll bars. Used to scroll through the range of available colors. 32 non-indexed and 32 indexed colors are visible at any one time.
 
 
To differentiate an indexed color from a non-indexed color a small icon appears in the lower right corner of the color swatch in the drawing or animation properties dialog. Show picture

 
 
If you click on a color swatch that contains an indexed color then the corresponding color in the palette is highlighted.
 
If you double click on a color swatch that contains an indexed color then the Color Picker dialog opens. Clicking OK in the Color Picker dialog closes it and the color in the color swatch is changed from indexed to non-indexed (same RGB).
 
If you right click the small icon in the lower right corner of an indexed color swatch, the color is changed from indexed to non-indexed (same RGB).
 
 
Changes in the color palette after version 10.0 of the PcVue software
 
 
There are 248 non-indexed and 248 indexed colors. The color palette of version 10.0 and earlier had 64 colors in total. 
 
Several of the non-indexed colors (33 to 96) are pre-configured as gradients.
 
Indexed colors 209 to 248 are used in PcVue's pre-configured libraries. They should not be changed by the user if the pre-configured libraries are used.
 
 

 
Unless otherwise stated, it is good practice to use indexed colors and document their use as part of the project specification (for example, Index 1 for mimic background, index 2 for button background, index 3 for �).
In particular, it saves engineering time if late changes in colors are required at the time of commissioning or to take into account users� feedback. If correctly documented and used the only change required will be in the color palette. No change will be required in the mimics and symbols you have already created.

Color Picker dialog
 

 
 
 
 
 
 
 
 
 
 
 
 
 
The Color Picker Dialog
 

 
Do not confuse the Color Picker dialog with the Color 
 Picker and Color Dropper tools.
 
 
Colors in the Palette are changed using the Color Picker 
 dialog. To display the Color Picker dialog, double-click on the color 
 in the Palette that you want to change. Show picture
 
There are three color styles available.
 
 
Solid - A single color.
 
Blinking - A blinking 
 color contains two color selections. The affected part of the drawing 
 element will alternate between the two colors at a pre-defined rate.
 
Gradient - A gradient 
 color contains between two and eight colors. The affected part of a drawing 
 element will be shaded starting with one color and gradually changing 
 to the other colors across the drawing.
 
 

 

 
 
 
 
The Color Picker dialog is also displayed when you double 
 click on a color box in any Drawing Properties dialog.
 
A color is defined by the three components of hue, saturation, 
 and brightness.
 
 
Hue describes the 
 color's shade or tint. It is measured on a circular spectrum running from 
 red to green to blue and returning to red. Hue is selected on the color 
 circle by the angular position of the color pick dialog with respect to the 
 center of the circle.
 
Saturation describes 
 the hue's purity. A color with a saturation of 255 is bright and vivid, 
 and a color with a saturation of 0 is a shade of gray. Saturation is selected 
 by the distance of the color pick dialog from the center of the color circle.
 
Brightness describes 
 the color's brightness. A color with brightness of 0 is black.

Applying a color animation to a 3D model
 

 
 
 
 
 
 
 
 
 
 
 
 
 
The 3D Color Animation Tab
 
Each type of 3D Color animation tab is very similar 
 to those used in 2D animation, the main difference being the addition 
 of a combo-box to select the materials to which the animation is applied, 
 and a list box to select which color properties are shown in the tab. 
 
 
How to use the Materials combo-box
 
When configuring a color animation you can select one 
 or more materials. The Materials combo-box has a non-standard behavior 
 to allow multi-selection capability. Show picture
 

 

 
 
 
 
 
The combo-box displays 
 all selected materials with each material separated by a semicolon.
 
The drop-down list 
 is not automatically closed when you click on an item. Instead, the item's 
 selection state is reversed. In that way, you can select (or de-select) 
 several items before closing the list. 
 
To close the drop-down 
 list, click once outside of it.
 
 
The materials available for selection depend on the level 
 at which the color animation is being applied.
 
 
 
 

 
 
Level
 
 
 
Materials available for selection
 
 
 

 
 
Symbol or group
 
 
 
No individual materials are available for individual selection. 
 The combo-box contains one item, All Child Object Materials, which allows 
 the animation to be applied to all materials in all models.
 
 
 

 
 
Model
 
 
 
All materials available in the model. 
 
 
 

 
 
Frame without a mesh
 
 
 
No individual materials available for individual selection. 
 The combo-box contains one item, All Child Object Materials, which allows 
 the animation to be applied to all materials in all sub-frames.
 
 
 

 
 
Frame with a mesh
 
 
 
All materials available in the frame, plus an additional 
 item, All Child Object Materials, which allows the animation to be applied 
 to all materials in all sub-frames.
 
 
 
 
How to select the color properties displayed in the 
 tab
 
A single color animation can change all properties provided 
 by a material. In order to keep the size of the Color Animation tab manageable, 
 the properties displayed at any one time are selected by a list box. Using 
 the list box, you can choose to display:
 
 
Diffuse color and 
 opacity
 
Specular color and 
 level
 
Emissive color
 
 
As a design aid, once you have selected and changed properties 
 in any one of the categories, its entry in the list box is displayed in 
 bold text. Show picture
