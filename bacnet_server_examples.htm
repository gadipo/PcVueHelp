Examples

Analog input object

Consider the following Analog Input Object configuration.

  • Object Name = TemperatureSensor1
  • Instance Number = 1
  • COV Increment = 0.1
  • Description = Main Hall Temperature sensor
  • Units = 62 (Celsius Degree)
  • Min Pres Value = -200
  • Max Pres Value = 200
  1. Object declaration. The Name attribute is the only one relevant so far and its value TemperatureSensor1 will be transferred to the Object Name at the next start-up of the project. Show picture

  2. From the Extended Text Attribute, the Object Identifier Property will be transferred at the next start-up project. The Object Type will be set to Analog Input (AI) and the Instance Number to 1. Show picture

  3. The project is restarted again and this object was created. The following properties are now available. Show picture

    The Description property of the variable sets the Description property of the BACnet object to "Main Hall Temperature Sensor".
    The string “°C” is matched to the BACnetEngineeringUnit numeric value 62 and transferred to the Units Property.
    The Minimum value set the Min Pres Value Property and the Maximum value set the Max Pres Value

  4. The Auto Completion String was BACnet:AI:1::::::COV_I and the COV Increment is set to 0.5. Show picture

Binary Input Object

Consider the following Binary Input Object configuration:

  • Object Name = ReedSwitch
  • Description = Emergency Door Open State Switch
  • Instance Number = 11
  • Active Text = Emergency exit #1 open
  • Inactive text = Emergency exit #1 close
  1. Object declaration. The Name attribute is the only one relevant so far and its value TemperatureSensor1 will be transferred to the Object Name at the next start-up of the project. Show picture
  2. From the Extended Text attribute, the Object Identifier property will be transferred at the next start-up. The Object Type will be set to Binary Input (BI) and the Instance Number to 11 (11). Show picture
  3. The Description attribute sets the Description property to Emergency Door Open State Switch. Show picture
  4. The Auto Completion String was BACnet:BI:11:::::ACT_T:INA_T. The Active Text set to "Emergency exit #1 open". The Inactive Text set to "Emergency exit #1 close". Show picture
