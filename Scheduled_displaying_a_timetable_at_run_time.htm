Displaying a Timetable at Run Time

Time Table may be displayed at run time using a special animation called Timetable. The colors in which the time table are displayed are defined by the Time Table and Exception Period attributes.

A Timetable display may be configured to be read only or to allow the Operator to change and save the time periods. For further details on how a Time Table is displayed at run time see the animation section in the Window Editor manual. Show picture

The WebScheduler also permits visualizing timetables and modifying time periods.
