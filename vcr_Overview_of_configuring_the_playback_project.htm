Configuring the Playback station

The project that runs is the same on the station that records VCR data and on the playback station, with server and client lists configured accordingly on the VCR archive unit.

The following configuration is required on the playback station:

Configuration

Settings

Command line

Add the -VCR switch to the command line used to start PcVue. For example:

C:\SV32\bin\sv32.exe -vcr -b C:\SV32\Projects

VCR type archive unit

The playback station must be in the list of clients of the VCR archive unit, and the property Path to files from dedicated station, must be configured properly. Normally this will point to the path of the archive unit on the recording station.

It is recommended that only client stations be used as playback stations.

Because variables on playback stations are simulated, their value no longer reflect the current process state. No station requiring availability of live data should be used as playback station. In particular, a data acquisition server, an archive server or any other station due to distribute live data to other sub-systems (web back end, OPC server...) must not be used as playback station.

Opening the VCR Player Data Dialog

The VCR Player dialog is opened by the user from a drawing element (such as a button) in a mimic. To create such an animation:

  1. Create a drawing element.
  2. Right-click and select the animation Run.Macro to apply a Macro animation.
  3. Put the single instruction IVCR_Open in the first line of the macro. Show picture

  4. Select OK to confirm the action.
  5. Select File.Save All to save the mimic.

To open the VCR dialog:

  1. Select Run mode.
  2. Select the drawing element. The macro will open the VCR Player Data dialog.

If that does not happen, check that you have applied the command line argument -VCR as described above.

Using the SYSTEM.VCRMODE Variable

The register variable SYSTEM.VCRMODE is used to indicate to the User the mode in which PcVue is running:

Value

Mode

0

Normal mode

1

VCR mode

The variable is set according to the mode in which PcVue was started. It does not imply that a VCR archive unit has been created.

Limitations in the Playback Project

The VCR facility Player plays back recorded values through the variables at a speed that is not related to real time. Therefore, the following animations will not display anything meaningful and should not be used on a playback station.

  • The Trend Viewer.
  • The Log Viewer.

These viewers should not be used for playback.
