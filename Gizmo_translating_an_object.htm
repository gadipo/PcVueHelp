Translating a Model

This topic describes how to use the Gizmo tool to move a model along a line.

  1. Select a model.
  2. Click on the Gizmo's green bar (the shaft rather than the arrow point). The arrow is highlighted in orange to indicate that it is selected.
  3. Drag the mouse. The model moves along the selected axis.
  4. You can use the Gizmo's other bars to move (translate) the model along its other local axes.
