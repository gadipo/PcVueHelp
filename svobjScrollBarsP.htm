Scrollbars Property

 Applies to 

Description

Sets or gets the visibility mode of the scrollbars. Read/Write.

Syntax

object.Scrollbars As ScrollBarMode

where ScrollBarMode takes these values:

Value

Effect

sbNone

Never display the scrollbars.

sbHorizontal

Always display the horizontal scrollbar.

sbVertical

Always display the vertical scrollbar.

sbBoth

Always display both scrollbars.

sbAutomatic

Display scrollbars automatically when all cells cannot be displayed at once.
