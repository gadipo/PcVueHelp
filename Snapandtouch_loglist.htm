Logged events list view

The Logged events list view displays a list of events from a selected Log List. The events can be further filtered using an optional filter. Show picture

Screen features

  • The displayed events are filtered by Log List and an optional filter.
  • The events displayed are limited by a time period set by the Timeframe Properties setting in the Settings view. The default is 6 hours.
  • Oldest events are displayed at the bottom, newest at the top.
  • The list can be scrolled using the swipe or drag gestures.

Menu actions

Select the Log List when multiple Log Lists have been added to the Mobile Server node in the Application Explorer.
Change the time period. You can choose from a number of preset periods, which run back from the current time, or custom period where the period is defined by entering a start and end time.
Filter the type of events that are displayed. Note that the filter set here is ANDed with the filter from the PcVue Log List.
