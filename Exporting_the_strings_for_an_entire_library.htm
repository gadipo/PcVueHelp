Exporting the Strings for an Entire Library

This process exports the strings from an entire library (local, common or custom).

How to export the strings from one or more libraries

  1. Open the Import/Export Tool using the menu command Display.Translation Import/Export.
  2. If you have not already done so, configure the file paths and export languages. See the topic Configuring the Import/Export tool.
  3. Select the library or libraries for export. You can select more than one library by clicking while the Ctrl key is held down.
  4. Start the export process using either the Export-NEW or Export-UPDATE button.
  5. The export function processes all the mimic, template and/or symbol files in the selected libraries. For large projects, this may take several seconds to run.

If any mimics are open at the time of the export the process will display a warning dialog then automatically close them.

Any changes you make to the Import/Export Tool configuration are only temporary unless you click the Save button.

Canceling the export process

You can cancel the export process at any time during its execution by clicking the Export button again (which is renamed Stop during the export process). The export will stop but changes made to the text files prior to clicking Stop are not undone.

What is the difference between Export-NEW and Export-UPDATE?

  • Export-NEW - Creates new import/export files. If a file of the same name already exists you will be asked whether you want to overwrite it or cancel the export
  • Export-UPDATE - Updates entries in existing import/export files and adds any new information. Items that have been deleted from the project since the previous export are not deleted from the text files.

Checking the Status Log

When the export process has completed, you should check the status log in the lower pane of the Import Export Tool dialog for any errors. Show picture

If you want a permanent record of the process or want to search for something in particular, you can copy the contents of the status log by using Ctrl+A and Ctrl+C and then pasting it into a text editor such as Windows Notepad.
