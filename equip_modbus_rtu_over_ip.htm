Modbus RTU over Ip

This driver supports communication over Ip with any compatible field device using Modbus RTU frames encapsulated into TCP packets.

Driver status

The driver status provides driver-specific information to supplement the frame general status. For more information on status in general, refer to the topic General communication status.

Driver status

Description
0x0102

Connection closed because of:

  • Incorrect IP address

  • Incorrect port number

  • Field device not connected

0x0005

Acknowledged - The slave has accepted the request and is processing it, but it will take a long time

0x0006
