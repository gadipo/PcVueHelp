Sets the internal cell pointer to the next selected cell if any.

Syntax

Function NextSelectedCell() As Boolean

Returns True if there is another cell in the selection, otherwise False.
Calling this function makes an internal copy of the current selection for later enumeration.
If the selection changes, you should call this method again.

See also the SelectedCellRow and SelectedCellCol properties to retrieve the cell indicated by the internal pointer.
