Using a Touch Screen

PcVue has a number of features designed to make the use of a touch screen easier and more ergonomic for the user.

Soft numeric and alphanumeric keypads

The soft (screen based) keypads are used for number and text entry with the following animations.

  • Send Register animation
  • Send Text animation
  • Security animation
  • Security - Login
  • Security - Password modification
  • Security - User confirmation
  • Security - Double signature
  • Trend viewer trace toolbox
  • Log Viewer

To find out more about configuring the soft keypads, see the topic Configuring the Soft Keypads.

To find out more about enabling the use of a soft keypad in the various animations, graphic controls and viewers, see the topic Soft Keypad Management.

Trend viewer display configuration

The Trend Viewer has a configuration tab specific to the use of a touch screen. Using the properties on this tab, you can choose to display large toolbars, large dialogs and large scroll bars. See the topic Configuring the Trend Viewer's Touch Screen Features for more information.

Other useful features

  • You can use an accelerator key with the majority of send animations.
  • You can disable the use of the mouse using the property Disable mouse click on any of the animations supporting an accelerator key.

Limitations

The following is a list of some of the limitations you may encounter when using a touch screen.

  • The Send - Recipe animation can only be used with the direct send property selected.
  • The Send - Region animation can only be used with the direct send property selected.
  • The Link - Note animation does not support accelerator keys and can only be used with the read only property selected.
  • The properties displayed in the legend of a Trend Viewer must be read-only.
