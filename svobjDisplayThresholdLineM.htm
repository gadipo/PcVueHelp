DisplayThresholdLine Method

              

Description

Show/Hide the horizontal line associated with the threshold value.

Syntax

DisplayThresholdLine(ThresholdNumber As Integer, Display As Boolean)

Part

Description

ThresholdNumber

The number of the threshold. Range 0 to 3.

Display

True to show line, False to hide it.

Example

Private Sub Shape1_Click()
  Dim c1 As TrdCurve
  Set c1 = Trend1.GetCurve(1)
  b = icurve.IsDisplayThresholdLine(1)
  icurve.DisplayThresholdLine 1, Not b
End Sub
