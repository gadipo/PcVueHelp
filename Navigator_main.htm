The Main Navigator

The main navigator is available in two variations known as the Navigator and the Runtime Navigator. Both take the form of form of a modeless dialog in which a thumbnail picture of the current active mimic window is displayed. Show picture

The reason for two navigators is so that one can be configured for design-time with full capabilities and the other for run-time with restricted capabilities. The main navigator can be opened from the menu command Display.Navigator or Display.Runtime navigator depending on which variation is required. Both navigators can also be opened automatically, see description of the properties below.

The main navigator supports the following actions (depending on the configuration).

  • Panning by moving the viewfinder with the mouse.
    • Left click, hold and drag.
    • Left click followed by using the mouse wheel scrolls vertically.
    • Left click followed by using the mouse wheel, and holding the shift key, scrolls horizontally.
  • Panning by using the command buttons located on the sides of the thumbnail.
  • Zooming by resizing the viewfinder. The viewfinder is resized by clicking on any of the four small handles on its border. The original aspect ratio is maintained so dragging any of the handles resizes both horizontal and vertical dimensions.

Navigator configuration

The navigator is configured using the Workspace properties dialog. There are two tabs, one for the Navigator and the other for the Runtime navigator. The available properties are identical except for those in the Runtime mimic display group, which are only available to the Runtime navigator. Show picture

  • Display at startup - Automatically open the navigator when PcVue starts.
  • Position - The position of the top left hand corner of the navigator when opened automatically at start-up. If Set at startup is not selected the navigator is opened at its previous position on startup.
  • Size - The size of the navigator when opened automatically at start-up. If Set at startup is not selected the navigator is opened at its previous size on startup
  • Operations
    • Control panning - If selected enables the panning buttons displayed on each side of the thumbnail.
    • Mouse panning - If selected enables panning by clicking and dragging the viewfinder.
    • Zooming - If selected enable zooming by dragging the handles on each side of the viewfinder.
  • Style - Configure the style properties of the window in which the navigator is displayed. Similar to those of a mimic window. See the topic Defining the window style for an explanation of the individual properties.
  • Runtime mimic display - Properties available for the runtime navigator only.
    • All mimics - The navigator displays all mimics.
    • Designated mimics - The navigator displays designated mimics only. That is mimics with the Mimic to be displayed in designated mimics mode property set.
      • Open/close navigator automatically - Open and close the navigator window automatically depending on if the active mimic has the Mimic to be displayed in designated mimics mode property set.

The colors used for the navigator borders are configured in the color preferences. See the Color Preferences topic.
