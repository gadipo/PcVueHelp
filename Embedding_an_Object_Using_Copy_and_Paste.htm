Embedding an Object Using Copy & Paste

  1. Open the application (Word, Excel etc.) with which the object is associated.
  2. Open the application file.
  3. Copy all or part of the application file to the Windows Clipboard (usually done using the Copy command on the Edit menu or Ctrl+C on the keyboard).
  4. Close the application.
  5. Open or select the HMI window in which the object is to be embedded.
  6. Paste the contents of the Clipboard into the window (using the Paste command from the Edit menu or Ctrl+V on the keyboard).
