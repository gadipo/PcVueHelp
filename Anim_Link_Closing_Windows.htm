The Link Close Animation

The Link Open animation may be applied to any drawing element although normally it will be applied to a text string with the text used as an indication to the user. Show picture

You specify the name of the window to be closed in the Window field. If the window has a Branch then it also must be specified.

If you use the substitution string * as the window name, the animation will close the window in which it resides. This is very useful when including the animation in a generic graphic object which may be inserted in any window.
