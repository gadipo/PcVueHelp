BREAK

See also

Force an exit from a block of instructions.

WebVue support - Yes.

Syntax

BREAK;

No return.

Execution

Forces an exit from a block of instructions.

Example

SUB main()

DIM i As Integer;

For (i=1;i<1000;i++)

    If (i>500) Then

       BREAK;

    End If

Next

 

PRINT("value = ",i);     

'value = 501

END SUB
