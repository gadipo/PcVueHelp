Configuring a Security devices Network

A Security devices Network object in PcVue configuration represents and provides access to a network of security devices. It allows grouping together security devices for which connections shall be handled by the same PcVue server or association of servers.

How to add a network

  1. Select the Security devices node in the Application Explorer and select Add a network from the task list. The Security devices Network dialog opens. The following screenshot was taken with the default configuration. Show picture


  2. Enter the name of the network in the Name text box. This is the name by which the network will be known in PcVue. You can accept the suggested default or enter one of your own.
  3. Enter a description of the network. The description appears in the contents pane of the Application Explorer and can be useful to identify the network later. The description is optional.
  4. Click on OK to confirm the configuration and close the dialog.

The Advanced tab

  • Networking
    • Clients - If the network is part of a multi-station application, select the name of the Clients List. Only stations in this list will be able to connect to the cameras associated with the network.
