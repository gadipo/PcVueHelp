Using Expressions in an Animation

Instead of using a variable name in an animation, you can use an expression containing operators, functions and variables.

The expression can be entered directly into the field normally used for the variable name, or it can be entered using the Expression Manager in which case a mnemonic referring to the expression model is entered in the variable field. An expression is differentiated from a variable name by immediately (no space) prefixing it with an equals sign. Show picture

For information on expression syntax including how to use the expression editor, functions, operators and the use of variables, see the topic Expression Engine in the Application Explorer help.

It is recommended to enclose text values with quotation marks, and it is mandatory for the expression to work as expected if the value contains a special character such as a space, a comma or a quotation mark. For example: StrRemove('ab,cdefgh', 3, 2).

Where is it not possible to use expressions?

  • In any animation where a user enters a value at run-time.
  • In the Trend Viewer.
