How to Draw a Polyline, Polygon, Bézier Curve or Bézier Shape

  1. Select the Polyline, Polygon, Bézier curve or Bézier shape button from the Draw toolbar.
  2. Position the cursor on the screen where the drawing is to start, and then press and release the mouse button.
  3. Move the cursor to the next node, and then press and release the mouse button.
  4. Repeat step 3 until the last but one node is drawn.
  5. Move the cursor to where you want to put the last corner and click the right button to complete the drawing.

The fill pattern of a polygon or Bézier shape may be changed in the same way as for a rectangle or ellipse. See the topic Drawing a Shape for further information.

A polyline does not enclose an area, so its Pattern properties are not used.

You can use the HMI Options dialog to configure how these shapes can be selected, as described in the topic HMI Options.General Options.

Positioning the points using the keyboard while drawing the shape

You can precisely position the points while drawing using the keyboard arrow keys. After a mouse click to start the drawing you can move the cursor using the following keys.

Keystroke Effect
Ctrl + Arrow pad Move the cursor one pixel in the direction of the selected arrow
Ctrl + Shift + Arrow pad Move the cursor one grid position in the direction of the arrow.

When the cursor is in the correct position, you fix the point using a mouse click as usual.

This can only be used while drawing the poly shape. Once you have completed it, it cannot be modified using the same mechanism.
