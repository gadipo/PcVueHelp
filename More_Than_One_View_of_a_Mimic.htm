Opening More Than One View of a Mimic

It is possible to have more than one view of the same mimic open concurrently. Each view is opened in its own window and may be independently moved, zoomed and switched between Design and Run modes. For example, you could have:

  • One view in Run mode and the other in Design mode.
  • One view zoomed in to change the fine detail and another view at full size to see the effect of those changes.

Show picture

How to open another view of a mimic

  1. Select the mimic by clicking anywhere in it.
  2. Select the command Window.New Window.

Or

  1. Right-click on the window title bar and select the command New Window from the context menu.
