Using the BIM control at run-time

At run-time the BIM control renders the selected Ifc file generating a navigable 3D scene. When an Ifc file is loaded for the first time, the BIM control parses it, calculates the geometry required for rendering the scene, and saves it in a cache file (.xbim). This can take several seconds depending on the size of the Ifc file. Subsequently, when the Ifc file has already been loaded in the past, the scene is rendered using the .xbim file which is much quicker.

Once the scene has been rendered, the user can interact directly with it using the below listed mouse actions and keystrokes, plus any Link Open or Run Program animations that have been added during the configuration.

Mouse actions and keyboard shortcuts

The following mouse actions and shortcuts can be used to interact with the BIM control:

  • Tilt - Right click and drag to tilt around the click point

  • Zoom - Mouse wheel or Ctrl + right click + mouse drag

  • Pan - Wheel click and drag or Shift + right click + drag

  • Create and orientate a cut plane - Shift + left click to create a cut plane in the default X and Y axis. Once a cut plane has been created, Shift displays a 3D cursor that can be used to orientate the cut plane in the 3D space by clicking and dragging the arrows or circles. Release the Shift key to cut the model. Show picture

  • Display the Ifc properties window - Tab + left click on an object opens the property window. Show picture

  • Move right - A or right arrow

  • Move left - Z or left arrow

  • Move down - Q or down arrow

  • Move left - D or left arrow

  • Zoom out - S or PgUp

  • Zoom in - W or PgDn

To remove a cut plane you can drag it outside of the model.
