Siemens PPI

This driver supports serial communication with Siemens S7-200 PLCs.

The PPI protocol supports up to 32 devices on a single network.

Communication uses either a serial or USB port of the PC and an adapter from Siemens.

  • USB port - 6ES7901-3DB30-0XA0 Simatic S7-200 USB/PPI-Cable MM Multimaster
  • Serial port - 6ES7901-3CB30-0XA0 Simatic S7-200, RS232/PPI-Cable MM Multimaster

Prerequisites

A PC configuration is required if you are using an USB cable, follow the steps below before configuring an equipment.

If you are using USB cable, the device is detected by the operating system and a dialog will be displayed prompting you to select a source location for the new device. Follow these steps :

  1. Insert the CD supplied with the cable and select it as the source.

  2. Select the access mode PC/PPI cable(PPI) then click the Properties button.

  3. Select the Local Port tab then the virtual COM-port for the PPI cable device. This is the COM number that will be used in PcVue's network configuration. Click OK twice to complete the configuration.

No PC configuration is required if you are using a Serial RS232 cable.
