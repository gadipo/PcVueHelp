The Drawing Object Information Tooltip

The drawing object information tooltip is enabled from the menu command Display.Object Information Tooltip. It enables the display of information about each drawing object as the cursor hovers over it. The information tooltip operates in design mode only. Show picture

The information that is displayed depends on the object type.
