Form Controls and other controls in WebVue

This topic outlines the extent to which the WebVue Client supports the controls. The following are not supported:

  • Grid control
  • Chart control
  • Pie/doughnut/pyramid control
  • Text-box form control
  • Messaging control
  • Web browser control
  • BIM control
  • ONVIF control
  • Video control
  • Third party ActiveX controls

Whilst the functionality provided by the graphic controls is very similar in WebVue there may be differences in appearance due to the web browser environment.

This summary is not exhaustive as there is some interaction between properties. Therefore, it is important to always check precisely how a mimic behaves in WebVue.
