SecuritySignature Method

 Applies to 

Description

Provides for 'signature' by checking the identity (User name and password) of an authorized User.

Syntax

Function SecuritySignature(AuthorizedUsers, AuthorizedProfiles, ShowDialog As Boolean, [Message As String], [UserName As String], [Password As String]) As Boolean

Part

Description

AuthorizedUsers

Provides a list of User names (array of strings) or a single user name (string) authorized to sign. Can be an empty string.

AuthorizedProfiles

Provides a list (array) of user profile names or a single user profile name. Users belonging to these profiles are authorized to sign. Can be an empty string.

ShowDialog

Whether to display the Password Confirmation dialog box.

Message

Optional. A message to be displayed in the Password Confirmation dialog box.

UserName

Optional. To provide the name of the User whose signing authorization is to be confirmed. (If it is omitted or empty, the current User's User Name is assumed.)

Password

Optional. To provide the password associated with the UserName. (Ignored if the ShowDialog parameter is set to True.)

Returns True if the password confirmation succeeds, otherwise False.
