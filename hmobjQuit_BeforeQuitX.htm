Quit and BeforeQuit Example

To display the message "Before (not) closing the project" then leave  the project open.

Private Function fvProject_BeforeQuit() As Boolean

   'Set the return value

   fvProject_BeforeQuit = True

   MsgBox ("Before (not) closing the project.")

End Function

Sub fvProject_Quit()

   MsgBox ("Now closing the project.")

End Sub

Note  The False cancel argument in BeforeQuit prevents quitting, so the Quit event is not triggered.

To display:

  • A dialog that enables the User to choose whether to cancel the closure.
  • The message "Now closing the project" before quitting the project.

Private Function fvProject_BeforeQuit(1) As Boolean

   'Set the return value

   fvProject_BeforeQuit = True

   MsgBox ("Before (not) closing the project.")

End Function

Sub fvProject_Quit()

   MsgBox ("Now closing the project.")

End Sub

 The cancel argument of True in BeforeQuit enables the dialog, then (if the user selects OK) the Quit event and the quit to take place.
