Communication Instructions

These series of instructions allows you to control the behavior of data acquisition drivers. They offer what it takes to start & stop communication, verify communication status, monitor and control various communication parameters as well as change some configuration elements.

The exact list of modes and capabilities vary depending on the data acquisition driver.

Instruction

Action

BACNET

Communication control for the BACnet client driver.

CIMWAY

Communication control for most of the polling-based data acquisition drivers.

DDE

Communication control for DDE exchanges with DDE server applications.

DDECONV

Communication control for DDE exchanges with configured DDE server applications.

FILETRANSFER

Control file transfer services for data acquisition drivers.

LAN

Communication control for PcVue client/server networking.

LONWORKS

Communication control for the LonWorks driver.
M60870_5

Communication control for the IEC 60870-5 client driver, IEC 101 and IEC 104.
M61850

Communication control for the IEC 61850 client driver.
MDNP3

Communication control for the DNP3 client driver.

OPC

Communication control for the OPC client driver.
OPCUA Communication control for the OPC UA client driver.

SNMP

Communication control for the SNMP Manager driver.
