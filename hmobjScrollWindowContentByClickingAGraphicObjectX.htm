Scrolling a Window's Contents by Clicking a Graphic Object Example

This example shows how to scroll a window's contents by clicking on a graphic object.

Scrolling makes the drawing element overlap with the text in the mimic. That is the correct behaviour in this case.

'graphic object Click event

Private Sub Shape1_Click()

Dim HScrollPos As Long

Dim VScrollPos As Long

Dim HObjectPos As Long

Dim VObjectPos As Long

 

'retrieve the horizontal and vertical scroll bar control positions

HScrollPos = Me.Windows(1).GetScrollPosition(fvScrollHorizontal)

VScrollPos = Me.Windows(1).GetScrollPosition(fvScrollVertical)

 

'If one of the position is 0 then retrieve the horizontal and vertical

' high limit positions of the scroll bar controls

' (to scroll to the far bottom right corner of the Window content)

If HScrollPos = 0 Or VScrollPos = 0 Then

    HScrollPos = Me.Windows(1).GetScrollHighPosition(fvScrollHorizontal)

    VScrollPos = Me.Windows(1).GetScrollHighPosition(fvScrollVertical)

 

' or (if both positions are not equal to 0) retrieve the horizontal and

' vertical low limit positions (0) of the scroll bar

Else

    HScrollPos = 0

    VScrollPos = 0

End If

 

'scroll the Window content  

Me.Windows(1).SetScrollPosition fvScrollHorizontal, HScrollPos

Me.Windows(1).SetScrollPosition fvScrollVertical, VScrollPos

 

'move the graphic object

HObjectPos = HScrollPos * (100 / Me.Windows(1).ZoomRatio)

VObjectPos = VScrollPos * (100 / Me.Windows(1).ZoomRatio)

Shape1.Move HObjectPos, VObjectPos

 

End Sub
