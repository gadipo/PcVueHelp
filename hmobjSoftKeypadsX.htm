Soft Keypads Example

Display a numeric keypad then display the input value from it:

Dim dblvalue As Double

Dim bOK As Boolean

bOK = thisproject.ShowNumericKeypad(dblvalue)

If (bOK) Then

MsgBox dblvalue

End If

Display an alphanumeric keypad then display the input value from it:

Dim strvalue As String

Dim bOK As Boolean

bOK = thisproject.ShowAlphaNumericKeypad(strvalue)

If (bOK) Then

MsgBox strvalue

End If
