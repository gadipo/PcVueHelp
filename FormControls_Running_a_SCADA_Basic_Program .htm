Running a SCADA Basic Program from a Form Control

You can configure a Form Control so that it runs a SCADA Basic program. The program can be triggered each time the selection changes and, for most Form Controls, when the mimic containing the Form Control is opened. The parameters to run a program are in the Operations tab. Show picture

Typically, the program would use the SCADA Basic instruction that is specific to the type of Form Control.

Instruction Use with
COMBOBOX Combo-box
LISTBOX List-box
CHECKLIST Check-box
OPTIONLIST Option-box
TREEVIEW Tree-view
TEXTBOX Text-box

See the Help for SCADA Basic for more information.

Option tab properties used when running a SCADA Basic program

  • Program - The name of the program to run.
  • Branch - The variable branch to be used with the program.
  • Function - The function to run. If no function is specified the MAIN function will run.
  • Arguments - A list of comma-separated arguments to be passed to the program. They must be retrieved using the GETARG instruction.
  • Initialization notification - Run the program when the mimic containing the Form Control is opened. (Check-box list, Option-button list, Combo-box list, List-box only)
  • Trigger on - The condition on which the program is to be run. Selected from Option changes only, Selection changes only or Option or selection changes. Check-box list and Option-button list only.

Properties specific to the Text-Box

  • Event trigger
    • Trigger - The program trigger.
      • Keyboard Input - Define the SCADA Basic program to execute when a particular key combination specified by the Key property is entered.
      • Text Changed - Define the SCADA Basic program to execute when a change to the text occurs.
    • Key - The key combination to trigger the program. To configure, click into the field and enter the key combination. It can be any single keyboard key or a key plus any combination of Ctrl, Shift and Alt.

The default configuration executes a SCADA Basic program upon validation of a text input, by running the script when the Enter key is pressed (Keyboard input).
