The Languages Animation

The Languages animation allows the user to select which of the two language settings are used. The language settings are configured in Languages Settings in the Application Explorer.

The language settings, Configuration 1 and Configuration 2, allow the configuration of not only the project and presentation languages but also locale related settings such as the time and date format.

The animation can be configured so that it selects a specific language setting, toggles the language setting or displays a dialog so that the user can make the choice.

How to configure the animation so that a user can select the language

  1. Right-click on a drawing element and select Animate.Send.Languages.
  2. Select the language using the Project drop down list box. The language choices displayed are those configured as the project languages but remember that what the animation is actually selecting is language configuration 1 or language configuration 2.
  3. If the animation is to change the language directly, without displaying a dialog box to allow the user to choose, select the send mode Direct.
  4. Select OK to apply the settings and close the dialog.
