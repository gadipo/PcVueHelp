Overview of Symbols

A Symbol is a collection of drawing elements and associated animations that have been grouped together and stored in a Library. From the Library, a symbol may be pasted into any window as many times as it is needed.

A symbol may represent anything from a simple close button for a window, to a very complex piece of equipment with hundreds of animations.

No special tools are needed to create graphical objects. They are created in the same way as any other drawing using the Workspace tools. Using Symbols in your project has several benefits.

  • Repetitious drawing is avoided considerably reducing development time.
  • The developer may create a reusable Symbol Library specific to the industry sector in which s/he operates.
  • When a symbol is pasted into a window, it is done as a reference. If a modification is to be made to the operation of a symbol, the changes need only be made once to the symbol in the Library, and all instances of it will be modified automatically.
