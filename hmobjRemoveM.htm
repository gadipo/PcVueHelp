Remove Method

 Applies to 

Un-subscribes a variable subscribed using the Add method

Syntax

returnValue = object.Remove(name As String)

Part

Description

name

A string containing the name of the variable to remove.

returnValue

True when removal is successfully.

False when an error occurs.

For variables: If the variable has been subscribed by an animation, it will not be un-subscribed. The Remove action will only release the VBA Add that would have been done.

There is also a Remove method for the Collection class in Microsoft VBA.
