Trend Page Advanced Properties

Show picture

  • Data output:
    • Export column headers:
      • Export variable name - Export variable name in the first row that is written.
      • Export variable description - Export variable description in the first row that is written.
    • Export row headers:
      • Export timestamp - Export the timestamp of each sample in the first column.
        • Format - The timestamp format.
          • Use both date and time - Output both the date and time. This is the default setting.
          • Use date - Output the date only.
          • Use time - Output the time only.
    • Start at column - The start column for data generated by the export.
    • Start at row - The start row for the data generated by the export.
    • If output exists - The behavior, for a specific page, when the Merge option has been selected in the main data export configuration. See the topic Data Export Advanced Properties for information about other options that affect the behavior of the data export, as a whole, if the output already exists.
      • Clear all - For an Excel output clear the entire sheet of all content. For a Csv output the contents of the entire file is cleared.
      • Clear page data - For an Excel output only, clear only data that has been generated by data export.
      • Append after last row - Append new data after any existing data. For an Excel output, you can specify an offset relative to the existing data. For a Csv output, the data is appended right after any existing lines.
      • Merge - Append new data after the position specified by the properties Start at column and Start at row, potentially overwriting previously exported data.
    • Raw mode:
      • Export the previous point - Tick this property if you wish the previous point to be exported when exporting raw data. The previous point the most recent data point just before the beginning of the time period considered for the export.
  • Statistics output - The same as for the data output except:
    • Export row headers:
      • Export statistic name - Export the name of the statistical data in the first column.
  • Aggregated statistics output - The same as for the data output, except:
    • Export column headers:
      • Export variable name and statistic - Export variable names and statistic names in the first row that is written.
    • Export row headers:
      • Export timestamp:
        • Fields:
          • Use beginning timestamp only - Only the timestamp of the start of the exported period is added.
          • Use ending timestamp only - Only the timestamp of the end of the exported period is added.
          • Use beginning and ending timestamps - Both the timestamp of the start and the timestamp of the end of the exported period are added.
