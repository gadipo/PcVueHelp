The Color Properties Tabs

Each color change animation is represented by a tab in the Properties dialog of the drawing to which it is applied. Although the appearance of the tab will vary depending on the type of color animation chosen, all of them contain common icons representing the pattern and border properties. Show picture

Using the color palette to select the colors

The dynamic colors used in an animation are selected using the color palette in the same way as when you color a static drawing element. You may use single, blinking and transparent colors.

Specifying a color for invalid status

On all the color animations you have the option to select a special color that will be displayed when any of the variables used becomes invalid. A variable may become invalid for a variety of reasons, the most common one being the loss of communication to external equipment such as a PLC or RTU.

It is recommended that you make use of this option. If you do not, when a variable becomes invalid the animation will display the color representing the last valid value of the variable. As a consequence, the user may be unaware that she/he may be viewing out of date information.

How many colors can I select?

The number of colors that you can select depends on the animation chosen. You do not have to use all of the colors. For example, the animation Color - Bit Group allows you to select up to 16 colors using 4 bits, but you could use it to select just 4 colors using 2 bits. However, whichever animation you choose, you must select at least 2 of the available colors as the transition to or from an unselected state is not recognized.
