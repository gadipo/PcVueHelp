Use of Third Party Editors

In a tool such as Microsoft Excel, the format in which the import/export files are generated (character separated value) is displayed as a spreadsheet. That is the most suitable tool for editing as it puts each line in a separate row and each text string in a separate cell. Show picture

Hints for using Excel

  • When you have finished editing, save the import/export file in CSV format using comma as the delimiter.
  • You can use Style.Conditional Formatting to shade the rows so as to distinguish the kinds of element to which they relate. Show picture.

In the latter case you should save the file in both the native spreadsheet format (XLS) to preserve the formatting and in CSV format for the Import/Export Tool.
