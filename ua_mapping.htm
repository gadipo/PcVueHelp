Mapping Variables to the UA Gateway

With the majority of OPC DA Servers, the address space, as seen in the left pane of the PcVue OPC Mapping dialog, has a one-to-one match with the OPC Item Id. This is not a mandatory requirement; it is just something that is commonplace. For example, in the following screenshot of a mapping dialog and a Kepware OPC DA server. Show picture

The ItemId Simulation Examples.Functions.Ramp1 is the same as its location within the address space as seen in the left pane of the mapping dialog.

When using the UA Gateway, the Address Space is similar to, but not the same as, the ItemId. This is normal behavior for the UA Gateway to avoid potential conflicts would two OPC servers expose items having the same Id. Show picture

The ItemId um:W10-TOLS-RM1:Kepware.KEPServerEX.V5:UAServer/KEPServerEX|Simulation.Functions.Ramp1 is different from its location in the Address Space (different prefix).

The setting Use alias as prefix in ItemId in the OPC COM ItemIDs tab of the UA Gateway's Administration Tool can be used to simply the ItemId replacing the prefix with a much simpler alias. However, the Address Space remains unchanged.

Deepening the UA Gateway browsing

Depending on the complexity of the connected OPC UA server, the UA Gateway may not fully expose the objects through its OPC DA Server. This can be improved using the setting Browse Next Level for Variables in the OPC DA (COM) tab of the UA Gateway's Administration Tool.

Timestamp setting

The UA Gateway allows the selection of the timestamp source for OPC UA objects it exposes through its OPC DA Server.

The selection is made using the setting Timestamp Source in the OPC DA (COM) tab of the UA Gateway's Administration tool. Show picture

The table below summarizes the resulting timestamping behavior depending on this UA gateway parameter combined with the configuration of the PcVue OPC DA client:

 PcVue OPC DA Client timestamp configuration UA Gateway timestamp source Mapped variables’ resulting timestamp
Default Internal OPC DA Client / SCADA host computer
Provided by server Internal OPC DA Client / SCADA host computer
Default ServerTimestamp OPC DA Client / SCADA host computer
Provided by server ServerTimestamp OPC UA Server / UA host computer
Default SourceTimestamp OPC DA Client / SCADA host computer
Provided by server SourceTimestamp OPC UA Server / UA host computer
