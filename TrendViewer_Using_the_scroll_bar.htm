Using the Trend Viewer's Scroll Bar

The scroll bar is used to select data in the display buffer for display on the chart.

You can get an approximate idea of the ratio of the number of values that may displayed in the chart to those held in the buffer by the size of the scroll button in relation to the area in which it may be moved.

The way in which the scroll bar tools behave changes slightly depending on which mode the Trend Viewer is in.
