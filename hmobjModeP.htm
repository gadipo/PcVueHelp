Mode Property

 See also 

Description

Returns or sets the mode of the window. Read/Write.

Syntax

object.Mode [= fvWindowMode]

or

object.Mode [= fvModeNavigator]

Mode can take one of the following fvWindowMode values:

  • fvWindowModeRun.
  • fvWindowModeDesign .

 Or

Mode can take one of the following fvModeNavigator values:

  • fvActiveWindow - The active window is automatically displayed.
  • fvUserDefinedWindow - The window displayed is the one selected using the "SetWindow" method.
