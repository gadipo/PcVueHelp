The Track and Polygon Markers

This topic explains the properties and behavior of the track and polygon markers. The track and polygon are very similar except that the track is an open shape (like the polyline drawing element) and the polygon is a closed shape. See the topic Managing Map Markers in General for general information about map marker properties and behavior.

There is a significant difference in the way that a track or polygon is rendered compared to the other markers. When a map is zoomed, any track or polygon zooms with it. All other markers stay the same size as defined when they were configured.

How to create a track or polygon

The process to create a track or polygon marker is similar to that of the other shapes except you left click several times, each click creating a node. You complete the marker by a single right click. Show picture

How to move a node

  1. Using the Marker toolbox, select Select mode.
  2. Click a single node. The selected node is highlighted. Show picture
  3. Drag the node to its new location.

How to delete or add a node

  1. Using the Marker toolbox, select Select mode.
  2. Right click a single node. A context menu is displayed offering the choice of adding or deleting a node.
  3. Choose Add or Delete as appropriate.

Changing the stroke (line) style using the StrokeDashArray property

The StrokeDashArray property allows you to draw a broken line by entering two or more numbers separated by commas. The value of each number configures the relative size of the mark and space. This can be illustrated by some examples.

StrokeDashArray  
1, 1
2, 2
2, 2, 4, 2
