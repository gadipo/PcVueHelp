Configuring the Trend Viewer's Touch Screen Features

The Touch screen tab contains properties that modify the Trend Viewers appearance at run-time to make it easier to use with touch screen type displays.

  • Display large toolbar - Show an enlarged toolbar.
  • Display large historical request dialog box - Show an enlarged dialog for requesting historical data.
  • Display large period and time scale management dialog box - Show an enlarged dialog for managing the period and time scale.
  • Display large scrollbar - Show an enlarged scrollbar.
