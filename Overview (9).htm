About the Database Manager

The Database Manager is a tool provided with PcVue. The purpose of the Database Manager is to provide easy access to databases that are regularly administered. The tools it provides may be used with any Microsoft Access or SQL Server databases.

The Database Manager is also used to configure HDS database built-in replication.

The Database Manager provides these capabilities: Show picture

  • Local and remote navigation and browsing to select database files,
  • Keeping an index of the locations of the databases you administer,
  • Viewing and maintaining database properties,
  • Viewing and maintaining database contents,
  • File shrinking and deletion,
  • Administering security measures,
  • Configuring HDS database built-in replication.

You can access these functions through the Database Manager's menu bar or from the context menu that opens when you right-click on an item in the left-hand pane.

For SQL Server administration, the Database Manager provides simple to use features, but does not replace Microsoft SQL Server Management Studio and its advanced capabilities.
