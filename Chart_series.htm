Configuring a Chart's Series

This topic is for the features configured by the Chart's Series tab that are used to configure the data source for each of the series and the style in which they are plotted. Show picture

Selecting the data source

Normally the data source will be a variable from the Variables Tree. As the Chart uses the historical data created by trend recording, the variable must be configured as Trend Recorded in the Application Explorer. Alternatively, the variable field can be left blank and both the X and Y values supplied programmatically.

The Description appears in the Legend. If a Description is not entered then the Description from the variable configuration is used. If the variable configuration does not have a description then the variable name itself is used.

Configuring the plot style

The plot style of each series can be individually configured. The main property that affects the appearance is the Type that can be selected from the following.

The other properties that influence the plot style are as follows.

  • Color swatch - Selects the color of the line, and also the fill if one of the solid types is selected.
  • Style - The width and style of the line, or border in the case of one of the solid styles.
  • Marker - The type of the mark that is positioned at each data point. The default is None, for which no mark is displayed.

Selecting the axes

The chart can display two vertical (Y) axes. One to the left and one to the right of the chart. A series can be allocated to either the left or the right Y axis. Unlike the Trend Viewer, where each trace can have its own range, in the Chart the range over which a series is displayed is determined by the axis to which it has been allocated. See the topic on Configuring the axes for further information. Show picture

Configuring the mouse over tooltip

The exact text displayed in the tooltip, when the mouse hovers over a data point, can be configured using the Format property. The default is X={X} ;Y={Y}. Characters are interpreted literally except {X} and {Y}, which are substituted with the X and Y values of the data point. If the point is timestamped then it also appears in the tooltip. Advanced options

  • Transparency - Allows a series to be displayed in a transparent color. Useful when series overlap and when using the Area plot style. Range 0 (opaque) to 100 (transparent) Show picture
  • Spline coef. - The coefficient used when calculating the spline. Range 0 to 120. Show picture
  • Shadow - Add a shadow effect to the plot.
  • Sort - Sort the series (or not) according to the X-axis value. If sorted the plot is always drawn left to right. Show picture
