SetDatePeriod Method

 Applies to 

Set the Trend Viewer's start and end times.

Syntax

Sub SetDatePeriod(StartDate As TRDDate, EndDate As Variant)

Part

Description

StartDate

The start time (left side) as a TRDDate object

EndDate

The end time (right side) as a TRDDate object. (Optional).

Example

Dim tStart As New TrdDate
tStart.Hour = 9
tStart.Minute = 30
tStart.Second = 0

Dim tEnd As New TrdDate
tEnd.Hour = 10
tEnd.Minute = 15
tEnd.Second = 0

Trend1.SetDatePeriod tStart, tEnd
