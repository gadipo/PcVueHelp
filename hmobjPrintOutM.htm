PrintOut Method

 Applies to 

Prints a hard copy of the calling object.

For the Application object the entire workspace except the toolbars and menu is printed.

For the System object the entire contents of the screen is printed.

For the Window object only the Window and its contents are printed.

Syntax

object.PrintOut([copies As Integer], [orientation As fvOrientation], [printer As String])

Part

Description

copies

Optional argument. Default: 1 A number that specifies the number of copies to be printed.

orientation

Optional argument. Default: fvPortrait Orientation can take one of the following fvOrientation values:
fvLandscape
fvPortrait

printer

Optional argument.
A string that indicates the printer to be used.
If not specified then the system default printer is used.
