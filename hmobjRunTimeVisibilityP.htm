RunTimeVisibility Property

 Applies to 

Description

Indicates the current visibility state of an object.

Syntax

Property RunTimeVisibility As Boolean

It is set from a Visibility animation or from VBA:

  • True: The object is visible.
  • False: the object is invisible.  

If it is set by both an animation and VBA program, the latest to do so determines the setting.

Warning

The RunTimeVisibility property does not provide the object visibility state that is set from a Symbol animation, Layers, Zoom Visibility Bounds and/or Color configuration or animation.

The RunTimeVisibility property also appears in the Properties List dialog (from menu Display.Properties List). The property value may be changed from that list (directly or using the Variable Links feature).

Because this is a dynamic property, its value is not saved (however the variable link configuration is saved).
