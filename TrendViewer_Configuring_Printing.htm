Configuring How a Trend Viewer is Printed

The properties on the Print tab manage the printing of a Trend Viewer when using the print button on the Trend Viewer toolbar.

  • Back color - An alternative background color for the chart when it is printed. Normally set to white.
  • Copies - The number of copies that will be printed for each Print Trend action.
  • Orientation - The orientation of the print. The default is Landscape.
