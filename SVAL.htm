SVAL

See also

Return the numeric value of a character string.

WebVue support - Yes.

Syntax

SglVal = SVAL(String);

Return type: SINGLE.

Argument

Meaning

String

The character string. Type STR.

Execution

If the string contains a character or character sequence that is recognized as non-numeric, zero is returned.

Example

DIM s1 As Single;
s1=SVAL("123.456");
