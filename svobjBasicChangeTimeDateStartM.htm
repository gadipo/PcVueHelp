BasicChangeTimeDateStart Method

Filters the events in the object's list by a start and end date.

Syntax

object.BasicChangeTimeDateStart (adTimeDateStart As Double, adTimeDateEnd As Double, lAdjust As Long) As Boolean

Part

Description

adTimeDateStart

Earliest date & time for events to be displayed.

adTimeDateEnd

Latest date & time for events to be displayed.

lAdjust
