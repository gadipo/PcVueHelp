How to Add a Trend Page

Each Trend Page can include data from one or more trended variables. A single Trend Page can generate up to three outputs.

  • Data output - Basic table of raw values (extracted as archived), or sampled data, at the configuring sampling rate for the selected variables.
  • Statistics output - Value statistics for the selected variables.
  • Aggregated statistics output - Aggregated value statistics per sub-period for the selected variables.
  1. Open the Application Explorer and expand the configuration tree to select a previously configured Data Export.
  2. From the task list, select Add a trend page. The Trend Page Creation dialog opens. The following screen-shot shows the default configuration. Show picture
  3. Configure the main properties.
    1. Enter the Name for the trend page. The name is the unique identifier of the trend page within PcVue and can also be used in the expression used to generate the output name.
    2. Enter the optional Description.
    3. Select which outputs are to be generated by ticking the Export data, Export statistics and Export aggregated statistics properties as required.
    4. Enter the Output name. This property is used to generate the sheet name when using an Excel output or the file name when using a Csv output. It can either be an expression or plain text. See the topic Using expressions in the folder, workbook, template or sheet names. Note that the name for the statistics output is configured in the Statistics tab.
    5. Select the Data mode from either Sample, Sample with synchronization or Raw.
    6. Click Apply to create the trend page.
  4. Select the Period tab to configure the default period. The properties in the Period tab are only used if you generate the export using SCADA Basic and do not specify the period in the EXPORT instruction. See the topic Understanding the time period used by Data Export for information about how the period properties are used.
  5. Select the Sampling tab and configure the sampling properties. See the topic Configuring the trend sampling. The Sampling tab does not appear if Raw data mode is selected.
  6. If you have chosen to generate either of the statistics outputs, select the corresponding tabs to configure what will be generated. See the topic Configuring the Trend Statistics and Aggregated Trend Statistics Output.
  7. Click the OK button to close the dialog. The final configuration step is to add variables to the trend page. See the topic How to add variables to a trend page.
