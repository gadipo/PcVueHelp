Purge a Database

Declare internal variables:

Dim PurgeDateStart As Date

Dim PurgeDateEnd As Date

Dim PercentToPurge As Variant

Enable/disable purging etc. processes

Private Sub Text2_Click()

Dim dbMgt As New HDSMgt

Dim DB As HDSMgtDatabase

    Set DB = dbMgt.Databases("DefaultDB")

    DB.EnableProcessing = Not DB.EnableProcessing

End Sub

Purge local DB by deleting records (5 minutes old) and reducing the size of the database

Private Sub Text3_Click()

Dim dbMgt As New HDSMgt

Dim DB As HDSMgtDatabase

    Set DB = dbMgt.Databases("DefaultDB")

    Call DB.StartPurge(5, hdsmgtMinutes, True)

End Sub

Try to compact the database contents:

Private Sub Text7_Click()

Dim dbMgt As New HDSMgt

Dim DB As HDSMgtDatabase

    Set DB = dbMgt.Databases("DefaultDB")

    DB.StartShrink

End Sub

Urgent purge (by 20% of data records):

Private Sub Text8_Click()

Dim dbMgt As New HDSMgt

Dim DB As HDSMgtDatabase

    Set DB = dbMgt.Databases("DefaultDB")

    Call DB.StartUrgentPurge(20)

End Sub
