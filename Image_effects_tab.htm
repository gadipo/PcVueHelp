Changing the Image on Mouse Over

The Effects tab allows you to select alternative images to be displayed at run time when the pointer is over the image (mouse over), or the image is selected by pointing and clicking (mouse down). This is particularly useful if the image is being used to start a VBA program and does not have any associated control zone animation. Show picture

When selecting the alternative images the following points should be noted.

  • If a transparent color was used with the main image, the same color will also be treated as transparent in the images selected in the Effects tab.
  • The image selected in the effects tab will be displayed as the same physical size as main image. If the images are of different sizes, they will be scaled accordingly.
