Changing the Appearance of an Image

To illustrate this topic, the author has used an image of the color palette, not the color palette itself.

Using the Transparent Color property

Some image formats support native transparency. That is one of the colors is defined as transparent and any background color is visible through it.

For image formats that do not support transparency, the HMI allows you to pick one of the image's (non-transparent) colors as transparent. When the image is rendered by the HMI, the background color shows wherever the color designated as transparent occurs. Show picture

To select an image color as transparent:

  1. Display the Image Properties dialog by double-clicking on the image.
  2. Select the Image tab and tick Transparent color. The cursor will change shape to a small circle.
  3. Click on the area of the image that contains the color to be made transparent.
  4. Click the OK button to confirm the change. Show picture

Adding a pattern to an image

As an alternative to displaying the background, you can replace the transparent color of an image using a pattern, identical to that of any solid drawing element, using the properties configured in the Base tab. Show picture

Adding a style to an image

You can change the style (button, relief, shadow etc.) of an image, as with any solid drawing element, using the Appearance properties configured in the Aspect tab. Show picture

Changing the style increases the space occupied by the image. You will be prompted to either increase the object size or rescale the image to retain the original size. It is preferable to increase the object size as re-scaling an image inevitably degrades its appearance.
