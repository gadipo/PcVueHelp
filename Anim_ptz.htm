Camera PTZ

The Camera PTZ animation is used to control the camera associated to an ONVIF control. It can be used as an alternative to scripting in SCADA Basic with the CAMERADISPLAY instruction. Show picture

  • Mimic and Branch - The name of the mimic and optional branch in which the ONVIF control is located. The wildcard * can be used to reference the mimic and branch in which the animation is contained.

  • ID - The identity of the ONVIF control. The identify can be discovered using the Graphic Explorer.

  • Command - A camera command selected from one of the following.

    • Goto preset - The Preset property can either be a string which is the name of the preset, or the name of a text variable which contains the name of the preset.

    • Goto home

    • Zoom in

    • Zoom out

    • Pan left

    • Pan right

    • Tilt up

    • Tilt down

    • Stop PTZ
