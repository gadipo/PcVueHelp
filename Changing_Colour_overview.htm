Overview of Changing color

Changing the color of a drawing element is one of the simplest and most commonly used ways of presenting process information to the user. You can use the status of bits and the value of register variables as the source of a color change.

All color animations allow you to change the border (line), background fill (pattern) and text of a drawing element to give a very wide range of color combinations. You may use the blinking and transparent options in your choice of colors. All the animations include the capability to select a special color to be used whenever any of the variables are invalid (for example when communication has failed).

The Pattern Property

Element

Effect

A drawing with a solid or colored button style

The whole of the pattern area changes.

A drawing with a hatched pattern

Only the color of the hatching changes.

Text

The color of the background changes.

Lines and polylines

No effect.

Image

The pattern color is displayed through the transparent area of the image.

The Border property

Element

Effect

A geometrical shape (rectangle, ellipse etc.)

The border changes color.

A line or polyline

The whole line changes color.

Text

The characters change color.

Image

The whole line changes color.

The image (picture) object supports all color animations except Legend.

If there is text in an image, you can change its color, or make it disappear, by making its color transparent. For details, see the topic Advanced Image Properties and the book Changing the Appearance of an Image.
