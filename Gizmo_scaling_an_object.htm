Scaling an Object

Using the yellow ball, you can scale a model and make it uniformly smaller or bigger. This is called XYZ uniform scaling.

Non-uniform scaling is available through the Properties dialog of the model.

Selected

Yellow ball

Scaling on

All three axes

 

About scaling ratios

Scaling ratios are expressed as percentages, so 200% means twice as large, while 50% means half the size.

  • A value of 0 makes the model invisible.
  • A negative value mirrors the model and the Gizmo along the axis. This makes all three values negative and puts the model upside down, left-to-right and back to front.
