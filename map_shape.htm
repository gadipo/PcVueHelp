The Map Shape, Text, Image and Symbol Marker

This topic explains properties and behavior specific to each type of marker. The track and polygon is covered in a separate topic, as the behavior of these markers is quite different. See the topic Managing Map Markers in General for general information about map marker properties and behavior.

The shape marker

The shape marker is used to place a solid shape on the map. The only property specific to the shape marker is its form, which can be selected from the following options.

Shape Description
Circle (Ellipse)
Square (Rectangle)
Rounded square (Rounded rectangle)
Triangle
Inverted triangle
Rhombus

The text marker

The text marker is used to place a multi-lingual string on a map. The following properties are specific to the text marker.

Property Description
Text The string to be displayed on the map. Multi-lingual.
Text color The color in which the text will be rendered. Entered either as a hexadecimal number or by clicking on the color swab and using the Color palette.
Font The font used to display the text, for example Arial.
Font size The size of the text in pixels.
Font weight The weight of the font, for example Bold. The font weights that are available will depend on the font that has been selected.
Font style The style of the font, for example Oblique. The styles that are available will depend on the font family that has been selected.

An When choosing a font and style you must make sure that it is available on the target PC.note.

The image marker

The image marker is used to place an image, from a library, on a map. The following properties are specific to the image marker.

Property Description
Image The image to be displayed on the map. Clicking the adjacent down-arrow button opens the image selector from where the image can be selected from a library.

If the height and width are both set to zero the image will be displayed full size, otherwise the image will be scaled to the given dimensions.

The symbol marker

The symbol marker is used to place a symbol, from a library, on a map. The following properties are specific to the symbol marker.

Property Description
Symbol The symbol to be displayed on the map. Clicking the adjacent down-arrow button opens the symbol selector from where the symbol can be selected from a library. The symbol is instantiated with the selected branch.

The ability to open a mimic from the symbol marker is disallowed as the symbol itself can contain a Link-open animation.
