Scroll Method

    Applies to          

Scrolls the object vertically.

Syntax

Sub Scroll(ScrollMode As ScrollMode, Line As Long)

The value of ScrollMode controls the manner of scrolling.

smToTop: go to top (equivalent to Home key)
smPageUp: up one page (equivalent to PageUp key)
smLineUp: up one line
smLineDown: down one line
smPageDown: down one page (equivalent to PageDown key)
smToBottom: go to bottom (equivalent to End key)
smToLine: go to line number Line: it will be displayed as the first line

The Line parameter is used for the smToLine mode only.
