The Map Objects Tree

The Map Objects Tree supports the following features and actions to manage the layers and markers. Show picture

  • Map Items tab - Tree display of map layers and map markers. The map markers sorted according to which layer they are attached.
    • The branch that represents each layer, displays the layer name, the number of markers it contains in round brackets, and the zoom range in square brackets.
      For example,: Countries (5) [1;20].
    • Clicking the Triangle icon expands or contracts the branch displaying or hiding the markers.
    • Clicking the Eye icon toggles the visibility of the layer's markers both in the Map Markers Editor and at run-time in any associated map.
    • Clicking a layer name selects it and displays the layer properties.
    • Clicking a marker name highlights the marker on the map and displays the marker's properties. All other objects are de-selected.
    • Double clicking a marker name centers the map on the marker, highlights the marker on the map and displays the markers properties.
    • Double clicking a layer name, computes the bounding box of all markers in the layer and locates the map on it. If necessary, it also adjusts the zoom level so the layer is visible.
    • Shift & click allows the selection of multiple markers. Ctrl & click adds a marker to the selection. Alt & click removes a maker from the selection.
  • Markers tab - List of all map markers irrespective of which layer they are attached to. The markers behave in the same manner as described above.

The default layer, displayed at the top of the Map Items Tree, is mandatory.
