Using a 3D Camera Model

A camera model is a 3D model that is either custom or built-in (the HMI offers the choice). When it is inserted in a window, the HMI automatically memorizes the current settings of the window camera. These settings can later be re-applied to it, allowing an instantaneous transition to a different view of the 3D world. You can modify (or update) the settings memorized by a camera model.

The types of camera model available are as follows.

  • Custom camera model - Any of the 3D models located in the project libraries.
  • Built-in camera model - A 3D model provided by the HMI and located in the HMI's resources. It is not available as a standard model (i.e. located in a library).

Except for the way they are inserted in a window, camera models of both types are processed identically by the HMI.

Inserting a camera model

You can insert a camera model as follows:

  • From the Insert menu, select one of the Insert Camera commands.
  • From the 3D Draw toolbar, select one of the Insert Camera commands. Show picture

When you insert a custom camera, the 3D model library is automatically opened for selection of the model that will represent the camera. However, when you insert a built-in camera, the built-in 3D model is directly inserted in the mimic.

Several camera models can be inserted in a single window.

When a camera model is inserted, it is usually (depending on the model shape) not visible because it is positioned at the current location of the window camera.

When a camera model is inserted, that model is not automatically selected.

Displaying a camera model

Unlike a standard 3D model, when a camera model is selected, the HMI does not show the model axes. This is because the HMI does not allow for moving, rotating or scaling a camera model directly in the mimic.

The parameters for camera model position and axes rotation are automatically calculated using the camera setting parameters and using the model's Z axis as the aiming axis of the camera.

When you use an orthographic view, the camera-aiming axis is always parallel to one of the world axes and always perpendicular to the orthographic projection plane (defined by the two other axes of the 3D world).
