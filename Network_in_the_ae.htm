Managing Networking in the Application Explorer

A project's networking configuration is displayed under the Communication.Networking node. It includes Stations, Nodes, Lists and Associations. The Application Explorer is used for network configuration and diagnostics. It displays run-time status information and offers commands for the configured network items which can be useful for troubleshooting.

State information is "as-seen" by the local station, that is the one on which the Application explorer is running:

  • All information about the local station are retrieved locally, independently of any connection to other stations.

  • All information about remote stations are retrieved through the Networking exchanges, and may or may not be available depending on the connection state with the selected remote station.

In particular, state information is for the connection between the local station, that is the one on which the Application explorer is running, and the selected station or node.

The maximum number of stations in a single networked application is 500.

About the Standby Active and Standby Passive subscription modes

Consider the following architecture:

  • DAS01, DAS02 and DAS03 are Data Acquisition Servers,
  • DAS_ASSOC01 is a single active Data Acquisition Association including DAS01, DAS02 and DAS03,
  • Client01 is a client of the association.

If DAS01, DAS02, DAS03 are running, and the Availability Rates within the association DAS_ASSOC01 are DAS01 = 100, DAS02 = 80, DAS03 = 60, then, in the Application Explorer running on the client station the following subscription modes are seen.

  • The subscription mode with the station DAS01 = Main
  • The subscription mode with the station DAS02 = Standby active
  • The subscription mode with the station DAS03 = Standby passive

If DAS01, DAS02, DAS03 are running, and the Availability Rates within the association DAS_ASSOC01 are DAS01 = 70, DAS02 = 80, DAS03 = 60, then, in the Application Explorer running on the client station the following subscription modes are seen.

  • The subscription mode with the station DAS01 = Standby active
  • The subscription mode with the station DAS02 = Main
  • The subscription mode with the station DAS03 = Standby passive
