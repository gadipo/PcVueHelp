Style Record Format

The style (font and format properties) of a text string is not recorded as part of the text string record. Instead, it is indexed and recorded separately. A text string record refers to the style via the style index, so that when several text strings are defined with the same style, the style is only recorded once.

The fields of a font and format record are as follows:

Field Description Data type Value
1 Record ID Text FONT
2 Style index Numeric 1 to n
3 Height - see table below Numeric -11 to -96
4 Width Numeric  
5 Weight Numeric Bold = 700 Normal = 400
6 Italic Boolean 0 or 1
7 Underlined Boolean 0 or 1
8 Font name Text "Arial" etc.
9 Strikeout Boolean 0 or 1
10 Character set Numeric Western = 0, Cyrillic = 204
11 Format - text alignment - see table below Numeric 36 to 86

The Height field

The actual font height can be ascertained using the following look-up table.

Height code Font size   Height code Font size
-11 8   -27 20
-12 9   -29 22
-13 10   -32 24
-15 11   -35 26
-16 12   -37 28
-19 14   -48 36
-21 16   -64 48
-24 18   -96 72

The Format field

Format code Text alignment
36 Left
37 Centered
38 Right
84 Multi-line and left
85 Multi-line and centered
86 Multi-line and right

Example

This is an example of a Font and Format record:

FONT,1,-21,0,400,0,0,"Arial",0,1,37
