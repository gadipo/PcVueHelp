IEC 61850 Network - Advanced Properties

IEC 61850 network advanced properties are found in the Advanced tab of the Network properties dialog. Show picture



  • Command parameters
    • Max parallel requests - The maximum number of requests (all requests sent to the device - read, write etc…) that can be processed at the same time. Default 50.
  • Networking
    • Servers - For a multi-station application the list of stations that are to behave as producers. If the application is run on any station not in the list, the IEC 61850 network will not be started.
  • Redundancy mode

    • Cold mode – If selected, the network is stopped on data acquisition servers in passive state.

    • Warm mode – If selected, the network is not stopped on passive servers.
