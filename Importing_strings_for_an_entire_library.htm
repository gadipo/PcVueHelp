Importing Strings for an Entire Library

This process imports the strings for an entire library (local, common or custom).

How to import the strings for an entire library

  1. Open the Import/Export Tool using the menu command Display.Translation Import/Export.
  2. If you have not already done so, configure the file paths and languages. See the topic Configuring the Import/Export tool for details.
  3. Select the library or libraries for import. You can select more than one library by clicking while holding down the Ctrl key.
  4. Start the import process using the Import button.
  5. The import function processes only the mimic, template and/or symbol files that are referenced in the translation files. For large projects, this may take several seconds to run.

If any mimics are open at the time of import, the process will display a warning dialog then automatically close them.

Any changes you make to the Import/Export Tool configuration are only temporary unless you click the Save button.

Canceling the import process

You can cancel the import process at any time during its execution by clicking the Import button again (which is renamed Stop during the import process). The import will stop but changes made to the files prior to clicking Stop are not undone.

Important note about selecting the language for the import

An import/export file may contain strings for several languages. The Import/Export tool indicates the languages available by showing the entry in the language list boxes with bold styling. Appended (in square brackets) to the language name is a list of the types of strings that are available (mimics, mimic templates and symbols) Show picture

If you select a language for which there is no information in the import/export file, the import will run to completion with no errors but no strings will be imported.

Checking the Status Log

When the export process has completed you should check the Status log in the lower pane of the Import Export Tool dialog for any errors. Show picture

If you want a permanent record of the process or you want to search for something in particular you can copy the contents of the Status log by using Ctrl+A and Ctrl+C and then paste it into a text editor such as Notepad and save it as a text file.
