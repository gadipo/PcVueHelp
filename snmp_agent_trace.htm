SNMP Agent Add-on Traces

Information regarding the SNMP Agent Add-on are logged in the traces and displayed in the Event Viewer with lines starting with usrmgrSNMPAgent.

Additional traces can be triggered using the SCADA Basic instruction TRACE as follows.

TRACE(Mode, ManagerID, HexFlag);

  • Mode - Turn the additional trace on or off
    • 1 - On
    • 0 - Off
  • ManagerID - The identifier of the SNMP Agent Add-on task. Set to 11 if the SNMP Agent is the only Add-on in use.
  • HexFlag - Hexadecimal combination flag
    • 001 - Display traps sent by the SNMP Agent.

Example of use of SCADA Basic TRACE instruction

TRACE(1,11,”1”); ‘Turn on additional trace for traps

TRACE(0,11,”1”); ‘Turn off all traces.
