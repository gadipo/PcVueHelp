TOL

See also

Convert a number to a LONG value.

WebVue support - Yes.

Syntax

LongVal = TOL(Num);

Return type: LONG.

Argument

Meaning

Num

The value to be converted. Any numeric type.

Execution

The number is converted to a LONG value by truncating its decimal part.

Example

SUB Main()

DIM lngValue as Long;

DIM lngResult as Long;

 

lngValue = 125;

lngResult = TOL( lngValue );

PRINT("Result: ", lngResult);

'Display "Result: 125"

END SUB
