Configuring the Trend Viewer's Value and Timestamp Formats

The Format tab contains properties that control the format in which values and timestamps are displayed at run-time.

Configuring the time and date format

The time and/or date format used in the Trend Viewer can be configured using the following properties.

  • Cursor mode - The date and time of the cursor position. Displayed in a box below the cursor in the area reserved for the time axis scale.
  • Time information - The date and time at the start and end of the trend chart. Only displayed when the Display time information property, in the Display tab, is ticked.
  • Date in timescale - The date in the time axis scale. Note that the date is only displayed if the period of the chart is greater than 10 hours.

The format in which time and date is displayed is configured using format characters.

Configuring the value format

Register variables can display a wide range of values and it is sometimes necessary to change the format in which they are displayed in the Trend Viewer to accommodate this.

  • The maximum number of decimal places that are displayed can be limited using the property Maximum number of decimal digits. The default is three. Note that leading or trailing zeros are never displayed.
  • When you need to display a very large value, you can choose to display it in exponential format. You do this by selecting the Greater than tick box and then selecting a value above which variables will be displayed in exponential format. For example, if you set the threshold to 1000 and the value was 1456 it would be displayed as 1.456e+3
  • When you need to display a very small value, you can choose to display it in exponential format. You do this by selecting the Less than tick box and then selecting a value below which variables will be displayed in exponential format. For example, if you set the threshold to 0.1 and the value was 0.0165 it would be displayed as 1.656e-2
