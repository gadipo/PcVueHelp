Overview of the Gizmo
 

 
 
 
 
 
 
 
 
 
 
 
 
 
Overview of the Gizmo Tool
 
The Gizmo tool shows you the orientation of the selected
 model in design mode and allows you to manipulate it using translation,
 rotation and scaling. Show picture
 

 

 
 
 
 
The Gizmo's size never changes, so as to make it easy
 to manipulate as you zoom in or out or change the scale of the model.
 
The Gizmo's axes are color coded as follows:
 
 
 
 

 
 
Axis
 
 
 
Meaning
 
 
 

 
 
Red
 
 
 
Model's X axis
 
 
 

 
 
Green
 
 
 
Model's Y axis
 
 
 

 
 
Blue
 
 
 
Model's Z axis
 
 
 
 

 
The icon that appears in a selected frame shows the frame's
 pivot and orientation of the X, Y and Z axes. It cannot be used for translation,
 rotation or scaling.

Scaling an object
 

 
 
 
 
 
 
 
 
 
 
 
Scaling an Object
 
Using the yellow ball, you can scale a model and make 
 it uniformly smaller or bigger. This is called XYZ uniform scaling.
 

 
Non-uniform scaling is available through the Properties dialog of the model.
 
 
 
 
 

 
 
Selected
 
 
 
Yellow ball 
 
 
 

 
 
Scaling on
 
 
 
All three axes
 
 
 

 
 
&#160;
 
 
 

 
 
 
 
 
 
About scaling ratios
 
Scaling ratios are expressed as percentages, so 200% 
 means twice as large, while 50% means half the size. 
 
 
A value of 0 makes the model invisible. 
 
A negative value mirrors the model and the Gizmo along the axis. This makes all three values 
 negative and puts the model upside down, left-to-right and back to front.

Translating an object
 

 
 
 
 
 
 
 
 
 
 
 
Translating a Model
 
This topic describes how to use the Gizmo tool to move 
 a model along a line.
 
 
Select a model.
 
Click on the Gizmo's 
 green bar (the shaft rather than the arrow point). The arrow is highlighted 
 in orange to indicate that it is selected. 
 
Drag the mouse. The 
 model moves along the selected axis. 
 
You can use the Gizmo's 
 other bars to move (translate) the model along its other local axes.

Green 
 
 
 
Blue

Local Z axis

 click on it and drag it.
