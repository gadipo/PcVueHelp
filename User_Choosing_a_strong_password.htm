Choosing a Strong Password

A strong password is one that is very difficult to guess. In PcVue, if you select the option Impose strong password in the User Accounts Preferences, you must then enter a strong password for all User Accounts.

What is a strong password?

 PcVue uses the following criteria to determine if a strong password has been configured.

  • It must contain between 8 and 16 characters.
  • It cannot contain the user name.
  • It must contain characters from at least three of the four following types of character:
    • Lower case alphabetic: a-z
    • Upper case alphabetic: A-Z
    • Numeric: 0-9
    • Special characters: ` ~ ! @ # $ % ^ * ( ) _ + - = { } | [ ] \ ? . /
      you cannot use non-printable control characters.

If you have not selected Impose strong password then you cannot use special characters in the password.

If you have selected Impose strong password then you can use special characters in the password.

In both cases, the user name is not case sensitive.
