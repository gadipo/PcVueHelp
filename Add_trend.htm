How to Add Variables to a Trend Page

  1. Select a previously configured trend page.
    1. Open the Application Explorer and expand the configuration tree to select a previously configured Data Export.
    2. In the right pane, select a trend page.
  2. Add variables to the trend page.
    1. From the tool bar, select Add a trend . The Trend Selector dialog opens. Show picture
    2. In the left pane, select the archive unit in which the variable is trended.
    3. In the right pane, select the trend to be added and click the OK button.
    4. Repeat steps a to c to add more trends as required.
  3. Use the Move up and Move down tools to change the order in which the trends appear in the trend page so that it fits your ordering requirement in the exported data outputs.
  4. Save the configuration and close the Application Explorer.

Only Proprietary and HDS archive units are shown in the Trend Selector dialog.
