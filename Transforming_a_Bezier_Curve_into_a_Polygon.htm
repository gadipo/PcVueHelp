Transforming a Bézier Curve into a Polygon

You can convert a Bézier curve into a Polygon. This can be useful when you want to use the path animation.

To transform Bézier curve into a Polygon:

  1. Draw a Bézier curve and adjust it to the final shape.
  2. Display the Bézier curve's Drawing Properties dialog and select the Aspect tab.
  3. In the shape drop down list box, select Polygon.
  4. Click OK to confirm the transformation.
