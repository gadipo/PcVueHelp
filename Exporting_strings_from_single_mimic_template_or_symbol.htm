Exporting Strings from a Single Mimic, Mimic Template or Symbol

This process exports the strings from a single element (mimic, mimic template or symbol). Before you use Import/Export for a single item, you must first configure the Import/Export Tool to select the destination import/export file and export language. See the topic Configuring the Import/Export tool.

If the Import/Export options are grayed out in the main menu or pop-up, ensure that the type of process (mimic, mimic template or symbol) is enabled and that the import/export file names are configured in the Translation Import/Export Tool dialog.

How to export the strings from a single mimic or mimic template

  1. Open the mimic or mimic template and select Design mode. If the mimic was already open, make sure that any changes that have been made are saved.
  2. Right-click on the mimic and, from the pop-up menu, select Mimic Translations.Export.

The export process proceeds without further interaction with the user unless any errors are encountered.

You can also select the Export command from the menu using Window.Mimic Translations.Export.

How to export the strings from a single symbol

  1. Open any mimic or mimic template that contains an instance of the symbol.
  2. Right-click on the symbol. From the pop-up menu, select Symbol Translations.Export.

The export process proceeds without further interaction with the user unless any errors are encountered.

You can also select the export command from the menu using Edit.Symbol Translations.Export.
