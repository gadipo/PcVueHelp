Overview of the HMI Options

Select the menu Configure.HMI Options to display the HMI Options Tree.

The tree contains a large number of properties, listed by category. They can be used for fine-tuning of the HMI. In most projects, very few of the HMI options will need to be changed.

The following categories are available.

  • General
  • Mimics
  • File formats
  • Regions
  • Scripting
  • Compatibility from 16 to 32 bit version
  • Soft keypad management
  • Low resource warnings
