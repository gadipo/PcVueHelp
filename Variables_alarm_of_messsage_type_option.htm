Using the Latch Behavior Property

The Latch Behavior property (formerly called Message type) is found in an alarm variable's Advanced tab and can be used to modify the behavior of alarm variables that have either an Internal or SNMP source. It cannot be used with any other source.

The behavior is best explained using diagrams. Note that for an internal variable alarm activation depends on which of the properties Positive (bit value is 1) or Negative (bit value is 0), also found in the Advanced tab, is selected. The default is Positive (bit value is 1).

In a multi-station application, Latch Behavior is only active where the variable is produced (on the server).
