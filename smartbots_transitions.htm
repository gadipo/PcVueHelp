Transitions

The following are all ways to create a new transition in a bot.

Context menu

On the State item in the tree view, click right mouse button to open the context menu and select New transition. Show picture

A new transition from the selected State to the Initial state will be created. You can then modify the destination of the transition in the Properties panel.

New transition button

Select the source state for the transition and use the New transition button in the tool bar. Show picture

A new transition from the selected state to the Initial state will be created. You can then modify the destination of the transition in the properties panel.

Mouse over

  1. Put the mouse over the source state for the transition on the Blueprint. Small squares appear around the state element. Show picture

  2. Hold the Left mouse button on one of the squares and drag the mouse to the destination State. Once the mouse will be over the destination State, similar squares appear around the destination state element. Show picture

  3. Drop the mouse button over one of the squares to create a transition. Show picture

Transition properties

Each transition state has the following properties.

Property name Type Description
Name String Name of the transition. Name must be unique among all bot transitions
Next state State Destination state for the transition

If the bot agent is in the source transition state, it will move to the destination state once all associated conditions are satisfied. Note that conditions need to be satisfied at least once, which can happen at different times for each conditions. Once a condition is satisfied, it will be marked as such and will not be tested further.
