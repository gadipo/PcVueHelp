Overview of the Color Palette

The Color Palette is displayed each time you select a color property from any drawing element's Properties dialog and may also be displayed using the Display.Color Palette command. Show picture

It is essential to understand the use of each area of the Color Palette before starting any drawing.

  • Selected colors. The colors currently in use.
  • Transparent color. Click here to choose one of the selected colors as transparent. When you use a transparent color for a drawing, the background will be seen through it.
  • Non-indexed colors. When you color a drawing using one of the colors from this area, the RGB code of the color is saved in the drawing. There are 248 non-indexed colors in the palette. If you change the color in the palette, the color in the drawing is not affected.
  • Indexed colors. When you color a drawing using one of the colors from this area, the color index in the color palette is used as a reference. If you change the color in the palette, the color of all drawings using its index will be affected. There are 248 indexed colors in the palette.
  • Scroll bars. Used to scroll through the range of available colors. 32 non-indexed and 32 indexed colors are visible at any one time.

To differentiate an indexed color from a non-indexed color a small icon appears in the lower right corner of the color swatch in the drawing or animation properties dialog. Show picture

  • If you click on a color swatch that contains an indexed color then the corresponding color in the palette is highlighted.
  • If you double click on a color swatch that contains an indexed color then the Color Picker dialog opens. Clicking OK in the Color Picker dialog closes it and the color in the color swatch is changed from indexed to non-indexed (same RGB).
  • If you right click the small icon in the lower right corner of an indexed color swatch, the color is changed from indexed to non-indexed (same RGB).

Changes in the color palette after version 10.0 of the PcVue software

  • There are 248 non-indexed and 248 indexed colors. The color palette of version 10.0 and earlier had 64 colors in total.
  • Several of the non-indexed colors (33 to 96) are pre-configured as gradients.
  • Indexed colors 209 to 248 are used in PcVue's pre-configured libraries. They should not be changed by the user if the pre-configured libraries are used.

Unless otherwise stated, it is good practice to use indexed colors and document their use as part of the project specification (for example, Index 1 for mimic background, index 2 for button background, index 3 for …). In particular, it saves engineering time if late changes in colors are required at the time of commissioning or to take into account users’ feedback. If correctly documented and used the only change required will be in the color palette. No change will be required in the mimics and symbols you have already created.
