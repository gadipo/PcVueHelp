Item Method

 Applies to 

Returns an object from a collection - the one that matches the arguments.

Syntax

object.Item(index As Variant) As Object
-or for the Mimics Collection only-
object.Item(index As Variant, [branch As Variant],[referenceSet As Variant], [region As Variant]) As Mimic
 

Part

Description

index

A Long with a range from 1 to Variables.Count, Graphics.count, Mimics.Countor Windows.Count representing the position of the item to be returned within the collection.
-or-
A String representing the name of the item to be returned within the collection. (Not when used with the Windowscollection.)

branch

Optional argument used only with the Mimics collection. A String, which represents the name of the branch of the mimic object, that specifies the item that must be returned. Default: ""

referenceSet

Optional argument used only with the Mimicscollection. The fvRefSet that must be passed to open the Mimic. referenceSetcan take one of the following fvRefSetvalues:

fvRefSetRealtime
fvRefSet1
fvRefSet2
fvRefSetTest

Default: fvRefSetRealtime

region

Optional argument used only with the Mimics collection. A Long that specifies the region that must be used to open the Mimic. Default: 1

Standard methods in Microsoft VBA include: Item (default) and ItemByID.
