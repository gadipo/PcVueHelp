Context Menus in the Graphic Explorer

The Graphic Explorer provides two context menus for access to tools associated with a graphic object:

  1. With the mimic open in Design mode, click on a graphic object's name in the Graphic Explorer's objects list.
  2. Right-click on the graphic object's name to open a context menu. Show picture

The Edit context menu

The Edit context menu is available only for objects that are located in the mimic. It is not available for an object located inside a group or symbol.

It is the same context menu as is displayed by right-clicking on a graphic object in a mimic.

Animations context menu

The Animations context menu is available for objects in a group or symbol. It can be used to add an animation to an object inside a group or symbol without having to un-group the group or symbol.

It is the same context menu as is displayed from Animate in the main menu or in the context menu displayed by right-clicking on the object and selecting Animate.
