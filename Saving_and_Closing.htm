Saving and Closing Mimics

Saving a Mimic

To save a mimic click the Save tool on the standard toolbar or select the command File.Save.

If you are saving a mimic for the first time then the Mimic Management dialog will be displayed and you must enter a mimic name. This name will be used in the future when opening the mimic and is also the name of the file in which the mimic definition will be saved.

Closing a Mimic

To close a mimic click the Close tool on the standard toolbar or select the command File.Close from the File menu.

You will be prompted to save the changes, if changes have been made to the mimic since it was opened and they have not been saved.

If the mimic is not displayed at 100% (normal) magnification, any changes to mimic size and position are not saved.
