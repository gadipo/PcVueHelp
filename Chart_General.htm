Configuring a Chart's General Features

This topic is for the features configured by the Chart's General tab that affect the Charts general appearance and operation Show picture

Title

The text, its position and its style, for the Chart's title.

The text is bilingual so a different string can be entered for Language Configuration 1 and Language Configuration 2 if the Bilingual option is selected in the Project Language settings.

If changing the font be sure to check that the selected font is available of the target platform.

The Style property is different to the other viewers and can be selected from Default, Embed, Emboss, Frame and Shadow. For compatibility with the other Viewers Default should be used.

Legend

The legend displays a description for each series plus its corresponding color and style. Show picture

The color swatch selects the background color of the legend display - the text is always black. You can select the position from Top, Bottom, Left, Right or None.

If a Description is not entered in the Series tab then the Description from the variable configuration is used. If the variable configuration does not have a description then the variable name itself is used.

Colors

Color swatches to select the colors of the background (frame) and plot (chart) areas.

3D

3D - Enable the 3D display style. The Chart may take significantly longer to render if using this style. Show picture
