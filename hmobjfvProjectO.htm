fvProject Object

 See also Example Properties Methods Events 

The fvProject object represents the currently opened project.

This object is available through the ActiveProject property of the Application object.

ThisProject is a pre-defined instance of the object.

Its properties include the UserRights object.

When available, these collection objects that behave as properties of the fvProject object:
Languages, Mimics, Recipes, Roles, Variables, Windows.
