Opening a Child Window

You can open a window as a child of another, already open, window. Child windows behave in a slightly different way.

  • Child windows stay on top of the parent window until it is closed. You cannot move a parent window on top of a child window.
  • When you close the parent window, any child windows also close.
  • A child window may only be moved within the bounds of the parent window.

To open a window as a child you must specify the name of the parent window using the Included in - Window property. Note that if the parent window has been opened with a branch then this also must also be specified. Show picture

If you use the substitutionstring * as the name of the Included In window, the window will always be opened as a child of the window in which the animation resides.

Similarly if you use * for the Branch, the child window will inherit the branch of the window in which the animation resides. This is very useful when including the animation in a generic graphic object which may be inserted in any window.
