Window Properties in WebVue

Display tab

Property WebVue support
Positioning on Workspace Yes
Background color Yes
Link mimic with window size Yes
Allow mimic to be larger than window Yes
Scale mimic to fit window Yes
Title Partial - Only static strings are supported, the use of substitution strings may have limitations.
System menu Yes
Iconizable Partial - If a mimic is minimized in the PcVue desktop application, it is displayed as a small square in the upper right corner. In WebVue, it is minimized to only the small title bar and shown in the upper left corner. The title bar is moveable to position the minimized window wherever needed.
Maximize Yes
Sizeable Partial - Sizeable is only available if Border is also enabled. If mimic is larger than window, it is possible to shrink and grow the window.
Moveable Yes
Foreground Partial - The foreground property is effective except in one special case.
If a mimic which has the foreground flag opens another mimic that does not have it, the new opened mimic is in the Background in the PcVue desktop application, in WebVue the new opened mimic will be in foreground.

Static Border Yes
