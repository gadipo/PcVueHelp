Resizing 3D models

When you use the resizing tools, models are resized along the 3D world axes that are parallel to the window borders. When they are displayed in a window using an orthographic camera mode, a 3D model's height and width are defined as follows.

Dimension

Meaning

Height

Size of the right (or left) side.

Width

Size of the top (or bottom) side.

The model's sides that are used are those of a virtual rectangle (not displayed) that is just big enough to encompass the 2D projection of the model's bounding box.
