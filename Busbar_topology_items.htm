Topology Items

All the Topology Engine items can be created with the configuration tool. The name of each item is uniquely assigned by the configuration tool and cannot be changed. All the other properties of the items can be edited. All items configuration dialogs have 3 buttons.

  • Reset - Undo last changes and restore default values for the item,
  • Apply - Apply the changes immediately (new start point for the Reset button)
  • Delete - Delete the selected item

Any changes you make are automatically applied whenever you select another item.

Voltages Configuration

The Voltages Configuration is accessed from the associated button in the menu bar. For each Voltage Level, it is possible to specify its bounds. Voltage levels must be ordered from the lowest (V0) to the highest (V6). Their bounds cannot cross one another (V1 max must be lower or equal to V2 min). Show picture

Networks

The first item to create is a Network. To insert a new Network, click the Add Network menu button. The project must have at least one network. Networks are used to split an electrical system into logical pieces. You can for example create one Network per mimic you need to design. The other types of items (nodes, switches…) can be linked together only if they belong to the same network.

The only property of a Network is its label. This label is used in the tree on the left side of the configuration tool. Show picture

Nodes

To insert a new Node, select a Network and click the Add Node menu button. There are 4 types of Node.

  • Passive,
  • Static,
  • Measured – Manual,
  • Measured – Communicating.

The common properties are as follows.

  • Label - The name used in the tree on the left side,
  • Branch - The branch used to instantiate the corresponding template,
  • Network - The Network that the Node belongs to,
  • Default Injection - The default voltage level of the Node,
  • Injection Type - The type of the Node.

For Passive and Static Nodes, no additional properties are required. Show picture

For Measured – Manual Nodes, the additional property Injection Variable Scale is required. It is possible to specify the scale for of the voltage value you set (1V, 10V, 100V or 1kV). Show picture

Finally, for Measured – Communication Nodes, two additional properties are required.

  • Injection Variable Scale, which works as for Measured – Manual Nodes,
  • Injection Variable Name, to specify the field variable that the Node is linked to. This variable can be a Bit or a Register.

Switches

To insert a new Switch, select a network and click the Add Switch menu button.

There are 2 types of Switch.

  • Manual,
  • Communicating.

The common properties are as follows.

  • Label - The name used in the tree at the left side of the tool,
  • Branch - The branch used to instantiate the corresponding template,
  • Network - The Network that the Switch belongs to,
  • Upstream Node - The first Node that the Switch is linked to,
  • Downstream Node - The second Node that the Switch is linked to,
  • Default Position Value - The default position of the Switch,
  • Switch Type - The type of the Switch.

According to the symbols library and as usual on single line diagrams.

  • The Upstream Node corresponds to the Node at the top or left side of the Switch in the diagram,
  • The Downstream Node corresponds to the Node at the bottom or right side of the Switch in the diagram.

For Manual Switches, no additional properties are required. Show picture

For Communicating Switches, the additional property Position Variable Name is required. It is used to specify the field variable that the Switch is linked to. This variable can be a Bit or a Register. You also have to specify the values corresponding to the Open and the Close positions. Show picture

Transformers

To insert a new Transformer, select a network and click the Add Transformer menu button.

Transformers’ properties are as follows.

  • Label - The name used in the tree at the left side of the tool,
  • Branch - The branch used to instantiate the corresponding template,
  • Network - The Network the Transformer belongs to,
  • Upstream Node - The first Node that the Transformer is linked to,
  • Upstream Voltage - The supposed Voltage level of the first Node,
  • Downstream Node - The second Node that the Transformer is linked to,
  • Downstream Voltage - The supposed Voltage level of the second Node.

According to the symbols library, and as usual on single line diagrams, the Upstream Node corresponds to the Node at the top or left side of the Transformer in the diagram. And the Downstream Node corresponds to the Node at the bottom or right side of the Transformer in the diagram. Show picture
