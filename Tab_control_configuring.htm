Configuring the Tab Control

The Tab Control is enabled by ticking the Enable Tab Control box in the Tab Control tab of the Window Properties dialog. Show picture

Configuring the size and position of the tab control

The size and location of the tab control are determined by the following properties.

  • X Position and Y Position - The location of the top left hand corner of the tab, in pixels, relative to the top left hand corner of the container mimic.
  • Height- The height of the tab control, including the tab itself, in pixels.
  • Width - The width of the tab control in pixels.

Since the tab control does not have to cover all of the container mimic, you can place drawing elements (including controls) in any unused part of the container mimic. Show picture

Automatically adjusting the size of the tab control according to the size of the container mimic

Using the Anchor Right and/or Anchor Bottom tick boxes, you can anchor the right and/or bottom of the tab control to the right and/or bottom borders of the container mimic. If the mimic is resized, the size of the tab control automatically follows. The Margin Right and Margin Bottom properties can be used to configure a margin between the tab control and the mimic. These properties are only enabled when the corresponding Anchor property is ticked.

When you tick Anchor Right the Width property is disabled. When you tick Anchor Bottom the Height property is disabled.

Configuring the icon size and text font

The Icon width and Icon height properties set the size of the icon (if any) displayed in the page's tab. The application scales the icon to suit the configured size. Icons are most commonly 32 x 32 pixels. The Font checkbox enables or disables the selection of a custom font. The Font button is used to display the Font dialog to let you select a custom font. When the custom font feature is not enabled, the system font is used.
