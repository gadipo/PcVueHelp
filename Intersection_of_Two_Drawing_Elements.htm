Intersection of Two Drawing Elements

The Intersection tool creates a polygon that is equivalent to the overlapping area of two drawing elements.

How to use the intersection tool

  • Select the first drawing element by clicking with the left mouse button.
  • Select the second drawing element holding the Shift button so that both drawing elements are selected.
  • Select the Intersection tool from the Arrange toolbar. A new polygon is created taking the color from the second shape.
  • Click and drag the polygon to its location. The original drawing elements remain unchanged.
