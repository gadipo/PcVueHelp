Overview of this Primer

If you are familiar with 3D software tools such as modelers or viewers, you may prefer to skip this section. Otherwise, it is essential for understanding the rest of this book.

The 3D tools are for representing a mimic with three-dimensional properties on a two-dimensional screen. The views of the mimic depict the movements of objects within a 3D mimic, from a user or camera's viewpoint.

This is different to giving static or animated 2D objects a 3D sculpted appearance. 3D objects are given a dynamically shaped appearance from all perspectives.
