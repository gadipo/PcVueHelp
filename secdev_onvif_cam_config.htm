How to add an ONVIF camera

ONVIF cameras are configured in PcVue via the Application Explorer.

  1. Select the Security devices network to which the camera will be added. From the task pane, select Add an ONVIF camera to open the ONVIF Camera dialog. Show picture
  2. Enter a Name for the camera. This is the alias by which the camera will be known in PcVue. You can also enter an optional Description. This is for reference only and does not appear elsewhere.
  3. Enter the camera IP address.
  4. Click on OK to confirm the configuration and close the dialog.

The Advanced tab

  • HTTP port - The HTTP port number. Default 80.

  • Authentication - Credentials to connect to the camera if required.

    • User name - The user name to connect to the camera.

    • Password - The password to connect to the camera.
