Animated GIF and AVI files
 

 
 
 
 
 
 
 
 
 
 
 
 
 
Using Animated GIF and AVI Files
 
Animated GIF or AVI files produce an animated image 
 by playing a number of frames at a predetermined interval. The number 
 of frames and the interval are fixed when the image is designed. 
 
Images of this type have an extra tab called Settings 
 in the Image Properties dialog to control their behavior at run time.

Using the Base tab to change an image's appearance
 

 
 
 
 
 
 
 
 
 
 
 
 
 
Changing the Appearance of an Image
 

 
To illustrate this topic, the author has used an image of the color palette, not the color palette itself.
 
 
Using the Transparent Color property
 
Some image formats support native transparency. That is one of the colors is defined as transparent and any background color is visible through it.
 
For image formats that do not support transparency, the HMI allows you to pick one of the image's (non-transparent) colors as transparent. When the image is rendered by the HMI, the background color shows wherever the color designated as transparent occurs. Show picture

 
To select an image color as transparent:
 
 
Display the Image Properties dialog by double-clicking on the image.
 
Select the Image tab and tick Transparent 
 color. The cursor will change shape to a small circle.
 
Click on the area 
 of the image that contains the color to be made transparent.
 
Click the OK button to confirm 
 the change. Show picture

 
 
Adding a pattern to an image
 
As an alternative to displaying the background, you can replace the transparent color of an image using a pattern, identical to that of any solid drawing element, using the properties configured in the Base tab. Show picture

 
Adding a style to an image
 
You can change the style (button, relief, shadow etc.) of an image, as with any solid drawing element, using the Appearance properties configured in the Aspect tab. Show picture

 

 
Changing the style increases the space occupied by the image. You will be prompted to either increase the object size or rescale the image to retain the original size. It is preferable to increase the object size as re-scaling an image inevitably degrades its appearance.

Effects tab
 

 
 
 
 
 
 
 
 
 
 
 
 
 
Changing the Image on Mouse Over
 
The Effects tab allows you to select alternative images 
 to be displayed at run time when the pointer is over the image (mouse over), or the image is selected by pointing and clicking (mouse down). 
 This is particularly useful if the image is being used to start a VBA 
 program and does not have any associated control zone animation. Show picture
 

 

 
 
 
 
When selecting the alternative images the following points 
 should be noted.
 
 
If a transparent 
 color was used with the main image, the same color will also be treated 
 as transparent in the images selected in the Effects tab.
 
The image selected 
 in the effects tab will be displayed as the same physical size as main 
 image. If the images are of different sizes, they will be scaled accordingly.

Advanced Image Options
 

 
 
 
 
 
 
 
 
 
 
 
 
 
Advanced Image Properties
 
The following advanced image properties are all found in the Picture tab of the image dialog.
 

 
To illustrate this topic, the author has used an image of the color palette, not the color palette itself.
 
 
EMF+ rendering
 
The EMF+ rendering property is available for WMF and EMF image formats only. 
 
Selecting EMF+ rendering instructs the HMI to use the EMF+ format for rendering the image. The main effect of this is to support image anti-aliasing. Show picture

 
Background blend
 
The Background blend property is available for PNG image format only. 
 
Selecting the Background blend property causes the HMI to blend (add) the image RGB values to those of the background. For example, if the image color was blue (0, 0, 255) and the background color was green (0, 255, 0) the color rendered would be cyan (0, 255, 255). Show picture

 

 
The color swab adjacent to the Background blend property in the Picture tab is only to preview the effect that any particular background color will have on the blended image. It has no effect on the image rendered in the mimic.
 
 
Opacity
 
The Opacity property is available for PNG image format only. 
 
Changing the Opacity property allows the background color to be seen through the image. The range of values for Opacity is 0 (totally transparent) to 255 (opaque and the default). Show picture
