SendSMS Method

 Applies to 

Composes a short message (SMS, text) for sending.

Syntax

Function SendSMS(
    SmsProfile As String,
    PhoneNumbers As String,
    Subject As String,
    Message As String,
    [SmsFormat As fvSmsFormat]
) As Boolean

Return value

True if the e-mail is sent successfully, otherwise False.

Part

Description

SMSProfile

The name of the SMS profile used to send the message. The profile provides information such as the serial port of the modem and the pin code of the SIM card for the modem.

PhoneNumbers

A list of phone numbers of the recipients. Semicolon-delimited.

Subject

The subject line of the message.

Message

The body of the message.

SMSFormat

The content type of the body of the message, depending on the service operator, modem and mobile device used. (fvSMSFormatAuto = 0, fvSMSFormatText, fvSMSFormatPDU7Bit, fvSMSFormatUnicode). Default: fvSMSFormatAuto.

The parameters' values are enumerated as follows.

Enumeration

Description

 

 

fvSMSFormat

 

fvSMSFormatText (= 1)

Plain text

fvSMSFormatPDU7Bit (= 2)

PDU 7 bit

fvSMSFormatUnicode (= 3)

Unicode

The sending status is recorded in the system variable SYSTEM.SMS.ProfileName.STATUS and in the Event Viewer. For the system variables's values, see the topic The Application Explorer.Variables.System Variables.Message Status Variables.

Example

Dim r As Boolean

r = SendSMS("SmsProfile", "0033661143609", "VBA test", "toto")
