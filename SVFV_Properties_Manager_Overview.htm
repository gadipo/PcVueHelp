Overview of the Drawing Properties Manager

Each drawing element has a set of properties appropriate to its function. These are accessed by right-clicking on the element and selecting Properties to open the Drawing Properties dialog. When you create an element, it inherits the properties of the most recently created element of the same kind. If no such element has previously been created in the session then global settings are used.

The Drawing Properties Manager gives you greater control over the ways in which properties are applied to drawing elements, both individually and in multiple selections. It also enables you to copy the properties of one element (called the reference element) to others.

  • Select the menu command Display.Drawing Properties Manager to open The Drawing Properties Manager dialog.

Operational modes

The Drawing Properties Manager operates in three modes.

  • From the drop-down list box at the top of the dialog, select a mode as follows.

Mode

Purpose

Global

To define the initial property settings of all subsequently drawn drawing elements.

Selection

To modify the properties of selected drawing elements.

Reference Object

To copy the properties of the reference drawing element (i.e. last selected) to the other elements in the selection.

In addition, you are able to create customized modes known as models. A model is a set of pre-defined values for selected properties only. A model can be used both to provide the initial property settings (as in Default mode) and to copy the properties of the reference drawing element to the other elements in the selection (as in Reference Object mode).

These properties are applied only when the Drawing Properties Manager is open and when the mimic is in Design mode.

Property tabs

The properties that are affected by the dialog are contained in six tabs. In Default mode, all tabs are always visible. In other modes, the tabs that are visible depend on the types of drawing elements that are selected. Whilst the six tabs have names similar to those found in the drawing elements' Properties dialogs, the content is not exactly the same.

  • Base - Pattern, border and shape properties. (Shape properties are in the Aspect tab of the drawing element's Properties dialog).
  • Text - Font, text colors and text format properties.
  • Picture - Image and playback properties.
  • Aspect - Appearance properties.
  • Visibility - Visibility bound and layers properties (Visibility properties are in the Advanced section of the Base tab in the drawing element's Properties dialog).
  • 3D Visibility - Visibility of 3D objects in terms of distance to camera, hidden sides and layers. This tab is only visible when at least one 3D object is selected.

If the active mimic (the one with focus) is in Run mode, a blank tab named None is displayed.

Storage

The values of properties used in Global mode are saved in the project's configuration folder, in the file DRAWDEFAULT.DAT.

The values of properties used in each of the property models are saved in the project's configuration folder in the file DRAWMODELS.DAT.

You can pre-define the global values of drawing properties in the Project Library and/or Shared Library folder (using the same file names, DRAWDEFAULT.DAT and DRAWMODELS.DAT).
