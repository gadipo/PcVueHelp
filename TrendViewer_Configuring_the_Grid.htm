Configuring the Trend Viewer's Grid

The Grid tab enables you to configure the appearance of the grid displayed in the chart area at run-time.

How the properties in the Aspect group affect the grid

  • Automatic - See below.
  • Display horizontal subdivisions - Enable display of the horizontal (Y-axis, value) subdivisions. The number of subdivisions is automatic and cannot be changed.
  • Display vertical subdivisions - Enable display of the vertical (X-axis, time) subdivisions.
  • Vertical line - the color and style for the lines representing the vertical divisions and subdivisions. Only available if Automatic is not ticked.
  • Horizontal line - the color and style for the lines representing the horizontal divisions and subdivisions. Only available if Automatic is not ticked.

What effect does the Aspect.Automatic property have on the grid?

  • If Automatic is ticked
    • If one or more of the traces have their scale enabled, the vertical and horizontal divisions and subdivisions are displayed as solid lines in the color of the rightmost scale. Show picture

    • If none of the traces has their scale enabled, only the vertical divisions and sub divisions are displayed in solid lines and in the default colors. Show picture

  • If Automatic is not ticked
    • If one or more of the traces have their scale enabled, the vertical and horizontal divisions and subdivisions are displayed in the line styles and colors defined in the Grid tab. Show picture

    • If none of the traces have their scale enabled, only the vertical divisions and sub divisions are displayed in the line styles and colors defined in the Grid tab. Show picture

How the properties in the Time-scale group affect the grid

  • Automatic - The time period represented by each vertical division and subdivision is automatically calculated depending on the total period of the horizontal axis. For example, if the horizontal axis is one hour, the divisions are 30 minutes and the sub-divisions 5 minutes.
  • Division - The period for the division. If Automatic is not selected.
  • Subdivision - The period for the subdivision. If Automatic is not selected.

If you do not use the Automatic option for the time scale there is the possibility that, when you zoom out (increase the period of the chart), there will be so many grid lines that they will obscure the chart.
