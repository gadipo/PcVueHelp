About Maintaining a Database

The functions described in this book are accessed from the Database Manager. To do so, you must first select a particular database: Show picture

  1. Open the Database Manager from the Windows Start button (or a desktop icon, if you have created one).
  2. Double-click on Access Databases or SQL Server Databases to open that part of the tree structure.
  3. Click on a database's name to select it.
  4. Right-click on a database's name to display its context menu. It is different for:
    • Access databases. Show picture

    • SQL Server databases. Show picture
