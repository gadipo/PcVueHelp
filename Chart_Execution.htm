Selecting a Chart's Run-time Tools

This topic is for the features configured by the Chart's Execution tab that select the Chart's run-time tools. Show picture

Toolbar

The position, width and configuration of the run-time toolbar. The width can be configured from 15 to 60 pixels. The icons representing the tools auto-size to accommodate the toolbar width.

Unlike the other viewers, the tools that are available from the toolbar are configured at design time not at run-time.

Scrollbar

The scrollbar is used to scroll the chart area when any of the zoom tools have been used. It does not appear when the chart zoom is 100% even if display of the scrollbar is enabled.

You can choose the scrollbar position from either inside Show picture, or outside the chart area. Show picture

On mouse over

Configures the behavior when the mouse hovers over a data point from one of the series.

  • Do nothing - Nothing is displayed.
  • Display value in tooltip - The values corresponding to the X and Y co-ordinates, and the timestamp, are displayed in a tooltip.
  • Display value in graph - The values corresponding to the X and Y co-ordinates, and the timestamp, are displayed in the chart. If there are multiple points with the same co-ordinates but different timestamps then they are all displayed. Show picture
  • Display all data points with the same X value - Display all data points from all series that have the same X value, in the chart.

Other tools

  • Right click pop-up menu - Enable the right click pop up (context) menu. When enabled, this menu is displayed when right clicking anywhere in the chart area at run-time and contains the same tools as the toolbar.
