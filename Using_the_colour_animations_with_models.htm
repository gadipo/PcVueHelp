Using the Color Animations with 3D Models

The color of a 3D model is determined by the model's materials. A color animation, when applied to a 3D model, changes properties that are supplied by the materials - it does not change the materials themselves. For example, it can change the emissive color for a mesh, but it does not change the configuration for that color in the material used to color the mesh.

Throughout this book you will see references to animating of materials. This term is used to simplify the explanation. What you are actually doing is animating the color or other property of a mesh that, if it were not animated, would be supplied directly by the material.

What material properties can be changed?

A single color animation can change any or all of the following material properties.

  • Diffuse color
  • Opacity (percentage)
  • Specular color
  • Specular level (percentage)
  • Emissive color

How many color changes can I make?

When configuring a color animation you can select one or more materials. The same color changes are applied to all selected materials. The number of color changes that you can make with a single animation depends on the animation type. For more information, see the book on Color Animation in the main HMI Help.

What is the difference when applying the animation at model level or frame level?

The level at which you apply a color animation affects what materials and/or instances of materials it animates.

Level

The color animation is applied to

Symbol or group

All materials of all models in the symbol or group.

Model

One or more of the materials depending on how the animation is configured. The color changes are applied to all frames where the selected materials are used.

Frame without a mesh

All materials used by its sub-frames.

Frame with a mesh

One or more of the materials used by the mesh and/or all materials used by any sub-frames.
