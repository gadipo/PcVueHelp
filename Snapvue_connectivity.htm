Diagnostic Views

The Diagnostic views are only available if Diagnostic is enabled in the Settings view.

Connectivity view

The Connectivity view shows the status of the connectivity features available on the device. In the following screen shot, all features other than NFC are available. Show picture

Environment view

The Environment view shows the geo-tags detected by the device. By default, only the GPS co-ordinates and geo-tags configured in the PcVueserver are displayed. The geo-tag display includes the name given in the PcVueconfiguration. Show picture

If SHOW ALL TAGS is selected, all detected geo-tags are displayed irrespective of if they are configured in the PcVueserver or not.
