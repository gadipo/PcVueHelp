Message Settings

The message settings are displayed from the Settings task when the Messages folder is selected in the Application Explorer's configuration tree. These settings affect the general operation of the message system within PcVue. Show picture

  • E-mail
    • Default e-mail profile - The e-mail profile used as a default when configuring a message. The default is None.
    • Default attachment encoding - The default encoding type for attachments. In version 10.0 of PcVue, only MIME encoding is supported.
    • Prohibited file extensions - Any file extensions that are prohibited for attached files, as a comma separated list.
    • Maximum attachment size - The maximum size of any attached file in KB. The default is 1000 KB.
  • SMS
    • Default SMS profile - The SMS profile used as a default when configuring a message. The default is None.
