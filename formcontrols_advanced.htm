Form Controls Advanced Options (Advanced tab)

Combo-box

  • Enable suggestion - Allows the user to type in the combo-box. The typed text is compared to the combo-box configured texts and matches are displayed in a list from where they can be selected. Show picture

  • Mode

    • Starts with - The typed text is matched to the start of the configured texts.

    • Contains - The typed text is matched anywhere with the configured texts.
