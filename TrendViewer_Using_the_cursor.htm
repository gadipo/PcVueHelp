Using the Trend Viewer Cursor

The cursor is used to interact with the chart area. Using the cursor, you can:

  • Display the value of each of the traces at a particular point in time.
  • Zoom the trend in the time (horizontal) axis.
  • Zoom the trend simultaneously in both the time (horizontal) axis and value (vertical axis).

You select the operating mode for the cursor from either the toolbar or a pop-up menu displayed by right clicking anywhere in the chart area. The cursor can be used in real-time, pause or historical modes. If you use it whilst running in real-time mode, the Trend Viewer will automatically switch to pause mode.

How to display the value of the traces

  1. Select the cursor or cross-hair cursor tool.
  2. Click anywhere in the chart area. The values for the traces at that point will be displayed in the Legend area or in the Scale area if visible.

How to zoom the time axis

  1. Select the zoom time axis tool. The mouse pointer will appear as a magnifying glass whilst it is over the chart area.
  2. Click on the chart at the point representing the new start time for the chart. Hold down the mouse button and drag the cursor to the point representing the new end time. Release the mouse button.

How to zoom both axes

  1. Select the zoom area tool. The mouse pointer will appear as a magnifying glass whilst it is over the chart area.
  2. Click on the chart at the point representing the new start time and maximum value for the chart. Hold down the mouse button and drag the cursor to the point representing the new end time and minimum value. Release the mouse button. The chart area will be redrawn with the new horizontal (time) and vertical (range) axes. Whilst the vertical axis is zoomed, a vertical scroll bar will appear on the right of the chart area, which may be used to scroll the viewable part of the chart within the original vertical axis range.
