BasicPrint Method

Applies to 

Initiates printing of lines in the object.

Syntax

object.BasicPrint (aiMode As Integer, format As Integer) As Boolean

Part

Description

aiMode

Specifies which lines to print:

0 all lines

1 selected lines

2 displayed lines.

format

Print formatting:

1 as configured in the ActiveX

0 separate the Alarm Viewer fields with tabs

(available from version 7.20c of the software onwards).

Equivalent to the Print button on the object's toolbar.
