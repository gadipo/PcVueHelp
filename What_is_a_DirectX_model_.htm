What is a 3D DirectX Model?

A 3D DirectX model is a file that contains information to render a 3D drawing. It can represent anything from a simple 3D shape such as a sphere to a highly complex drawing such as an entire building.

Whatever it represents, a 3D model contains the following information.

  • Frames
  • Pivot
  • Meshes
  • Materials

Frames

A frame is a container that applies geometric transformation (position, rotation and scaling) to its contents. A frame contains a pivot about which rotation takes places, and either a single mesh, other frames (known as child or sub frames) or both.

A frame becomes a child frame when it is linked to a (parent) frame during the model design. A child frame does not have to be contained with the bounds of the parent frame. As well as its own transformation, a child frame inherits any transformation of its parent frame, moving relative to the parent frames' pivot.

Pivot

The pivot is the point about which rotation takes place. The position of the pivot determines the model or frames location within a 3D mimic. Each model has a pivot point and each frame and child frame within a model has a pivot point.

Meshes

A mesh provides the surface geometry and is made up of a collection of faces (polygons) bound together. A mesh is always contained within a frame. A mesh references one or more materials.

Materials

A material provides the visual representation of the surface created by a mesh. The definition of a material may be contained within the mesh it is used or within the model. A material that is defined within the model can be used by several meshes.

Lighting

A major difference from other 3D applications is that there are no explicit light sources, or reflections, in the HMI. Even though any object can be configured with a glowing color, it cannot illuminate other objects, nor can it reflect them in a surface of its material.
