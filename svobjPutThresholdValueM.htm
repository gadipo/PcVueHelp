PutThresholdValue Method

 Applies to 

Description

Set the value of the specified threshold.

Syntax

PutThresholdValue(ThresholdNumber As Integer, NewValue As Double)

Part

Description

ThresholdNumber

The number of the threshold. Range 0 to 3.

NewValue

The new value for the threshold.

Example

Private Sub Shape1_Click()
  Dim c1 As TrdCurve
  Set c1 = Trend1.GetCurve(1)
  c1.putThresholdValue 0, 900
End Sub
