Using the Log Viewer's Time Filter

The Log Viewer Time Filter dialog is opened from the toolbar using the button and is used to search for records from a particular time.

Using it you can either enter a From date and time and a search direction Show picture, or a From and To date and time Show picture.

  • Later - Searches for records forward in time starting at the From date and time.
  • Earlier - Searches for records backwards in time starting at the From date and time.

The number of records returned is limited by the size of the display buffer.
