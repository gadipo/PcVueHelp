Arrange toolbar
 

 
 
 
 
 
 
 
 
 
 
 
The Arrange Toolbar
 
Drawing elements are resized, moved, aligned, locked 
 and grouped using the Arrange toolbar.
 

 
Most of the icons on the Arrange toolbar only become active 
 when a drawing element - or at least a pair of drawing elements - is selected.
 
 
Click to:
 
 
 
 

 
 

 
 
 
 
 
Lock a drawing element. 
 
 
 

 
 

 
 
 
 
 
Unlock a drawing element.
 
 
 

 
 

 
 
 
 
 
Toggle (turn on and off) display of the drawing grid.
 
 
 

 
 

 
 
 
 
 
Toggle (turn on and off) the alignment of drawing elements 
 to the grid.
 
 
 

 
 

 
 
 
 
 
Un-group the selected drawing group.
 
 
 

 
 

 
 
 
 
 
Group the selected drawing elements.
 
 
 
 
Click to flip a drawing element
 
 
 
 

 
 

 
 
 
 
 
About its vertical axis. 
 
 
 

 
 

 
 
 
 
 
About its horizontal axis. 
 
 
 
 
Click to move one or more drawing elements:
 
 
 
 

 
 

 
 
 
 
 
To the front.
 
 
 

 
 

 
 
 
 
 
To the back.
 
 
 

 
 

 
 
 
 
 
Forward.
 
 
 

 
 

 
 
 
 
 
Backward
 
 
 
 
For more information, see the topic Drawing 
 element ordering. 
 

 
For the next set of tools, the reference is the drawing 
 element with solid handles. The other selected elements line up with it.
 
 
Click to move the selected drawing elements:
 
 
 
 

 
 

 
 
 
 
 
Horizontally so that the left edge aligns with that of 
 the reference.
 
 
 

 
 

 
 
 
 
 
Horizontally so that the right edge aligns with that of 
 the reference.
 
 
 

 
 

 
 
 
 
 
Vertically so that the upper edge aligns with that of 
 the reference.
 
 
 

 
 

 
 
 
 
 
Vertically so that the lower edge aligns with that of 
 the reference.
 
 
 

 
 

 
 
 
 
 
Horizontally so that the right edge aligns with the left 
 edge of the reference.
 
 
 

 
 

 
 
 
 
 
Horizontally so that the left edge aligns with the right 
 edge of the reference.
 
 
 

 
 

 
 
 
 
 
Vertically so that the top edge aligns with the 
 bottom edge of the reference.
 
 
 

 
 

 
 
 
 
 
Vertically so that the bottom edge aligns with the top 
 edge of the reference.
 
 
 

 
 

 
 
 
 
 
Horizontally so that the center aligns with that of the 
 reference.
 
 
 

 
 

 
 
 
 
 
Vertically so that the center aligns with that of the 
 reference.
 
 
 

 
 

 
 
 
 
 
Horizontally so that they are evenly spaced.
 
 
 

 
 

 
 
 
 
 
Vertically so that they are evenly spaced.
 
 
 
 
Click to resize the selected elements so that:
 
 
 
 

 
 

 
 
 
 
 
The width matches that of the reference.
 
 
 

 
 

 
 
 
 
 
The height matches that of the reference.
 
 
 

 
 

 
 
 
 
 
The width and height matches that of the reference.
 
 
 
 
Click to create a new polygon using Boolean operators 
 by:
 
 
 
 

 
 

 
 
 
 
 
Adding the sum of the areas two drawing elements (Union).
 
 
 

 
 

 
 
 
 
 
Taking the overlapping area of two drawing elements (Intersection).
 
 
 

 
 

 
 
 
 
 
Subtracting the overlapping area of one drawing element 
 from the other (Extraction).
 
 
 
 
Click to reverse the order of points:
 
 
 
 

 
 

 
 
 
 
 
Reverse the order of points in a polygon, polyline, B�zier 
 curve or B�zier solid. Handy when using the Boolean graphic tools and 
 the Path animation.
