BACnet Server PICS

1.1. Product Description

 PcVue is a SCADA Software that can act as a BACnet Application Specific Controller (B-ASC) through a dedicated Add-on. PcVue is able to communicate with BACnet workstations via BACnet/IP.

1.7. Device Address Binding

Is static device binding supported? No. This is currently necessary for two-way communication with MS/TP slaves and certain other devices.
