Load Method (Map Layer)

              

Loads layers and markers from the specified file. Old layers and markers are overwritten.

Syntax

object.Load(FilePath As String, Xpath As String, Async As Boolean)

Part

Description

FilePath

A string containing the name of the file including the path.

Xpath

XPath expression to filter loading markers. Optional.

Async

Indicates if the layers and markers should be loaded in a different process. Optional.
