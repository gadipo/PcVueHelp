Keeping a Graphic Object Visible When Scrolling Example

This example shows how to keep a graphic object visible (at the top left corner of a Window), whatever the scroll bar control positions may be.

Scrolling makes the drawing element overlap with the text in the mimic. That is the correct behaviour in this case.

'mimic WindowScroll event

Private Sub Mimic_WindowScroll(ByVal ScrollDirection As fvScrollDirection,_  

  ByVal ScrollRequest As fvScrollRequest, ByVal PreviousPosition As Long,_

  ByVal NewPosition As Long, ByVal Window As Integer)

 

'calculate the X or Y position of the corner according to

'  scroll bar position and window ZoomRatio

Dim pos As Long

pos = NewPosition * (100 / Me.Windows(Window).ZoomRatio)

 

'set the graphic objects left or top position according to scroll bar control

If ScrollDirection = fvScrollHorizontal Then

    Shape1.Left = pos

ElseIf ScrollDirection = fvScrollVertical Then

    Shape1.Top = pos

End If

 

End Sub
