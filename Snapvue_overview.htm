SnapVue Overview

SnapVue is a mobile application that displays contextual information on a mobile device such as a smart phone or tablet. By contextual we mean that the information depends on the location of the device, known as the zone, and the user's profile. The location of a device is detected using geo-tag technologies including Bluetooth beacons, QR codes and NFC tags.

The information is supplied by the PcVue Web & Mobile back end server and primarily comprises of the real-time and historical value of process variables. Show picture

See Web and Mobile Client Applications Overview for an overview of Web & Mobile client applications, including their common requirements and deployment steps.

The configuration steps specific for SnapVue access to PcVue are as follows:

  • Configure the Mobile Server settings in the Application Explorer. It includes the variable and alarm properties that should be displayed in the SnapVue app detailed views, and the default filter for alarm notification. See the book Mobile server.
  • Configure one or more SnapVue users. See the topics Properties that affect access to Web Apps and Web Services in the Application Explorer book and Configuring mobile applications users.
  • Configure the geographical data in the Application Explorer. The geographical data comprises of zones, the geo-tags corresponding to those zones and the actions available from each zone. See the book Geographical Data.
  • Have users install the SnapVue mobile application on their mobile device.

Summary of the main SnapVue views

The views are all accessed from a drawer menu opened by a swipe from the left border of the screen, or from the menu tool found at the top left of most views. Show picture

  • Home - The primary view comprising of one or more context sensitive tiles, one for each action available for the detected location. See the Home view topic.
  • Chats - Access to the optional Instant Messaging system.
  • Settings - SnapVue user configurable settings. See the Settings view topic.
  • Accounts - Configure user accounts and display their status. See the Accounts view topic.

SnapVue makes use of the Notification Area to notify the user of the status of selected alarms and current location detection.

Licensing SnapVue

The components necessary to run SnapVue are included on the PcVue distribution media or the Windows distribution media.

The maximum number of simultaneous SnapVue sessions is license-protected.

Terminology

For readers not familiar with mobile platforms terminology here are some of the terms in common use:

  • View - The term used for a graphic display page occupying the entire screen.
  • Notification area - A small area, not used by any app's views, reserved for notifications requiring the user's attention. Normally at the top of a smart phone or the bottom of a tablet.
  • Gestures - The finger movements to interact with a touch screen:
    • Touch or Tap - Touch the screen Press and lift immediately.
    • Long touch or Long tap - Touch the screen, wait, lift.
    • Swipe - Touch the screen, move, lift.
    • Drag - Long touch, move, lift.
    • Double touch or Double tap - Two touches in quick succession.
    • Pinch open - 2-finger touch, move outwards, lift.
    • Pinch close - 2-finger touch, move inwards, lift.
  • Drawer menu - A menu generally opened by a left swipe. Known as a drawer because of the way in which it opens.
