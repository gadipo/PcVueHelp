Changing the Appearance of a Shape

The appearance of a shape may be changed using the Aspect tab in the Drawing Properties dialog. Show picture

The following options are available.

Regular. The shape is displayed as a normal rectangle. You can change the fill style and color.

Button. The shape takes the appearance of a button. The fill color and style is fixed. The border size can be changed using the Thickness property.

Colored button. The shape takes the appearance of a colored button. You can change the fill style and color and the border size color.

Relief. The shape takes a sunken appearance. You can change the fill style and color and the border size and color.

Inverse Relief. The shape takes a raised appearance. You can change the fill style and color and the border size and color.

Shadow. The shape is displayed as a normal rectangle with a colored shadow. You can change the fill style and color and the shadow width and color.

If selecting the Button style you can also choose the option System look & feel. The button appearance will change to that of a button in the operating system.
