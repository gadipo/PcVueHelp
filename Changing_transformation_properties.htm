Changing Transformation Properties

The transformation properties determine a model or frames position with respect to its parent.

Property

Description and action

Pivot

The position of the pivot determines the position of the model or frame relative to its parent.

Scale

The scale of the model or frame relative to its parent.

Angle

The orientation of the model or frame relative to its parent

For a model, the parent is the entire mimic. For a frame, the parent is either the model or another frame depending on the location with the model hierarchy.

The default settings for a model (when a model is first added to a mimic) are Pivot (0, 0, 0), Scale (100%, 100%, 100%) and Angle (0, 0, 0).

The default settings for a frame will depend on the design of the model that it is part of.

It is important to understand that all properties are relative to the frame or model's parent. For example, if the scale of a frame is changed from 100% to 200% the size at which it, and any sub-frames, is rendered on the screen will double although the scale of the sub-frames remain unchanged.

How to change transformation properties

  1. Open the 3D Properties dialog by either double clicking on the model or its icon in the Graphic Explorer.
  2. Enter new values for the properties to be changed.
  3. Click the OK button to apply the changes and close the dialog
