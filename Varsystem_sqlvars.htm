Sql Variables Status Variables

Flow regulation.

SYSTEM. SqlVariables.XON R Bit Automatic

0 Flow regulation activated.
1 Flow regulation not activated.

Number of transactions lost during flow regulation.

SYSTEM. SqlVariables.Translost R Register Automatic

Reset to 0 when XON is 1. Incremented for each lost transaction.

Read and write flow regulation.

SYSTEM.<StationName>.SqlVariables.Requests.Read.FlowRegulation R Bit Automatic

and

SYSTEM.Localhost.SqlVariables.Requests.Read.FlowRegulation R Bit Automatic
SYSTEM.<StationName>.SqlVariables.Requests.Write.FlowRegulation R Bit Automatic

and

SYSTEM.Localhost.SqlVariables.Requests.Write.FlowRegulation R Bit Automatic

0 Flow regulation activated.
1 Flow regulation not activated.

Queries skipped by flow regulation

SYSTEM.<StationName>.SqlVariables.Requests.Read.SkippedByFlowRegulation R Register Automatic

and

SYSTEM.LocalHost.SqlVariables.Requests.Read.SkippedByFlowRegulation R Register Automatic
SYSTEM.<StationName>.SqlVariables.Requests.Write.SkippedByFlowRegulation R Register Automatic

and

SYSTEM.LocalHost.SqlVariables.Requests.Write.SkippedByFlowRegulation R Register Automatic

Reset to 0 when FlowRegulation is 0. Incremented for each query skipped

Queries skipped by overlapping

SYSTEM.<StationName>.SqlVariables.Requests.Read.SkippedByOverlapping R Register Automatic

and

SYSTEM.LocalHost.SqlVariables.Requests.Read.SkippedByOverlapping R Register Automatic
SYSTEM.<StationName>.SqlVariables.Requests.Write.SkippedByOverlapping R Register Automatic

and

SYSTEM.LocalHost.SqlVariables.Requests.Write.SkippedByOverlapping R Register Automatic

Never reset to 0. Incremented for each query skipped by overlapping.
