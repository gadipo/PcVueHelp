Creating a Chart

The Chart animation is flexible in both its appearance and operation and as a result, there are many configurable properties. The basic procedure to create a Chart is as follows.

  1. Open the mimic in which the Chart will be inserted and select Edit mode.
  2. Select the Chart tool from the Insert.Chart command on the menu.
  3. A Chart will be created and its properties dialog automatically opened and showing the X Data Source tab. Show picture

Once you have created a Chart, and closed its properties dialog, it can be moved, sized and copied the same as any other drawing element.

The Chart configuration dialog contains several tabs each for a particular aspect of the configuration.

  • Display - General display features. See the topic Configuring the general features.
  • Aspect - General appearance - the same as the Aspect tab of all drawing elements.
  • Execution - Properties that select the tools available to the User at run-time. See the topic Selecting the run-time tools.
  • X Data Source - See the topic Configuring the X data source.
  • X Axis - See the topic Configuring the axes.
  • Y Axes - See the topic Configuring the axes.
  • Series - See the topic Configuring the series.
  • Reference series - See the topic Configuring the reference series.

You can specify more properties to control the graphical element visibility. From the Aspect tab, behind the down arrow at the bottom of the Properties dialog box:

  • Specify zoom visibility bounds so that the graphic control is only visible for predefined zoom levels.

  • Specify layers so that the graphic control is only visible if one of the selected layers is displayed.

These are even more useful when combined with user permissions and decluttering templates.
