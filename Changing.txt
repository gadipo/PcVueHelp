Changing an embedded object
 

 
 
 
 
 
 
 
 
 
 
 
 
 
Changing an Embedded Object
 
There are two ways to change an embedded object.
 
 
You can edit it directly 
 in the window in which it is embedded.
 
You can open it in 
 the original application.
 
 
An embedded object is surrounded by a solid border when 
 a window is in Edit mode.
 
How to edit an embedded object
 
 
Open or select the 
 window in which the object is embedded and select Edit mode.
 
Double-click on the 
 embedded object. The object will be surrounded with a hatched border and 
 the toolbars from the objects original application will be displayed within 
 the Workspace.
 
Make your changes 
 to the object.
 
When finished, 
 click anywhere outside of the object or its toolbars. The toolbars will 
 be closed and the object returned to its static state. 
 
 
Show picture
 
 

 

 
 
 
 

 
When you are editing an object in this way, many of the 
 original application�s tools may not be operable.
 
 
How to open an embedded object
 
 
Display the Drawing 
 context menu by clicking on the object using the right mouse button.
 
The last entry in 
 the menu is context sensitive dependent on the type of object you have 
 selected. For example, for a spreadsheet, it will be Spreadsheet Object. 
 Attached to this last entry will be a further sub-menu with the choice 
 of Edit or Open. Select Open.
 
The object will now 
 be opened within the original application, where you can then make any 
 required changes.
 
To return to the 
 HMI when you have finished making any changes, select Quit or Exit from 
 the application�s File menu.

Overview of changing color
 

 
 
 
 
 
 
 
 
 
 
 
 
 
Overview of Changing color
 
Changing the color of a drawing element is one of the 
 simplest and most commonly used ways of presenting process information 
 to the user. You can use the status of bits and the value of register 
 variables as the source of a color change.
 
All color animations allow you to change the border (line), 
 background fill (pattern) and text of a drawing element to give a very 
 wide range of color combinations. You may use the blinking and transparent 
 options in your choice of colors. All the animations include the capability 
 to select a special color to be used whenever any of the variables are 
 invalid (for example when communication has failed).
 
The Pattern Property
 
 
 
 

 
 
Element
 
 
 
Effect
 
 
 

 
 
A drawing with a solid or colored button style
 
 
 
The whole of the pattern area changes.
 
 
 

 
 
A drawing with a hatched pattern
 
 
 
Only the color of the hatching changes.
 
 
 

 
 
Text 
 
 
 
The color of the background changes.
 
 
 

 
 
Lines and polylines
 
 
 
No effect.
 
 
 

 
 
Image
 
 
 
The pattern color is displayed through the transparent 
 area of the image.
 
 
 
 
The Border property
 
 
 
 

 
 
Element
 
 
 
Effect
 
 
 

 
 
A geometrical shape (rectangle, ellipse etc.)
 
 
 
The border changes color.
 
 
 

 
 
A line or polyline
 
 
 
The whole line changes color.
 
 
 

 
 
Text
 
 
 
The characters change color.
 
 
 

 
 
Image
 
 
 
The whole line changes color.
 
 
 
 
The image (picture) object supports all color animations 
 except Legend. 
 

 
If there is text in an image, you can change its color, 
 or make it disappear, by making its color transparent. For details, see 
 the topic Advanced Image Properties and the book Changing the Appearance of an Image.
 
 
Example
 
In this example, the Bargraph animation is applied to 
 an image in which the yellow color outside of the red dots is configured 
 to be transparent at run time: Show picture
 

 

 
 
 
 
The register variable's values can range from 0 to 100. 
 The current value is 25, so the blue color fills the lower part of the 
 image's transparent area: Show picture

Changing materials properties
 

 
 
 
 
 
 
 
 
 
 
 
Changing Materials Properties
 
The properties that effect materials configuration are 
 as follows.
 
 
 
 

 
 
Property
 
 
 
Description and action
 
 
 

 
 
Material 
 
 
 
Displays a list of materials contained within the model 
 or frame. The font used for the material name changes to indicate its 
 status.
 
 
Bold font, e.g. Material #0: The current configuration 
 is different from the original configuration of the model.
 
Italic font, e.g. Material #0: When you click OK to validate the dialog, the material's current configuration is be applied to all sub frames.
 
 
 
 

 
 
Original 
 
 
 
Sets the current material to its original configuration 
 (from the model file). The button is disabled if the object's configuration 
 is the original one.
 
 
 

 
 
Apply to Sub-Objects 
 
 
 
Indicates that the current material's configuration must 
 also be applied to all sub-frames that reference the same material.
 
 
 

 
 
Diffuse Color 
 
 
 
Sets the Diffuse color. A diffuse color can be a plain, 
 blinking or transparent color. If the diffuse color is transparent, meshes 
 using that material will be rendered invisible. 
 
 
 

 
 
Specular Color 
 
 
 
Sets the Specular color. A specular color can be a plain, 
 blinking or transparent color. If the specular color is transparent, 
 the material is processed as if the color is set to black.
 
 
 

 
 
Emissive color 
 
 
 
Sets the Emissive color. An emissive color can be a 
 plain, blinking or transparent color. If the emissive color is transparent 
 color, the material is processed as if the color is set to black.
 
 
 

 
 
Opacity
 
 
 
Sets the transparency percentage, from 0 (invisible) to 
 100 (fully opaque).
 
 
 

 
 
Specular Level
 
 
 
Sets the specular level, from 0 (dullest highlights) to 
 32767 (sharpest highlights).
 
The sharpest possible highlights may be limited to a specular 
 level value of less than 32767. When the level is over that limit, the 
 surface is rendered with no highlights.
 
 
 
 
How to change material properties at the model level
 
 
Open the 3D Properties 
 dialog by either double clicking on the model or its icon in the Graphic 
 Explorer.
 
Select the material 
 to be changed from the Material combo-box.
 
Enter new values 
 for the properties to be changed.
 
Tick the Apply to 
 Sub-objects box.
 
Click the OK button 
 to apply the changes and close the dialog. 
 
 

 
When 3D Properties dialog is opened at the model level, 
 it displays the materials configuration from the original model. It does 
 NOT display any changes that have subsequently been made using the 3D 
 Properties dialog.
 
 
How to change material properties at the frame level
 
 
Open the 3D Properties 
 dialog by double-clicking on a frame icon in the Graphic Explorer.
 
Select the material 
 to be changed from the Material combo-box. A frame may have zero, one 
 or more materials.
 
Enter new values 
 for the properties to be changed.
 
If you require the 
 change to be applied to any sub-frames also using that material, tick 
 the Apply to Sub-objects box.
 
Click the OK button 
 to apply the changes and close the dialog 
 
 

 
When 3D Properties dialog is opened at the frame level, 
 it displays the actual materials configuration including any changes that 
 have been made using the 3D Properties dialog.

Changing the appearance of a shape
 

 
 
 
 
 
 
 
 
 
 
 
 
 
Changing the Appearance of a Shape
 
The appearance of a shape may be changed using the Aspect 
 tab in the Drawing Properties dialog. Show picture
 

 

 
 
 
 
The following options are available.
 
 
 
 

 
 

 
 
 
 
 
Regular. The shape is displayed as a normal rectangle. 
 You can change the fill style and color.
 
 
 

 
 

 
 
 
 
 
Button. The shape takes the appearance of a button. The 
 fill color and style is fixed. The border size can be changed using the 
 Thickness property.
 
 
 

 
 

 
 
 
 
 
Colored button. The shape takes the appearance of a colored 
 button. You can change the fill style and color and the border size color.
 
 
 

 
 

 
 
 
 
 
Relief. The shape takes a sunken appearance. You can change 
 the fill style and color and the border size and color.
 
 
 

 
 

 
 
 
 
 
Inverse Relief. The shape takes a raised appearance. You 
 can change the fill style and color and the border size and color.
 
 
 

 
 

 
 
 
 
 
Shadow. The shape is displayed as a normal rectangle with 
 a colored shadow. You can change the fill style and color and the shadow 
 width and color.
 
 
 
 
If selecting the Button style you can also choose the 
 option System 
 look & feel. The button appearance will change to that of a 
 button in the operating system.

Changing the node order
 

 
 
 
 
 
 
 
 
 
 
 
 
 
Changing the Node Order
 
The node order for a polygon or polyline is initially 
 set by the order in which it was drawn. In design mode, if you position 
 the pointer (mouse) over a node a tool tip is displayed showing the node 
 number and the number of nodes. Show picture
 

 

 
 
 
 
You can change which node is node 1. All other nodes are renumbered 
 in sequence. For example, a polyline has 7 nodes and the original node 
 4 is changed to node 1. Node 5 becomes node 2, node 6 becomes node 3, 
 node 7 becomes node 4, node 1 becomes node 5 etc.
 
How to change the node order
 
 
Select the polygon 
 or polyline.
 
Right click on the 
 node that is to become node 1 and, from the context menu, select the Change 
 Node Order tool. The shape is redrawn. Show picture

 
 
What is the effect of changing the node order?
 
When applied to a polyline, the appearance changes as 
 the first node is the point at which the polyline is drawn. Show picture
 

 

 
 
 
 
When applied to a polygon, the appearance does not change. 
 However, if the polygon is used with a Path animation, the run mode operation 
 will change as the first node is the starting point for moving the shape 
 to which the animation was applied. Show picture

Configuring transformation properties
 

 
 
 
 
 
 
 
 
 
 
 
Changing Transformation Properties
 
The transformation properties determine a model or frames 
 position with respect to its parent.
 
 
 
 

 
 
Property
 
 
 
Description and action
 
 
 

 
 
Pivot
 
 
 
The position of the pivot determines the position of the 
 model or frame relative to its parent. 
 
 
 

 
 
Scale
 
 
 
The scale of the model or frame relative to its parent.
 
 
 

 
 
Angle
 
 
 
The orientation of the model or frame relative to its 
 parent
 
 
 
 
For a model, the parent is the entire mimic. For a 
 frame, the parent is either the model or another frame depending on the 
 location with the model hierarchy.
 
The default settings for a model (when a model is first 
 added to a mimic) are Pivot (0, 0, 0), Scale (100%, 100%, 100%) and Angle 
 (0, 0, 0).
 
The default settings for a frame will depend on the design 
 of the model that it is part of.
 
It is important to understand that all properties are 
 relative to the frame or model's parent. For example, if the scale of 
 a frame is changed from 100% to 200% the size at which it, and any sub-frames, 
 is rendered on the screen will double although the scale of the sub-frames 
 remain unchanged.
 
How to change transformation properties
 
 
Open the 3D Properties dialog by either double clicking on the model or its icon in the Graphic Explorer.
 
Enter new values for the properties to be changed.
 
Click the OK button to apply the changes and close the dialog
