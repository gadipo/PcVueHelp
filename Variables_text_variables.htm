Text Variables

A Text Variable is used to store an ASCII string. Text variables are used to display data that does not conform to normal numeric formats, such as date and time, to provide messages to the User, and to allow the User to enter textual information.

Configuring Range and Display Formats

Text strings are normally displayed unformatted. However, in the special case of displaying the time you can select the display format by specifying a format string.

  • Maximum size - The maximum number of characters that may be stored in the variable. Range 1 to 32000.
  • Format - Format specification for the Text variables TIME and DATE. Two formats may be specified, one for each of the two languages, separated by the pipe "|" character. The format string #D/#M/###Y|#M/#D/#Y would display the 29th of September 1995 as 29/09/1995 for the first language and 09/29/95 for the alternative language.

Character encoding for equipment source

When a text variable is linked to an equipment source the property Source encoding in the Advanced tab allow you to configure the character encoding methode used by the device so that strings are properly decoded on the PcVue side. Show picture

The following options are available most of which follow widely documented IT standards other than Siemens.

  • Windows (ANSI)

  • MS-DOS (PC-8)

  • Unicode (UTF-8)

  • Siemens string - Specific to Siemens equipment. The first 2 bytes contain the maximum and actual size of the string. The remaining bytes are the string itself.
