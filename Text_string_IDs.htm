Text String IDs

Each text string ID uniquely identifies a text string within a file. The text strings that the Import/Export Tool produces come from three sources.

  • Mimic properties - for example the mimic title.
  • Drawing element properties - for example the text displayed by a text drawing element.
  • Animation properties - for example the format string of a label animation.

The Translation Import/Export Tool uses the same identifier that is used by the scripting language to reference the drawing elements. When a drawing element is included in a group, the identifier is composed of the group and element identifiers separated by a dot, for example GROUP2.TEXT1.
