Changing the Node Order

The node order for a polygon or polyline is initially set by the order in which it was drawn. In design mode, if you position the pointer (mouse) over a node a tool tip is displayed showing the node number and the number of nodes. Show picture

You can change which node is node 1. All other nodes are renumbered in sequence. For example, a polyline has 7 nodes and the original node 4 is changed to node 1. Node 5 becomes node 2, node 6 becomes node 3, node 7 becomes node 4, node 1 becomes node 5 etc.

How to change the node order

  1. Select the polygon or polyline.
  2. Right click on the node that is to become node 1 and, from the context menu, select the Change Node Order tool. The shape is redrawn. Show picture

What is the effect of changing the node order?

When applied to a polyline, the appearance changes as the first node is the point at which the polyline is drawn. Show picture

When applied to a polygon, the appearance does not change. However, if the polygon is used with a Path animation, the run mode operation will change as the first node is the starting point for moving the shape to which the animation was applied. Show picture
