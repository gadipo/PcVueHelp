Overview of the Gizmo Tool

The Gizmo tool shows you the orientation of the selected model in design mode and allows you to manipulate it using translation, rotation and scaling. Show picture

The Gizmo's size never changes, so as to make it easy to manipulate as you zoom in or out or change the scale of the model.

The Gizmo's axes are color coded as follows:

Axis

Meaning

Red

Model's X axis

Green

Model's Y axis

Blue

Model's Z axis

The icon that appears in a selected frame shows the frame's pivot and orientation of the X, Y and Z axes. It cannot be used for translation, rotation or scaling.
