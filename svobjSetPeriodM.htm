SetPeriod Method

 Applies to 

Sets the period and, optionally, the start time for the Trend Viewer.

Syntax

Sub SetPeriod(period As Double, [StartTime As Variant])

Part

Description

period

The period (in milliseconds).

StartTime

The start time (optional).

Example

Dim tStart As New TrdDate
tStart.Hour = 9
tStart.Minute = 30
tStart.Second = 0
Trend1.SetPeriod 300000, tStart
