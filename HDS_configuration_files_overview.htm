The HDS Configuration Files

The configuration of the HDS is saved in two files.

  • The HDS.INI file is located in the Bin folder of PcVue and affects the operation of the HDS for all projects.
  • The HDSCONF.DAT file is located in the project's C folder and only affects the operation of the HDS for that project.

Most of the properties in the configuration files are generated as a result of configuring the HDS as seen from the Archives node in the Application Explorer. However, there are some properties, used for fine tuning of the HDS, that can only be changed by directly editing the configuration files using a text editor such as notepad.

Before changing any of the properties in the HDS file, both PcVue and the HDS must be shutdown. The best way to ensure this is to restart the operating system and edit the files before PcVue is started.

You should only change HDSCONF.DAT or HDS.INI under the supervision of technical support.
