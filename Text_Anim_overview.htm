Overview of the Text Animations

The text animations provide real-time information to the user in the form of a character string. All the text animations use a text drawing element as the support. The text drawing element provides the font, color and the style (inverse, button etc.) properties. The following text animations are provided.

  • Label animations - displays text from variable's configuration (associated label or description) or context sensitive information such as the branch of a symbol.
  • Text animations - Displays the real time value of a text variable.
  • Register animations - Displays the real time value of a register variable
  • Pre-selected text - Displays text that is configured in the animation itself.

What is the difference between the Label and Pre-selected Text animations?

The label animations display texts that are configured elsewhere - not in the animation itself. If the source of the texts is edited (for example an associated label), the texts that the animation displays change without having to re-configure the animation.

The pre-selected text animations display texts that are configured as part of the animation. The only way to change the texts that are displayed is to re-configure the animation.
