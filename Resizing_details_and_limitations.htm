Resizing: details and limitations

To resize a 3D model properly, without skewing it, the HMI must resize the model along one, two or all of its axes according to the resizing tool and the direction of the 3D model's axes (depending on the model's rotation settings).

The following tables provide information on what to expect when resizing a 3D model. The examples show the model to be resized - but not the reference model - in a window using the Top View orthographic camera mode.

The model is resized along two of its axes: the axes not parallel to one of the world axes (the model's Y and Z axes in the example).

Limitation - Since the same scaling ratio (between the reference model and the resized model's height) is used for both axes, the model's width may also change. This occurs when none of the model axes is parallel to the world horizontal axis (which is not the case in this example).
