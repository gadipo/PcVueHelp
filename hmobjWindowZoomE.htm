WindowZoom Event  

 Applies to 

Syntax

object.WindowZoom(ByVal ZoomRequest As fvZoomRequest, ByVal ZoomRatio As Long, ByVal Window As Integer)

Part

Description

ZoomRequest

Indicates the action the User has requested as follows:

  • fvZoomIn: Zoom in the Window content (for Zoom In and Magnifying Glass tools).
  • fvZoomOut: Zoom out the Window content.
  • fvZoomOff: Set back the zoom ratio to 100% (for Zoom In and Magnifying Glass tools).

ZoomRatio

Provides the new (current) zoom ratio stated as a percentage, after the zoom request has been executed.

Window

Provides the window index (the index in the mimic's Windows Collection) for which the event has been fired.

Each time you zoom the contents of a window, the mimic's WindowZoom event is fired in VBA. That event is associated with the Mimic object to which the Window object belongs.

Zooming is subject to these Window properties: EnableZoom, MinimumZoom and MaximumZoom.

A zoom ratio of 100% does not imply that the User's request was Zoom Off.

In the HMI the command Magnifying Glass is similar in effect to Zoom In, and Cancel Magnifying Glass is similar to Zoom Off, including firing the WindowZoom event.
