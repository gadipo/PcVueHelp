Configuring the Log Viewer Context Menu

The Menu tab is used to select which tools are available at run-time from the context menu. Show picture


The tools that are displayed at run-time also depend on if any logs are selected at the time the context menu is opened. See the topic Using the context menu.

The tools are grouped as follows:

  • Log display filters - See the topic Configuring the Log Viewer Filter.
    • Alarm changes
    • User actions
    • Bit changes
    • Alarm masks
  • Others - Other general commands, change the time filter, print selected log events, change the blinking mode and activate/de-activate the auto-refresh.

The behavior at run-time, what exactly the user will have access to on right-click also depends on other features such as context mimics, as well as how mouse inputs handling options are defined in the HMI General Options.

See Using the Using the context menu for more information.
