Variables Tree Instructions

Instruction

Action

ALARM

Test or force the state of an alarm.

GETTREE

Return the current branch.

GROUPALARM

Start or stop a selected group or all groups of alarm synthesis.

POPULATION

Create a variable population.

REFRESH_DB

Asynchronous assignment of value to variables from an ASCII file.

SENDLIST

Send a list of commands and set-points to a field device as would a recipe.

SET

Construction of a list of commands and set-points.

STATION_FILTER

Manage the application of population filters to functions and data.

SVBRANCH

Change the status of a group of variables according to their branch.

TEMPORARY_DB

Creation and deletion of temporary variables.

TREE

Set a branch.

VARIABLE

Test or force the status of a variable.
