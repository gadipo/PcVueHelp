ShowNumericKeypad Property

              

Description

Displays the numeric keypad and accepts an input value.

Syntax

Function ShowNumericKeypad(Value As Double, [Mode As fvKeypadOpeningMode = fvAutoMode], [AllowModeSwitch As Boolean = False], [Caption As String], [Password As Boolean = False], [DisplayInitialValue As Boolean = True]) As Boolean

 

Part

Description

Value

Input value.

Mode

Format of the input value (see enumeration below).

AllowModeSwitch

Whether to allow the user to choose the input format.

Caption

Title for the input dialog box.

Password

Whether to display the input value in clear or as '*' characters.

DisplayInitialValue

Whether to display the previous value of the variable.

Enumeration of fvKeypadOpeningMode

Value

Description

 fvAutoMode

Automatic mode.

 fvBinaryMode

Binary mode.

 fvDecimalMode

Decimal mode.

 fvHexadecimalMode

Hexadecimal mode.

For further information, see this topic in the main Help: Configuration.Preferences and Options.Configuring the Soft Keypads.
