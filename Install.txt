The installation menus
 

 
 
 
 
 
 
 
 
 
 
 
 
 
The installation menus
 
The PcVue installation process proposes the following menus.
 
Select the installation language
 
Displayed automatically when the media is inserted or by running Setup.exe. Show picture

 
In addition to being used during the installation, the language you select is also used once PcVue is installed:
 
 

 
As the startup language of the software, that is, the language for any message or dialog box displayed before the project is loaded.
 
 

 
Upon project creation, as the first project language and first display language. The second project and second display language default to English. Once PcVue is running, you will be able to change the language settings as part of the project settings.
 
 
 

 
Once PcVue is running, you will be able to change the language settings as part of the project settings.
 
 
The toolbar
 

 
 
 
From left to right:
 
 
Click to come back to the Main menu.
 
Click to come back to the Previous menu.
 
Click to display the Add-ons, Drivers and Tools menu that proposes complementary products from our partners. See below.
 
Click to browse media contents with the 
 Windows Explorer.
 
Click to access the PcVue web site using the default web browser.
 
Click to access the PcVue web based tutorials using the default web browser.
 
Click to open the Readme file.
 
Click to open the Installation help.
 
Click to exit the setup.
 
 
Main menu
 
 
HMI / SCADA - Will install components required to deploy PcVue on a station of type HMI Desktop, SCADA Station, Client Station, Communication Server, HMI Web or Historical Server. 
 
Geolocation Monitoring Station - Will install components required to deploy a Geolocation Monitoring station.
 
Web & Mobile Back End - Will install components required to deploy a Web & Mobile Back End Server.
 
Studio Engineering - Will install components required to deploy a Studio Engineering station.
 
Web Deployment Tools - Will install PcVue web deployment tools and packages. If you deploy a web architecture, these tools must be installed on the computer that hosts your IIS web server.
 
 
The HMI / SCADA menu
 
 
Web & Mobile access - Install Web back end role. Add components required to support the interface between PcVue and the IIS Web server when you need to deploy one of following Web or Mobile app: WebVue, the WebScheduler, TouchVue or the Web Services Toolkit. If you deploy a Web architecture, at least one of the PcVue machines must have this role, in order to interface with the IIS Web server.
 
BIM control - Add the components that are required to use the BIM control.
 
Video controls - Add the components required to use the ONVIF and the Video controls.
 
Web browser control - Add the components required to use the Web browser control (i.e. the Edge WebView2 runtime if not already installed on the computer).
 
Libraries - Add reusable libraries of images, symbols, mimics & mimic templates.
 
Quick Start project - Add the Quick Start Project, a pre-configured project that you can use to create your own project in a few clicks.
 
Demo project - Add the Demo project, a sample project allowing you to explore the main features of PcVue.
 
Extra documentation - Add supplementary documentation, in addition to the online help:
Addenda: Documentation items not available in the online help, in particular descriptions of the main configuration file formats.
 
SDK & API - Add SDK and API documentation, code components and samples:
XML Generic Import Toolkit,
SV Manager Toolkit,
CimWay driver Toolkit,
Web Services Toolkit. 
 
 
Geolocation Monitoring Station, Web & Mobile Back End and Studio Engineering menus
 
As for the HMI / SCADA menu with the following additions.
 
 
Web & Mobile access - Install Web back end role. Add components required to support the interface between PcVue and the IIS Web server when you need to deploy one of following Web or Mobile app: WebVue, the WebScheduler, TouchVue or the Web Services Toolkit. If you deploy a Web architecture, at least one of the PcVue machines must have this role, in order to interface with the IIS Web server.
 
Geolocation back end - Add components required to support geolocation-based services and apps.
 
Geolocation back end with tracking - Add components required to support geolocation-based and tracking-based services and apps.
 
Instant Messaging - Add components required to run the Instant Messaging server.
 
 
Web Deployment Tools
 
Install the Web Deployment Console. See the chapter Installing the Web Deployment Tools for more information.
 
The Add-ons, drivers and tools menu
 
The exact list of Add-ons, drivers and tools may vary, but usually includes the following:
 
 
License Utility(*) - Install the standalone License utility package that can be used independently of PcVue.
 
Sentinel Driver(*) - 
 Install the hardware protection key drivers.
 
Dream Report - Reporting software.
 
OPC-UA Gateway - OPC-UA client & server gateway.
 
 
(*) Also installed as part of the main installation process.
 

 
Some of these components are a chargeable option. They can be installed but, unless they are enabled by the appropriate license, they will not run or they will run in demonstration mode with limited functionality.

Installing PcVue&#160;
 

 
 
 
 
 
 
 
 
 
 
 
 
 
Installing PcVue&#160;
 
Prerequisites installation
 
The prerequisites are components that must be installed before PcVue, or the Web Deployment Tools, can be installed and operate correctly. The necessary prerequisites are bundled in with PcVue and no additional licenses are required. 
 
When the installation is started, it first checks for the presence of the prerequisites on the computer and installs the missing ones.
 
The installation wizard
 
The PcVue installation automatically follows that of the prerequisites.
 
A wizard guides you through the installation
 process, prompting for information where necessary. The exact process will
 depend on the options you choose, the operating system and what software components are already installed
 on your computer but the main stages are as follows.
 
At each step, after configuring any options click the OK or Next button to proceed. Clicking the Cancel button at any time will abort the installation.
 
 
Welcome to the InstallShield Wizard - This is the first step of the installation itself. Click Next to continue. Show picture

 
License agreement - Before you can continue with the installation you should read the License Agreement and select I accept... or I do not accept... as appropriate. Click Next to continue only if you accept the agreement. Show picture

 
Products and Projects Folders - Suggests a destination folder for PcVue executable files and for the projects and libraries.
To install the software elsewhere, use the Change button to select another path; likewise for the projects and libraries. Show picture

 
Setup Type - Select the setup type from the following options. Show picture

 
 
Typical - Installs the most common features and preselected options for the software, taking into account the selection made in the Features Selection Menu. For example, if you selected Web back end, the additional components required for the Web back end role as well as the components required for the basic PcVue installation, will be selected for installation.
 
Custom - Allows you to select exactly which components you want to install. The selection is the same as for Typical. Recommended for advanced users. Show picture
 

 
 
 
 
Ready to Install the Program - Asks whether you wish to complete the installation or to review or change the options you selected.
Click Install to proceed with the installation, Back to return to the options or Cancel to abort the installation process. Show picture

 
Installing status: A progress bar indicates progress with installing each component, then a dialog confirms that the installation is complete. Click on Finish.
 
 
Completing the installation 
 
Once installation is completed, you may be prompted to
 re-boot your computer. This re-start is sometimes necessary to register some of the files
 needed to run PcVue with the operating system.
 
Version update
 
If you start the installation on a computer where a previous version is already installed, the installation process will automatically update the currently installed version.
 
A version can be updated by installing from a Full DVD, a Standard package or a Cumulative patch.
 
In most cases, the installation will update the product without changing the list of installed features. If, as part of the update, you need to change this list, you are required to use the Maintenance procedure called Modify after the update.
 See Maintaining PcVue or the web deployment tools for more information.
 
 
In some cases, depending on the exact content of the update, the installation process need to uninstall the existing version before re-installing the new one. You then have the opportunity to change the list of installed features at the same time as the update.

Installing the Web Deployment Tools
 

 
 
 
 
 
 
 
 
 
 
 
 
 
Installing the Web Deployment Tools
 
&#160;PcVue web deployment tools are required on the computer that has the role of web server, where the IIS Web server is installed. Depending on the chosen architecture, this can be on the same or different computer to PcVue itself. The main component of the web deployment tools is the Web Deployment Console.
 
When either the PcVue installation, or the Web Deployment tools installation is started, it first checks for the presence of the prerequisites. See the topic Installing PcVue.
 
A wizard guides you through the installation process, prompting for information where necessary. At each step, after configuring any options click the OK or Next button to proceed. Clicking the Cancel button at any time will abort the installation.
 
 
Welcome to the InstallShield Wizard - This is the first step of the installation itself. Click Next to continue. Show picture

 
License agreement - Before you can continue with the installation you should read the License Agreement and select I accept... or I do not accept... as appropriate. Click Next to continue only if you accept the agreement. Show picture

 
Destination Folder - Suggests a destination folder for the installation. To install the software elsewhere, use the Change button and select another path. Show picture

 
Ready to Install the Program - Click Install to proceed with the installation. Show picture

 
Installing status: A progress bar indicates progress with installing each component, then a dialog confirms that the installation is complete. Click on Finish.
