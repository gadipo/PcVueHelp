Applying an Animation to a 3D Model in General

You can apply an animation to an entire model or to an individual frame.

  • If you apply an animation to a model it will affect the entire model.
  • If you apply an animation to a frame, it will affect the frame and any sub-frames.

How to apply an animation to a 3D model or frame

  1. Open the mimic containing the 3D model and select Design mode.
  2. Open the Graphic Explorer via the menu Display.Graphic Explorer.
  3. Select the model or frame in the Graphic Explorer.
  4. Right-click on it to display a pop-up menu and select Animate to display the list of available animations.
  5. Select the animation. The 3D Drawing Properties dialog is opened with a new tab corresponding to the animation chosen.
  6. Select and enter the properties for the animation.
  7. Click the OK button to confirm the animation.

It is recommended that you use the Graphic Explorer to apply animation to a 3D model or its individual frames as it allows you to see any animation that has already been applied.

Animations that have been applied at the frame level will not appear in a 3D model's Properties dialog when that is opened by double-clicking on the model.
