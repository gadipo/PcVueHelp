Drawing 3D Text

You use the command Insert 3D Text to place a 3D text string in a mimic.

The 3DText tab

  1. Select the command Insert 3D Text from the Insert menu or the 3D Draw menu. Show picture

  2. In the 3D Text tab of the Drawing Properties dialog, configure the basic properties of the 3D text. For details of the Materials properties, see the topic Changing Materials Properties. Show picture

  3. For details of the Pivot, Scale and Angle properties, see the topics Understanding Euler XYZ and XYZ Axis Angles and Changing Transformation Properties.

The Advanced tab

In the Advanced tab of the Drawing Properties dialog, you can configure the orientation and appearance of the 3D text. Show picture

  • The Alignment property positions the pivot at the left, center or right of text. For instance, the option Left places the leftmost edge of the text so that, when viewed from the front, the text appears to the right of the Gizmo. Show picture
  • The Billboarding property provides an option View Plane that makes the text always face toward the camera. Otherwise, as with any 3D model, when the camera turns around the text the viewer sees all the sides of the text including the edges and the back. Show picture
  • The Extrusion property configures an appearance of depth. A value of 0 restricts the text to a plane surface (2 dimensional) whilst other values give it depth. Show picture

Where depth is required, an extrusion value of 0.05 to 0.15 usually yields the most legible effect at various angles.

If there is no billboarding and no extrusion, the text becomes invisible when viewed edge-on.

Animation

A 3D text object supports the animations supported by other 3D models, plus those of the text animations. Show picture

In Run mode you can select and execute the command animations of a 3D text by clicking anywhere on the volume defined by the text bounding box. Show picture
