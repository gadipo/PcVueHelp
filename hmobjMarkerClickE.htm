MarkerClick Event

              

Occurs when a Map Control is clicked.

Syntax

object.MarkerClick(Latitude As Double, Longitude As Double, Handled As Boolean     )

Part

Description

Longitude

A double representing the map longitude at the position of the click.

Latitude

A double representing the map latitude at the position of the click.

Handled

A boolean used to redefine the marker behavior.

Events are fired only if the EnableEvents property of the underlying object has been set to True. This happens automatically once the first event script has been associated with the object.
