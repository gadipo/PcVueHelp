Overview of the 3D Properties Dialog

The 3D Properties dialog provides access to either the properties of an entire model, or the properties of a specific frame.

  • To display the properties of a 3D model, either double-click on the model in Design mode or double- click the icon representing the model in the Graphic Explorer.
  • To display the properties of a frame, double-click on the icon representing the frame in the Graphic Explorer.

The 3D Properties dialog displays a single tab (3D Base) for the model or frame properties plus a tab for each animation that has been applied. The properties available from the 3D Base tab can be divided into two broad categories.

  • Materials properties - A list of the materials used and their properties.
  • Transformation properties - Position (Pivot), scaling (Scale) and rotation (Angle).

Displaying the properties of a 3D model

When displaying the properties for an entire model the 3D Base tab displays:

  • A list of materials for the entire model.
  • Transformation properties relative to world space (the 3D mimic within which the model is located). Show picture

Displaying the properties of a frame

When displaying the properties for a particular frame, the 3D Base tab displays:

  • A list of materials only for that frame.
  • Transformation properties relative to its parent (either another frame or the model depending on its position in the hierarchy). Show picture

Because a frame does not always have a mesh (it may just be used as a parent of other frames) the materials list may be empty.
