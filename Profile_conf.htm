How to Configure a Profile

  1. Configuring a profile.
    1. Select the Profiles folder in the Application Explorer configuration tree and, from the task list, select Add a profile. A dialog to configure a new profile will open. Show picture


    2. Enter the name for the profile. Profile names are limited to 40 alphanumeric characters and are not case sensitive.
    3. Enter a description of the profile. The description appears in the contents pane when the Profiles folder is selected. The description is optional.
    4. Enter a role for the profile. The role is optional.
  2. Selecting the profile access rights.
    1. Use the other tabs of the Profile dialog to select the profile's access rights. See the Profile Properties book for a full description of all of the rights.
  3. Click OK to create the profile and close the dialog.

Profile default settings

When you create a new profile, some access rights are enabled by default. This is done purposely to prevent a User from accidentally locking themselves and/or other users completely out of PcVue. See warning below.

  • Administration
    • Profile rights
      • Exit
      • Desktop
      • Development
      • Diagnostic tasks
    • Profiles and user management
      • Change profile rights
        • Administration
        • Command, acknowledgment and masking
        • Window and layers
        • Recipe
        • WebVue
        • Application Architect
        • Data analysis

You must have at least one Profile/User that has either the following rights.

  • Change profile rights - Administration
    OR
  • Create profiles and Create profile associations

If you do not do this, you will have the situation where you cannot change the profile of a user, nor can you create a new profile and associate it with a user. You will be permanently prevented from changing anything associated with User Accounts.
