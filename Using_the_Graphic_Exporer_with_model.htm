Using the Graphic Explorer with a 3D Model

The Graphic Explorer is essential to getting the most from 3D models as it allows you to display, change and animate the properties of the model's individual frames. Without the Graphic Explorer, you are only able to display, change and animate properties of the model itself.

How to select a 3D model and display its properties

  1. Select the menu command Display.Graphic Explorer to open the Graphic Explorer.
  2. Select the 3D mimic containing the model by clicking on its icon in the display tree.
  3. Expand the display tree to show the available drawing elements by clicking the + sign beside the icon representing the mimic.
  4. Select the 3D model's name in the Graphic Explorer tree. Show picture
  5. If the mimic is in Design mode, the selected model is surrounded by a green box and its pivot point is displayed. Show picture
  6. To display the properties of a model, double-click on the corresponding icon. The Model Properties dialog is opened. Show picture

How to select a frame of a 3D model and display its properties

  1. Select the model as above.
  2. Expand the display tree by clicking the + sign beside the icon representing the model, to show the top-level frames.
  3. To select a frame, click the corresponding frame icon. If the mimic is in Design mode, the selected frame is surrounded by a yellow box and its pivot point is displayed. If the frame has child frames, both frame and child frames are surrounded with a yellow box. Show picture
  4. To display the properties of a frame, double-click the corresponding item. The Frame Properties dialog is opened.
  5. To display any child frames, click the plus sign (+) beside the frame icon and repeat the previous step.
