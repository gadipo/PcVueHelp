Overview of the Security Animations

The Security animations enable you to provide buttons on a mimic from which a user can log in, log out, display or manage details of the current user or any user and apply extra security measures.

When you use the Animate.Security menu, the option you choose applies one of two animations: Show picture

Option Applies to this animation Use with soft keyboard
Login User Yes
Logout User N/A Display only
User information User N/A Display only
Password modification User Yes
User manager User No
User confirmation Security Yes
Double signature Security Yes

User animation

The User animation enables you to provide buttons on a mimic from which a user can log in, log out or display details of the current user or of any user. Show picture

The Security animation

The Security animation is applied to a drawing element before applying a Control Zone animation to provide additional security. Show picture

Using the Soft keypad with the security animations

On touch screens, you can use PcVue's soft keypad with those options that require user input. See the table above. To enable the soft keypad use the option Soft keypad management.Touch screen.Use soft keypad for Login and security animations in the HMI Options. See the topic Soft Keypad Management.
