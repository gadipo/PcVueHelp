CONVERT

See also Example

Convert an alphanumeric string to Hexadecimal, Octal, Binary or BCD, and vice-versa.

WebVue support - Yes.

Mode

Mnemonic

Syntax
1 BINTOA 1
2 OCTTOA 1
3 HEXTOA 1
4 BCDTOA 1
11 ATOBIN 2
12 ATOOCT 2
13 ATOHEX 2
14 ATOBCD 2
15 DEGTORAD 3
16 RADTODEG 3

Syntax 1

StrVal = CONVERT(Mode, Val);

Argument

Meaning

Val

Numerical value to be converted. Any numeric type.

Return type: STR.

Execution

Mode

Mnemonic

Action

1

BINTOA

Convert a numerical value (base 10) into a character string representing the binary value.

2

OCTTOA

Convert a numerical value (base 10) into a character string representing the octal value.

3

HEXTOA

Convert a numerical value (base 10) into a character string representing the hexadecimal value.

4

BCDTOA

Convert a numerical value (base 10) into a character string representing the BCD value.

Syntax 2

LongVal = CONVERT(Mode, Val);

Argument

Meaning

Val

The string to be converted. Type STR.

Return type: LONG

Execution

Mode

Mnemonic

Action

11

ATOBIN

Return a numerical value (base 10) of a character string represented in its binary form

12

ATOOCT

Return a numerical value (base 10) of a character string represented in its octal form

13

ATOHEX

Return a numerical value (base 10) of a character string represented in its hexadecimal form

14

ATOBCD

Return a numerical value (base 10) of a character string represented in its BCD form

Syntax 3

DblVal = CONVERT(Mode, Val);

Argument

Meaning

Val

An angle value to be converted. Any numeric type.

Return type: DOUBLE.

Execution

Mode

Mnemonic

Action

15

DEGTORAD

Convert an angle value in degrees to the equivalent in radians.

16

RADTODEG

Convert an angle value in radians to the equivalent in degrees.

Example

SUB main()
DIM ch1 as str;
ch1 = "01111111111111111111111111111111";
print( convert("ATOBIN",ch1)); 'Return: 2147483647
ch1 = "A9F7";
print( convert("ATOHEX",ch1)); 'Return: 43511
ch1 = "24";
print(convert("BINTOA",convert("ATOBCD",ch1) ));
'Return: 100100
END SUB
