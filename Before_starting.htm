Before starting the installation

Before you begin

Never run two installation processes at the same time on a PC (whatever item of software you are installing). In particular:

  • Do not run PcVue installation while Windows updates are being installed.
  • Never launch or accept a Windows update installation while PcVue installation is running.

Before any installation, removal or maintenance operation on PcVue, you should back up any project and library that you wish to retain. More generally, you should take backups regularly. See the Deployment book for more information about how to backup and restore your projects and libraries.

If you are using any of PcVue web based feature, that is WebVue, TouchVue, SnapVue, the WebScheduler or the Web Services Toolkit, you need to install the following items:

  • Web deployment tools, on the computer that will take the role of the web server.
  •  PcVue station with the web & mobile back end role.

See Role of the Web Server and Web & Mobile Back End and Web Deployment Console Overview for more information.

The installation process depends on what is already on the PC and on the software you are going to install.

  • Fresh installation, when the version is not yet installed.
  • Maintenance mode, if you wish to repair your installation, add/remove features, or uninstall PcVue.
  • Updating the installation, when a previous release of the same version is already installed.

The installation media comes in the following form:

  • Full DVD - Either as a DVD, or as a downloadable package, containing the full set of installation modules including all Add-ons and tools. Can be used to perform a fresh installation or update an existing installation.

  • Standard package - A smaller downloadable package, without the Add-ons. Can be used to perform a fresh installation or update an existing installation.

  • Cumulative patch - Downloadable package designed to update an existing installation. Cannot be used for a fresh installation.

Starting an initial installation

The set-up program guides you through the installation process by prompting you for information and automatically determining your system configuration.

  1. Check that enough disk space and memory are available on the target PC (as specified in Operating System and PC Requirements).
  2. Ensure no other installation or automatic update is running.
  3. Check that the system and software meet the requirements.
  4. Insert the PcVue media into the DVD drive. If you have Auto Run enabled, the set-up program will start automatically, otherwise you can start the installation manually by locating and running Setup.exe from the root of the DVD.
  5. Follow the instructions on the screen as described in the topics after this.

Throughout the installation process, always restart the PC when requested to do so.
