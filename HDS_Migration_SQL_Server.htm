How to use the SQL Server Management Studio

The SSMS will be the place for some of the actions you will have to lead in order to safely migrate your HDS databases. Here are the instructions to connect to a particular SQL Server instance.

  1. Launch the SSMS tool via the Start → All programs → Microsoft SQL Server → SQL Server Management Studio program folder

    2. Or press Windows key and then type ssms.

  2. Click on File → Connect Object Explorer… Show picture

  3. Select the SQL Server name hosting your HDS database, enter your Login & Password and click on the Connect button. Show picture

You are now connected to a particular SQL Server instance and can access all databases it hosts.
