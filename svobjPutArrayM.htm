PutArray Method

 See also Applies to 

Writes the content of a string in a block of cells.

Syntax

Sub PutArray(TopRowAs Long, LeftColAs Long, BottomRowAs Long, RightColAs Long, ColSeparatorAs String, RowSeparatorAs String, BufferAs String) As String

 

Part

Description

TopRow

Identifies the top row.

 LeftCol

Identifies the left-hand column.

BottomRow

Identifies the bottom row.

RightCol

Identifies the right-hand column.

ColSeparator

The column separator.

RowSeparator

The row separator.

Buffer

Identifies the buffer.
