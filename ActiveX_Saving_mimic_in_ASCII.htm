Saving a Mimic Containing an ActiveX Control in ASCII Format

Mimics may be saved in binary (the default) or ASCII format.

If you save a mimic containing an ActiveX control in ASCII format an additional file is created in the mimic folder named MimicName.BINARY, where MimicName is the name you have given the mimic. The .Binary file contains the value of any ActiveX properties and is not readable with a text editor.

If you want to save the mimic as only one file, you must select the option Save Properties as Text (ASCII mimic only) in the ActiveX control's Properties dialog.

Show picture

The ActiveX properties will then be saved in ASCII format in the main mimic file.

You should not select the option Save Properties as Text (ASCII mimic only) if the mimic is saved in binary.
