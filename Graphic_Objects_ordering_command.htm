Graphic Objects Ordering Command

The objects in a mimic have a particular order. At first that is the order in which they were added to the mimic. When you use the ordering tools, they modify that initial order.

You can reorganize and maintain the list of objects in the Graphic Explorer to the order in which they appear in the mimic. To apply the ordering command:

  • Click on the button Display according to Mimic Objects ordering. Show picture

When ordering is turned off

When the Ordering command is turned off, the Graphic Explorer works as follows.

  • It is not notified of any change of the order of objects in the mimic.
  • When you create a group or symbol, the new graphic object (group or symbol) is always added (if the corresponding filtering command is turned on) at the end of the Graphic Explorer's objects list.
  • When you un-group a group or symbol, the drawing elements that it contains are always added (if the corresponding filtering command is turned on) at the end of the Graphic Explorer's objects list.
  • When you turn on a filtering command, the objects that match the filter are always displayed at the end of the Graphic Explorer objects list.

When ordering is turned on

When the ordering command is turned on:

  • The Graphic Explorer reorganizes its objects list according to the object ordering in each mimic.
  • This order is automatically updated when the order of objects in the mimic changes or when a filtering command is turned on.
