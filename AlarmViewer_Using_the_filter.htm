Using the Alarm Viewer's Filter

The alarm filter is displayed from the small command button below the scroll bar. From the filter dialog box, you can select the Domain and Nature, the alarm states and levels. Show picture



Changing the alarm filter at run-time is only a temporary operation, when the window containing the Alarm Viewer is closed, the changes to the filter are lost.

Any SQL filter entered in the Alarm Viewer's configuration is displayed. It can be temporarily cleared by clicking the Clear button, but not changed at run-time.
