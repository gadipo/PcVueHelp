CenterToMarker Method

              

Locates a map on the given marker. If the marker is multipoint (track or polygon), then the map relocates on the first point of the marker. Generates an error if collection is loading..

Syntax

object.CenterToMarker(MarkerID As String ,LeyerID As String  [,Window As Short] )

Part

Description

MarkerID

A string containing the name of the mimic.

LayerID

Optional argument. Default: "" String that specifies the name of the branch that must be used to CenterToMarker the mimic.

Window

Optional argument representing the index of the window if several windows are instantiated for the mimic.
