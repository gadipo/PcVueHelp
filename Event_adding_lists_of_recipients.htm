Adding Lists of Recipients

The Recipients dialog opens when you click the ellipsis button adjacent to any of the Recipient fields of either the E-mail tab or the SMS tab of the dialog used to create an event to trigger a message. It allows you to interactively build a list of recipients containing User names, User profiles and Substitutions.

To send a message to the same recipient via e-mail and via SMS, you include it once in the E-mail tab and once in the SMS tab.

Adding recipients

The list of recipients is automatically updated each time you tick an entry in the Users tab, Profiles tab Show picture or Variable substitution tab Show picture.

When you have selected all recipients click OK to return to the Create Event to Trigger a Message dialog.
