GetCursorValue Method

 Applies to 

Get the Value and Quality for a curve, at the cursor point.

Syntax

Function GetCursorValue (nID As Integer, value As Double, valid As Boolean) 

Part

Description

nID

Curve number.

value

Value at the cursor.

valid

True if value at the cursor is valid, else False.

Returns False if the cursor is not currently displayed.

Example

Dim curveID As Integer
Dim value As Double
Dim valid As Boolean

bok = Trend1.GetCursorValue(curveID, value, valid)
