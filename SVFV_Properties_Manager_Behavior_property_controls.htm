Behavior of Drawing Property Controls

Most of the property controls used by the Drawing Properties Manager are standard Windows controls. However, the behavior of some of them has been altered. The following table describes the behavior of each kind of property control:

Property control

Shown as

How to set the property value

Color Selector

  • Click to display the Color Palette then click on a color in it.
  • Double-click to open the Color Picker. In it, select a color then click on OK.

Drop-down or combo box

Type in a value then press the Enter key, or select a value from the drop-down list.

Edit

Type in a value then press the Enter key.

Edit with spin control

Type in a value then press the Enter key, or select a value with the spinner.

Option buttons

Click on an option button.

Tick-box

Tick or untick the box.

When you type a value into an edit box, combo box or spin control box, then close the dialog or click outside it, the property's value reverts to the previous setting.
