Changing Materials Properties

The properties that effect materials configuration are as follows.

Property

Description and action

Material

Displays a list of materials contained within the model or frame. The font used for the material name changes to indicate its status.

  • Bold font, e.g. Material #0: The current configuration is different from the original configuration of the model.
  • Italic font, e.g. Material #0: When you click OK to validate the dialog, the material's current configuration is be applied to all sub frames.

Original

Sets the current material to its original configuration (from the model file). The button is disabled if the object's configuration is the original one.

Apply to Sub-Objects

Indicates that the current material's configuration must also be applied to all sub-frames that reference the same material.

Diffuse Color

Sets the Diffuse color. A diffuse color can be a plain, blinking or transparent color. If the diffuse color is transparent, meshes using that material will be rendered invisible.

Specular Color

Sets the Specular color. A specular color can be a plain, blinking or transparent color. If the specular color is transparent, the material is processed as if the color is set to black.

Emissive color

Sets the Emissive color. An emissive color can be a plain, blinking or transparent color. If the emissive color is transparent color, the material is processed as if the color is set to black.

Opacity

Sets the transparency percentage, from 0 (invisible) to 100 (fully opaque).

Specular Level

Sets the specular level, from 0 (dullest highlights) to 32767 (sharpest highlights).

The sharpest possible highlights may be limited to a specular level value of less than 32767. When the level is over that limit, the surface is rendered with no highlights.

How to change material properties at the model level

  1. Open the 3D Properties dialog by either double clicking on the model or its icon in the Graphic Explorer.
  2. Select the material to be changed from the Material combo-box.
  3. Enter new values for the properties to be changed.
  4. Tick the Apply to Sub-objects box.
  5. Click the OK button to apply the changes and close the dialog.

When 3D Properties dialog is opened at the model level, it displays the materials configuration from the original model. It does NOT display any changes that have subsequently been made using the 3D Properties dialog.

How to change material properties at the frame level

  1. Open the 3D Properties dialog by double-clicking on a frame icon in the Graphic Explorer.
  2. Select the material to be changed from the Material combo-box. A frame may have zero, one or more materials.
  3. Enter new values for the properties to be changed.
  4. If you require the change to be applied to any sub-frames also using that material, tick the Apply to Sub-objects box.
  5. Click the OK button to apply the changes and close the dialog

When 3D Properties dialog is opened at the frame level, it displays the actual materials configuration including any changes that have been made using the 3D Properties dialog.
