Putting 3D Models into a Mimic

When you open or create a 3D mimic, the 3D tools contained in the 3D Draw and 3D Mode toolbars become active.

  1. Put the 3D window into Draw mode.
  2. Click on the menu command Insert.3D Model. The Insert 3D Model dialog opens with a list of available models, i.e. DirectX (.x) files, marked by this icon .
  3. Select one of the models distributed with the HMI, for example cone.x, and click the Insert button. Show picture

  4. Switch to Run mode to see how the model appears. Show picture

You can only insert files in Microsoft DirectX format. For details of how to do so, see the topic Making 3D Models Available to the HMI.

If you are starting from scratch, or from an understanding of 2D techniques, please refer to the Getting Started Manual for 3D Mimics.
