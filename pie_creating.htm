Creating a Pie Chart

The Pie chart is flexible in both its appearance and operation and as a result, there are many configurable properties. The basic procedure to create a Chart is as follows.

  1. Open the mimic in which the Pie chart will be inserted and select the Design mode.
  2. Select the Insert.Pie/doughnut/Pyramid command on the menu.
  3. A Pie chart will be created and its properties dialog automatically opened. Show picture

Once you have created a Pie chart, and closed its properties dialog, it can be moved, sized and copied the same as any other drawing element.

The Pie chart configuration dialog contains several tabs each for a particular aspect of the configuration.

  • Display - General display features. See the topic Configuring a pie chart's general features.
  • Aspect - General appearance - the same as the Aspect tab of all drawing elements.
  • Chart - Configuration of the display chart. See the topic Configuring a pie chart's display.
  • Colors - Configuration of the colors for the segments. See the topic Configuring a pie chart's colors.
  • Content - Configuration of the segments. See the topic Configuring a pie chart's contents.

You can specify more properties to control the graphical element visibility. From the Aspect tab, behind the down arrow at the bottom of the Properties dialog box:

  • Specify zoom visibility bounds so that the graphic control is only visible for predefined zoom levels.

  • Specify layers so that the graphic control is only visible if one of the selected layers is displayed.

These are even more useful when combined with user permissions and decluttering templates.
