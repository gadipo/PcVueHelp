VariableFilter Method

 Applies to 

Changes the object to show each variable's name, title or both.

Syntax

Sub VariableFilter(Filter As String, Type As VariableType)

Filter corresponds to the regular expression in the Variables tab of the object's Drawing Properties dialog .

Type specifies the type of variable:

vtAlarm - Alarm
vtBit - Bit
vtRegister - Register
vtText - Text

These can be combined with the '+' operator, for example 'vtAlarm+vtRegister'.

This mode only works in variable tracking mode (with the Variable Tracking property selected).

This method is equivalent to the SCADA Basic verb SELECTOR, mode VARIABLE.
