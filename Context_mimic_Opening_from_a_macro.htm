Opening a Context Mimic from a Macro

A context mimic may be opened from a macro animation using the Context instruction. In this case, the mimic is opened when the macro is run which, like all control zones, occurs when the support drawing element is clicked on with the left mouse button.

Unlike opening a context mimic by clicking on an animation with the right mouse button, where only specific mimics may be opened, any mimic may be opened as a context mimic using the Context instruction.

The CONTEXT instruction

The syntax of the Context instruction is as follows.

CONTEXT,WinName,VarName

Where:

WinName

The name of the mimic to be opened.

VarName

The name of the variable for the context of the mimic.
