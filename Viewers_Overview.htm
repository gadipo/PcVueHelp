Overview of Graphic controls and Viewers

Graphic controls and Viewers are higher-level graphic objects that can be inserted in mimics.

Alarm viewer The Alarm Viewer is a control that displays a list of the current state of alarms.
Log viewer The Log Viewer is a control that displays events and actions that have been archived through a Log List.
Trend viewer The Trend Viewer is a control that produces a display of up to 100 register or bit variables plotted against time.
Grid control The Grid control provides a tabular display of text cells.
Sql grid control The Sql grid control enables the display data using the results of Sql queries.
Messaging control The Messaging Control provides access for a user to the Instant Messaging service.
Data Export Wizard The Data Export Wizard allows you export data directly from the Trend Viewer or the Grid Control to an Excel workbook, Csv file or the clipboard.
Chart control

The Chart (XY Plot) is a control that plots the historical values of one to eight variables (Y-Axis) against one other (X-Axis).

Pie Chart control

The Pie Chart displays one or more values as a pie, doughnut or pyramid style chart.
Map control The Map control displays interactive geographic maps, with an overlay of animated symbols representing real world objects.
Form controls The form controls include a combo-box, a list-box, a check-box-list, an option-button-list, a tree view and a text box.
Web browser control The Web browser control allows you to embed a standard web browser in a mimic to render web content.
BIM control The BIM control displays, and allows the user to interact with, a 3D BIM model defined by a .Ifc file.
ONVIF control The ONVIF control displays a video stream from an ONVIF compatible camera. It includes PTZ control.
Video control The Video control displays a general-purpose audio-video stream from either a network stream or a file.
ActiveX controls Mimics are ActiveX containers and offer you the possibility to insert and use third party ActiveX controls.

The tools to insert any of the controls or viewers in a mimic are found in the Insert menu.

You insert a control or a viewer in a mimic and apply a basic configuration to it in the same manner as for other drawing elements. However, the specialized features are also configurable, as appropriate to each kind of control.
