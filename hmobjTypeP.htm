Type Property

 Applies to 

Description

Returns the type of the variable as it was returned by the server. Read-only.

Syntax

object.Type [= fvVariableType]

Type can take one of the following fvVariableType values:

fvVariableTypeAlarm
fvVariableTypeAny
fvVariableTypeBit
fvVariableTypeRegister
fvVariableTypeText.

where fvVariableTypeAny means that the returned type is unknown.

The type value of an Alarm variable is equals to 9 as it is both an Alarm and a Bit.

A Type property and a VarType method appear in the Language Reference book of Microsoft VBA Help.
