Overview of LonWorks

The interface between PcVue and the LonWorks network uses an ActiveX control (software component), provided by Dialog semiconductor (formerly Echelon), known as the LonWorks LCA Object Server (or LonWorks Server). Information about the configuration of the network is provided by the LonWorks Database (LonDB) which must be first created using a 3rd party configuration tool.

 PcVue LonWorks interface supports the following features.

  • Compatible with IzoT Net Server 4.5,
  • Supports network variables and configuration properties IP (SNVT's, UNVT's, SCPT's and UCPT's),
  • Supports LonMark Object and Equipment Plugins.

The LonWorks configuration is accessed from the Application Explorer Project.Communication.Data acquisition.LonWorks.

Whilst you are configuring the LonWorks within PcVue there may be periods from a few seconds to several minutes whilst the application requests the LCA Object Server. During this time, a message dialog is displayed. You can close the dialog and cancel an operation by pressing the escape (Esc) key.
