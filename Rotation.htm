3D Rotation Animation

The 3D rotation animation is similar to its 2D equivalent in that one variable is used to interpolate linearly from a minimum value and angle to a maximum value and angle. What is different is that the axis about which the rotation takes place is a vector defined by X, Y and Z properties. Show picture

You can apply a rotation animation to a symbol or group, a single model or a frame. Rotation takes place about the pivot point.

Understanding the vector used for the rotation axis

The axis about which rotation takes place is defined by a vector with X, Y and Z properties. By using a vector the animation can rotate an object about one plane (as with a 2D object) or about a combination of two or three planes. This can be seen in the following picture.

As the axis is a vector, the values used for the X, Y and Z properties are not important. What is important is the ratio between them. For example, an axis with X, Y and Z of 1, 2 & 3 is exactly the same as one with 10, 20 & 30.
