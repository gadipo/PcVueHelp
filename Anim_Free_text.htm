Displaying free and pre-selected text

The Pre-selected Text animations display one or more texts that are configured in the animation itself.

Animation

Behavior

Text bit

2 texts according to the status of a single bit. Show picture

Text bit group

16 texts according to the binary combination of up to 4 bits. Show picture

Text Alarm

8 texts according to the 8 possible states of an alarm (Off, On - Nack, On - Ack, Off - Nack, Unavailable, User masked, program masked & variable masked). Show picture
Text register bit

2 texts using a bit masked from a register with a binary rank. For example, if you use the 3rd bit, when the binary representation of the register is xxxxxxxxxxxx1xx the text selected for 1 will be displayed. When the binary representation of the register is xxxxxxxxxxxx0xx the text selected for 0 will be displayed (x = either 1 or 0). Show picture

Text register value

Selection of up to 9 texts according to value of a single register. See the page Understanding the Thresholds Used in Color Register Value for an explanation of threshold operation. Show picture

In all but the Text Alarm animation, a further text may be configured that is displayed when the variable or variables are invalid.

Each text may be defined for each of the project languages.

Each of text animations may be configured with a quality sequence from the Sequence Manager.

Animation effects

Each Text animation displays a specific text according to the variable's value and the way the animation is configured.

The animation only processes variable values that are defined as available in the animation configuration (according to the check boxes associated with the variable values). If a variable value is not defined as available, the last text displayed remains unchanged.
