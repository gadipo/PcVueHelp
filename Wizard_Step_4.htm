Step 4 - Client and Server List Generation

The final step generates the client and server lists for the client and server stations you have created using the wizard. The following lists are created.

  • One list for each station, with a name based on the station's name
  • One list for each association, with a name based on the association's name.
  • One list of the names of all the stations, with a default name LCALL.
  • One list of the names of any client stations, with a default name of LCCLIENTS.
