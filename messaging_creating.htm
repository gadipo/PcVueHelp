Inserting a Messaging Control in a Mimic

You insert a messaging control in a mimic as follows.

  1. Insert the messaging control.
    1. Open the mimic in which the messaging control is to be inserted and select Design mode.
    2. From the Insert menu select Messaging control. An empty rectangular drawing element appears in the mimic to represent the messaging control and its properties dialog opens.
  2. Select the Aspect tab to configure the messaging control's appearance. Show picture
    1. Use the Border and Appearance properties to configure the style of the controls' border. These are almost identical to those used when configuring a solid drawing element such as a rectangle.
    2. Use the Position and Size properties to fine-tune the control's position and size.
    3. Use the Font and colors properties to change the background color and the font used for any text displayed by the control. Note that you must first tick the corresponding boxes to enable these properties.
