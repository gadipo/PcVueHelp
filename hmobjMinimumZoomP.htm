MinimumZoom Property

 Applies to 

Description

Returns or sets the minimum limit of the zoom factor that can be applied (in Run mode) to the object using the Zoom tools. Read/Write.

For a Graphic object the object will only be visible if the zoom range applied to its container is within the range defined by the MaximumZoom and MinimumZoom properties.

For a Window object the MinimumZoomproperty limits the zoom factor that can be applied in run mode.

Syntax

object.MinimumZoom [= Long]

The range for MinimumZoom is 0 to 6400 where 100 represents the original size. MinimumZoom must be less than MaximumZoom.

This limit is not taken into account when zooming is performed by a script.
