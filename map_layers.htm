Managing Map Layers

Map layers are managed using the Map Objects Tree and Layers Toolbar of the Map Markers Editor. Information about the operation of the Map Objects tree can be found in a topic of the same name here.

The Map Layers toolbar

The Map Layers toolbar contains two tools.

Icon Description
Add a new layer. The Layer properties are displayed with a default configuration for the new layer.
Delete the selected layers and selected markers (if any).

The default layer, displayed at the top of the Map Objects Tree, is mandatory and cannot be deleted.

The Map Layers properties

The map layers properties are displayed whenever a layer is selected. Show picture

  • Name - The name of the layer as displayed in the Map Objects tree.
  • Description - An optional description of the layer. The description is displayed in a tool-tip when the mouse hovers over the layer in the Map Object tree.
  • Zoom levels - The minimum and maximum zoom levels for which the layer is visible, at run time, in a map. The minimum and maximum can be set to the same level. In the above example, the minimum is 4 and the maximum 6.
