Displaying a Register as a Numeric Value

There are two animations that display a register as a numeric value called Register and Register & Bits. The animation Register & Bits is provided for compatibility with earlier versions of the software. Both animations use a text field as the support. The text field provides the font, color and the style (inverse, button etc.) properties.

Selecting the display format

The display format is selected using the format field in the Display Register tab.

Selects either the auto or the custom formats.

Allows the format to be supplied by a variable in the server.

Compatibility note - Register & Bits

The animation Value - Register & Bits, which allows you to display a register and change its color using a combination of up to 4 bits (2 for the text color & 2 for the background color), is provided for compatibility with earlier versions of the software. The functionality it provides is similar, but not identical, to using the Text - Register plus Color – Four Bits animations on the same drawing.
