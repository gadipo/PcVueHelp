Video control overview

The Video control displays a general-purpose audio-video stream (AV) from either a network stream or a file. The supported formats are Rtsp, .mp4, .avi, .mp2 and .wmv

At run-time the Video control has a context menu that can be used to control the AV stream. More advanced functionality can be added with SCADA Basic scripting. See the instruction VIDEODISPLAY for details.

While the Video control can display the RTSP stream from an ONVIF source, it cannot be used to control a camera (PTZ commands in particular). For this purpose, use the ONVIF control instead.
