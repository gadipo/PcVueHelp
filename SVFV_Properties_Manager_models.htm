Using Drawing Properties Models

A model is a set of pre-defined values for selected properties. The Drawing Properties Manager provides tools for managing properties models by creating, configuring, saving and deleting them. It is used either:

  • To initialize the properties of a drawing element when it is created (as in Default mode).
  • To apply a selection of a drawing element's property values to the drawing elements in the selection (as in Reference mode).

Creating a model

  1. Open the Drawing Properties Manager and select Default, Selection or Reference mode. The mode and drawing elements that are selected will determine the available properties and their values for the model.
  2. Click the Create Model button. The model creation dialog is displayed.
  3. In the Model Creation dialog, type a name into the Model combo box. Show picture

  4. Tick the properties that the model is to support.
  5. Click on OK to close the Model Creation dialog and create the new model.

Configuring an existing model

  1. Click the Configure Model button to open the Model Configuration dialog. Show picture

  2. Tick the properties that the model is to support and close the dialog.
  3. Change the properties that the model supports as required. Un-supported properties are grayed out.

The buttons for configuring, saving and deleting a model only appear when a model exists and is selected in the Mode drop-down list.

Saving a model

The Save button is displayed when an existing model is selected in the Mode combo box. It is disabled when there is no modification to save.

If you change a model and then attempt to close the Drawing Properties Manager, you will be prompted to save the model.

Deleting a model

The Delete button is displayed when an existing model is selected in the Mode combo box.

  1. Click on it to open a dialog asking you to confirm that you wish to delete the model.
  2. Click on OK to close the dialog and delete the model.

Creating new drawing elements using a properties model

When a model is selected in the Mode box, new drawing elements that are created or inserted in a mimic are initialized with the model's property values. If the new drawing element has properties that are not supported by the model, the value for the corresponding property in the global mode is used.

How to modify existing drawing elements using a properties model

  1. Select one or more drawing elements.
  2. Select the property model to be used.
  3. Click the command button Apply all to apply all the available model properties to the selected drawing elements.
    Or
  4. Click the command button Apply <tab> to apply just the properties of the selected <tab>.

In either case, the command is disabled if there is no difference between the model's property values and those of the selected drawing elements.
