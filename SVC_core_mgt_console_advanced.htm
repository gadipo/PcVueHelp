SV Core Management Console advanced use

The SV Core Management Console offers advanced features, including the ability to set some sensitive system parameters applying to Windows service applications, and a command line interface.

GUI resources tab

Show picture

Use this tab to tune Windows' resources when running PcVue as a service.

  • Size of the desktop heap - Change the size of the desktop heap. Increasing this value can be useful if PcVue is running as a service and has the web back end role, in order to server more WebVue clients. It should only be increased with great care as a too high value can put the computer in a no-boot situation.
  • Maximum number of user objects - Change the maximum number of user objects.
  • Maximum number of GDI objects - Change the maximum number of GDI objects.

Do not change these settings unless you fully understand the impact on the operating system. If in doubt, contact your system administrator or your support authority.

Using the SV Core Management Console from a command prompt

The SV Core Management Console can be run from a command prompt where it can be used to configure the run mode. Starting svCoreMgmt.exe without any arguments displays a list of the commands that are available. Show picture

The SV Core Management Console requires Administrator privileges. It must not be used to change the run mode whilst PcVue is running.
