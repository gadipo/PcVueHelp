Basic Trend Viewer Trace Configuration

The Trend Viewer traces are configured using the Curves and Advanced curves tabs. This topic covers the options most commonly used which are found in the Curves tab. Show picture



Selecting the variable for each trace

The name of the variable for each of the traces can either be manually entered or selected using the Variable Browser. You can select any register or bit variable but only variables that have been configured with trend recording should be used on a historical trend.

The Description field can be used to enter an optional description for the variable that will appear in the Trace Toolbox when opened at run-time. If you use the Legend, the variable's Description is automatically taken from its configuration.

The small tool to the left of the Description field can be used to temporarily hide a trace at run-time. This facility is also available from the Trace Toolbox and the Legend.

Configuring the trace appearance

The appearance of each trace can be configured using the following properties.

  • Line - The color and width of the trace.
  • Style - The style of the line used for the trace.
  • Mark - A marker that is plotted each time the variable attached to a trace changes. If the default of Null is chosen no marker will be plotted.
  • Type - Selects the way in which the trace will be displayed.

Each point representing a change in value is connected to the previous one with a single line.

Each point representing a change in value is connected to the previous one with a vertical and horizontal line.

Each point representing a change in value is plotted with no interconnecting lines.

Joining points together with a single line is often used for a drawing a trace because of its appearance. However, you must remember that the values are only valid at each apex, anywhere else on the line is interpolated. The more technically correct option is to join each point with a horizontal and vertical line.

Scaling the Y-axis

The Y-axis is independently scaled for each trace. For example, it could be 0 to 100 for one trace and -20 to 50 for another. If the Min and Max are both left at the default of 0 then the values are automatically taken from each variable's configuration. To display the scale at run time select the tool adjacent to the max and min.

Selecting the Auto fit property causes the Y axis to be auto scaled at run-time so that the minimum and maximum value of the Y-axis is taken from the actual minimum and maximum values of the visible trace. This option is disabled for a real-time trend.

Configuring the Y-axis to have a logarithmic scale

You can configure the Y-axis to have a logarithmic scale by selecting Semi-log in the Type drop-down list box. You must also then select a Maximum and Minimum using the drop-down list boxes that replace the normal Maximum and Minimum fields. When using the Semi-log option, Maximum and Minimum must be in the range 1e+20 to 1e-13. Show picture



Semi-log means that the Y-axis is logarithmic but the X-axis remains linear.

When using a logarithmic Y-axis the Auto fit option is not available.
