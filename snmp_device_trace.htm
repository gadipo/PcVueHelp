SNMP device traces

The SNMP Device Properties dialog contains a Trace tab that can be used to enable additional messages for the Event Viewer (F7). Show picture

  • Error level 1 to Error level 4 - Internal use only. Only enable if asked to do so by your support authority.
  • Full details - Adds additional information to any trace messages.
  • Polling (Get) - Display a trace message for each device poll. (SNMP Get message)
  • Polling (Ping/ICMP) - Display a trace message for each ICMP poll.
  • Writing (Set) - Display a trace message for each write message. (SNMP Set message)
  • OIDs browsing (GetBulk) - Display a trace message when browsing OIDs. (SNMP GetBulk message)
  • EngineID discovery - Display a trace message when discovering the EngineID of the SNMP agent.
  • Trap / Inform - Display a trace message each time a Trap / Inform message is received. In particular can be used in conjunction with Full details to discover the OIDs associated with Trap#6.
  • Save - Saves the trace configuration so that the same configuration will be used even after PcVue has been restarted.
  • OK - Starts the trace and closes the dialog. The trace configuration is temporary and is not remembered when PcVue is restarted.
