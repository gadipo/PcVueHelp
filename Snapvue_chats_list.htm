Chats List View

The availability of Instant Messaging is subject to license options.

The Chats List view provides access for a user to the Instant Messaging service. The Instant Messaging service allows a user to communicate, by text message, with one or more other users of the PcVue desktop application or SnapVue.

Instant messaging uses a conversation known as a chat. Chats are created, and owned, by a specific user. Once a chat has been created, the owner can invite other users and change user permissions. Chats are managed on SnapVue from the Chats view. Show picture

Tapping a chat displays the Chat view from where messages can be read and written and, if the user has the correct permissions, the chat settings can be changed.

How to add chat

  1. Tap the large '+' sign at the top of the Chats view. The New Chat view is displayed. Show picture


  2. Enter the Chat name. The chat name is only visible in the configuration and cannot be changed once the chat has been created.
  3. Enter the chat Title. It appears as the identifier in the Chats List and can be edited at run-time by a user with Owner privileges.
  4. Select the Storage time for messages. Messages are automatically discarded after this period.
  5. Select the users that will have access to the chat, and their privileges.
  6. Click the Done button to confirm creation of the chat or Cancel. The Chats view is displayed.

Once a chat has been created, it is never deleted.
