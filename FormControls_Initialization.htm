Initializing a Form Control

A Form Control's initialization defines how it appears when the mimic containing it is opened.

  • The Option-button, List box and Combo-box controls can have one item pre-selected, as defined by the Sel column in the Content tab or by the value of a variable.
  • The Check-box list control can have one or several items ticked, as defined by the Sel column in the Content tab.
  • The Tree-view control does not have the option of pre-selection but you can choose whether it appears with the nodes expanded.

The parameters that control initialization are in the Operations tab. Show picture

Option-button, List-box and Combo-box

The initialization pre-selection is defined by the Option drop-down list.

Command Meaning

Default item selection

The default selection as specified in the Content tab.

Index variable value

 The item whose index equals the value of the Index variable.

User data variable value

The item whose user data equals the value of the User data variable.

If the variable or its value is not usable, the default is used instead.

Tree-view list

The initialization is defined by the Expand drop-down list.

Command Meaning

None

 No nodes are expanded.

Roots

 Only the root nodes are expanded.

All

 Root nodes and all sub-nodes are expanded.
