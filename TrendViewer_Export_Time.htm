Export Wizard Step 2 - Time Range

This step of the Data Export Wizard selects the time period for which the data is exported. By default, the period is taken from the Trend Viewer or Grid itself. Show picture

You can choose the period using either a Start time and End time or a Start time and Period. The period is configured as a quantity (1 to 100) of time units (minutes, hours, days, weeks, months or years).

The maximum number of data samples per variable is 12000. If the period that you enter causes this to be exceeded, the most recent are used.

Select the Next button to continue to the next step.
