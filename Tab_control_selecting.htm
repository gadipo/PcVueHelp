Selecting the Mimics for Tabs

You select the mimics to be shown in tabs via the lower part of the Tab Control tab in the Window Properties dialog as follows. Show picture

You can configure as many tabs as can be displayed across the top of the container mimic.

You can left-click on a property to open an edit control for modifying its value.

  • Page - The page number of the included mimic. This setting is allocated each time a new page is added and is used to configure the included mimic that is first displayed when the container mimic is opened.
  • Mimic - The name of the mimic. The mimic name may be typed in directly or selected from the mimic selector dialog. Substitution strings such as #I, #U and #Mn can be used. They are processed in the same way as it would in a Link Open animation.
  • Mimic branch - The variable branch to be used with any animations in the mimic. The branch name may be typed in directly or selected from the Branch Selector dialog. Substitution strings such as #B and #WB can be used. They are processed in the same way as it would in a Link-Open animation.
  • Page caption - The caption that will appear in the tab corresponding to the mimic page. The size of the tab automatically adjusts to accommodate the length of the string. Substitution strings such as #B, #W and #WN can be used. Substitution strings starting with #W are interpreted as information about the mimic in the tab, not the container mimic.
  • Icon - Optional. An icon to be displayed on the tab. The icon file must be in the standard (.ICO) Windows icon format. Only icons that have been saved in the bitmap folder (BMP) of PcVue's libraries can be used. The icon is scaled to fit Icon width and Icon height properties.

For more information, on substitution strings, see the topic Using Substitution Strings or Variable Attributes in an Animation.

If any of the #M (#M1 to #M20) substitution strings are used for the mimic name, the corresponding branch and label are used for the Mimic branch and Page caption respectively. See the topic The Links tab for more information.

How to add a new page to the list of included mimics

To add a new page to the end of the page list, left-click New page to select the line and enter the properties.

To add a new page in the middle of the page list right click the line immediately below the position for the new page and select New from the context menu. Show picture

The context menu also contains commands to copy, paste, cut and delete lines.

Selecting the included mimic that is first displayed when the container mimic is opened

The Startup page drop-down list box selects the first page mimic that is displayed each time the container mimic is opened.

Configuring the fitting style for an included mimic

These properties specify how the page mimics appear within the tab control.

  • Position - The Position X and Position Y properties of each included mimic are automatically adjusted so that the top left hand corner of the included mimic is located at the top left hand corner of the tab control mimic area and the mimic does not overlap the page tab itself. The Position property should be ticked in most, if not all, applications.
  • Size - The included mimic's Width and Height are automatically adjusted to fit the tab control mimic area.
  • Control - The included mimic's style is modified to disable display of the border and title bar so only the content of the mimic is displayed.

 PcVue dynamically modifies some properties of the included mimics to accommodate the fitting styles. If an included mimic is saved, the changes will be saved and the original settings will be lost.
