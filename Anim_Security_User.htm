Using the User Animation

The User animation enables you to provide elements (usually buttons) on a mimic from which a user can log in, log out or display details of the current or any user.

The option you choose from the menu is pre-set in the dialog. If you then select another option from the drop-down list that is what will be applied.

To allow a user to log in

  1. Select the drawing element to which the animation is to be applied.
  2. From the menu, select Animate.Security.Login.
  3. Click on OK to apply the settings.

At run time when the user clicks on the element, the Security - Log On dialog is displayed.

To allow a user to log out

  1. Select the drawing element to which the animation is to be applied.
  2. From the menu, select Animate.Security.Logout.
  3. To display the Login/Logoff dialog for confirmation at run time, tick the property Show Dialog Box.
  4. Click on OK to apply the settings.

At run time when the user clicks on the element, the Security - Log Out dialog is displayed.

To allow display of a user's properties

  1. Select the drawing element to which the animation is to be applied.
  2. From the menu, select Animate.Security.User Information.
  3. To allow selection of the user from the User Properties dialog displayed at run time, tick the property Allows User Selection.
  4. Click on OK to apply the settings.

If you do not tick this box then only the properties of the current user can be displayed.

At run time when the user clicks on the element, the User Properties dialog is displayed.

To allow modification to the properties of Users and Profiles

  1. Select the drawing element to which the animation is to be applied.
  2. From the menu, select Animate.Security.User Manager. This option is then shown in the drop-down box as Security Editor.
  3. Click on OK to apply the settings.

At run time when the user clicks on the element, the User Manager dialog is displayed.

To allow modification to passwords

  1. Select the drawing element to which the animation is to be applied.
  2. From the menu, select Animate.Security.Password Modification.
  3. Click on OK to apply the settings.

At run time when the user clicks on the element, the Security - Change Password dialog is displayed.
