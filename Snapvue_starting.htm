Starting SnapVue

The SnapVue app is started by tapping the app icon. Show picture

If it is the first time ever that the app has been started on a device, you must configure an account. After the splash screen has been displayed for a few seconds, the Accounts view is displayed from where the account is be configured.

If an account has already been configured, but you have not previously logged in or the login has expired, you must log in. After the splash screen has been displayed for a few seconds the Accounts view is displayed from which the account to log in with can be selected.

If you have already logged in recently, and have not explicitly logged out, the Home view is displayed. Login are remembered even if the app has been stopped and the device turned off.

Starting the SnapVue app also starts the notification service and the geolocation detection service that run in the background.
