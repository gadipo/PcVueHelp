Displaying the Name of the Variable Used in an Animation

The following mimic displays the name of the variable used in an animation at run time.

  1. Create a mimic named CTXVAR. The Popup Behavior property must be enabled.
  2. In draw mode add a text string long enough to display the name of a variable (40 characters). The text string should be positioned in the top left hand corner of the mimic.
  3. Adjust the width and height of the mimic so that it is just large enough to display the text string. The border and drag bar of the mimic should be turned off.
  4. In Animate mode apply a Text with Bit animation to the text string. The text variable SYSTEM.CONTEXT.NAME should be used.
  5. Save the mimic.
  6. From the Preferences sub-menu on the Configure menu, select the Context Mimics dialog. Select the Bit animations, Register animations and Text animations tick boxes. Enter the name CTXVAR in the corresponding drop-down list boxes as the name of the mimic to be displayed.
  7. Confirm the configuration by selecting the OK command button.
