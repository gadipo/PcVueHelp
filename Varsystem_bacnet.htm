BACnet Status Variables

In the following system variable names:

  • <Network> is the name of a configured BACnet network,
  • <Device> is the name of a configured BACnet device,
  • <Schedule> is the name of a configured BACnet schedule object,
  • <Calendar> is the name of a configured BACnet notification class object,
  • <Log> is the name of a configured BACnet log object (event log, trend log or trend log multiple).

SYSTEM.BACNET.XON

R

Bit

Manual

Flow regulation status.
0 Flow regulation activated. Loss of exchanges possible
1 Flow regulation not activated.
