Entering a Text String

Strings are entered by the user from a control zone created with the Send Text animation. This animation may be applied to any of the solid drawing elements, but it is normally used with Text in which case the string may be displayed. Strings may be entered either from the keyboard, or by using the mouse and a displayed keyboard. Show picture

Selecting the text entry method

At runtime, when a user clicks on a Send Text control zone, a dialog is displayed containing the description of the text variable (or its name if it does not have a description) and the current string. The appearance of the dialog and the way in which the string is entered is determined by the Send mode properties.

  • Keyboard - The new string is entered by typing it directly into a field in the dialog.
  • Keypad - Similar to Keyboard except that an on-screen keyboard is displayed as part of the dialog. This is used in conjunction with the mouse to enter a string without using the physical keyboard.
  • Direct - The value is entered directly into the text field of the support drawing element using the keyboard. This option does not display a dialog.

Entry options

Property Description Keyboard Keypad Default
Direct The value is entered directly into the text field of the support drawing element. Yes No Yes
Display text Display the real time value of the text in the support drawing element. Yes Yes Yes
Password The entered value is displayed as a number of asterisks allowing the user to enter a value in secret. Yes Yes No
Empty field The current value of the text is not displayed. Yes Yes No
Default value The string used in Default mode. No No Yes
Show default value on mouse-over The default value is displayed when the cursor is over the control zone. No No Yes

Forcing the use of the Soft keypad with the Text-send animation

On touch screens, you can force the use of PcVue's soft keypad for all sending modes. To enable, use the option Soft keypad management.Touch screen.Use soft keypad for Send-register and Send-text animations for all sending modes in the HMI Options. See the topic Soft Keypad Management.
