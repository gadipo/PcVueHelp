Equipment Status Variables

For more information about the possible value of the general status variables detailed in this topic, refer to the topic General communication status.

For the possible values of the driver-specific values, refer to the topic specific to the driver in use.

If the EQT frame name is set to STATUS, it may generate error messages in the Event viewer. In such cases, you can customize the suffix for status variables to avoid these errors. For detailed instructions on changing the frame name, refer to the topic Equipment settings.
