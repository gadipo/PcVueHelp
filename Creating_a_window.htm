Creating a 3D Window

To create a 3D window:

  1. Run the HMI. If the DirectX software is correctly installed, you will see the New 3D icon in the standard toolbar. Show picture

  2. Click on it. The new 3D window will show an empty 3D scene with a checkered floor. Show picture

If the 3D button is absent, or you see a standard 2D mimic or a 3D mimic without a floor, please refer to the Checking DirectX on your PC topic.
