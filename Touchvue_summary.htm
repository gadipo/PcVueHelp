TouchVue dashboard view

The Dashboard view displays a summary of the variables from a given account. It is opened by tapping the account name in the Menu Drawer. Show picture

Screen features

The screen is divided into three areas.

  • Trend. A simplified version of the Trend View. Tapping the Trend takes you to the Trend View.
  • Alarms. A summary of the most recent alarms. Tapping a single alarm takes you to the Variable details view for that alarm. Tapping Go To Alarms takes you to the Alarms list view.
  • Lower menu bar. Tapping the icons takes you to the Browsing, Watch List or Logged events views.
