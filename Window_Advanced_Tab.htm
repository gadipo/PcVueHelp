Advanced Tab

The Advanced tab contains further properties to configure the window's behavior at run time. Show picture

Beep

The beep option generates a "beep" every few seconds on the PC's speaker whilst the window is open thus alerting the user to its presence. The beep is only generated if an enabling bit variable is selected and its value matches the selected state.

Screen Resolution

The property Design Screen Resolution is used to scale mimics automatically to appear in their original physical size when a project is run on a PC with a different screen resolution to that for which the mimics were originally designed.

It is typically used when an old project is moved to a new platform (in terms of PC specification, operating system and/or version of PcVue) that has a screen with a higher resolution than the original.

To enable mimic scaling you must also tick the option Adapt Mimic Size to Screen Resolution in the HMI Options.

Option Meaning
Design screen resolution

The screen resolution for which the mimic was designed. The drop down list box contains the commonly supported resolutions up to 2560 x 1600.
Current screen resolution The screen resolution of the PC on which the project containing the mimic is running. If the PC has more than one screen, the resolution of the primary screen is used.
