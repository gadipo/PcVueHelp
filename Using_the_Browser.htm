Using the Symbol Browser

To open the Library Browser dialog, select the menu Display.Library.

The Symbol Library Browser displays the contents of a Symbol Library as thumbnail pictures sorted into alphabetic order, rather than as just a list of names. The Library to be displayed is selected using the list box at the top of the browser. Show picture

The Browser may be used to select and paste symbols into a window.

To select and paste a symbol into a window

  1. Click on the symbol thumbnail in the Library. Hold down the mouse button and drag the cursor to the location in the window.
  2. Release the mouse button. A copy of the symbol will appear in the window.

What does it mean when a thumbnail is hatched?

When a thumbnail is hatched, it means that the symbol contains one or more animations. A thumbnail that is not hatched does not contain any animations.

What does it mean when a thumbnail contains the marker?

When a thumbnail contains this marker, there is problem with the symbol. In most cases, this will be due to the symbol referring to another symbol or image that does not exist. It can occur if you have copied symbols from one project to another without copying all the Libraries.
