Using the Selection Mode of the Drawing Properties Manager

The Selection mode is used to display and directly modify the properties of selected drawing elements. It works with one or several selected drawing elements from which it collects the property values.

This mode is mainly used for directly modifying properties, especially with multiple objects selected in a mimic. For example, you could change the font used in all selected Text drawing elements.

To copy the properties of one object to another object or to several objects, you should instead use the Reference Object mode or the Model mode of the dialog.

How properties are displayed with a single selection

When a single drawing element is selected, the Drawing Properties Manager automatically displays the appropriate tabs and updates its property controls with the values of the drawing element's properties.

The property controls for properties that the drawing element does not provide are disabled. For example, if a rectangle is selected then in the Base tab, the Shape, Curvature and Angles property controls and names are disabled.

Controls associated with properties that the drawing element provides but that are not available because of the current configuration are also disabled. For example, if the border is set with a Dashed and/or Dotted style, the Border Thickness property control and name will be disabled.

Properties for a multiple selection

When several drawing elements are selected, the Drawing Properties Manager works much as it does for a single selection:

  • It displays the appropriate tabs according to the objects that are selected.
  • It disables the property controls and names that are not provided by at least one of the objects.

When several drawing elements provide different values for a property it displays the reference drawing element value where possible and the property controls are displayed as undetermined. The following table shows the undetermined representation for each type of property control.

Rules when changing a selected property value

When a property value is modified, the following rules and restrictions apply.

Restriction

Comment

The Drawing Properties Manager directly applies the new value to all elements in the selection.

There is no Apply button.

Only the modified property value is applied, meaning that the Drawing Properties Manager does not change any other property even if it is associated with the modified one.

For example, changing the pattern style property from Solid to a hatched style will not set the hatch color property. The drawing element will use the last hatch color they have been set with, or a default color.

A drawing element ignores a property modification when the property is not applicable to it because of its current configuration.

For example, drawing elements with the pattern style set to Invisible will ignore any changes to pattern color.

The drawing element's internal process changes other properties to avoid producing an incompatible set of properties.

For example, following the modification of the border line style from Solid to a dashed and/or dotted style, the drawing element's internal process will automatically change the line thickness of the objects to 1, because it is not possible to have a dashed and/or style with a thickness wider than 1 pixel.

For details of how to use the dialog's controls to change the values of properties, see the topic Behavior of Drawing Property Controls.

Modifying an object in a symbol

You can select a drawing element inside a group or symbol by using the Graphic Explorer. Since a control in the Drawing Properties Manager directly causes a property value to change, the HMI will ask to save the symbol whenever a property of a drawing element located inside the symbol is changed.
