Data Export Overview

Data Export allows you to configure and generate export of PcVue's historical data.

Summary of main capabilities

  • Export of Trend and/or Log data in Microsoft Excel 2010 workbook (.xlsx) o .csv files.

  • Ability to extract data stored in Proprietary and HDS archive units.

  • Ability to export raw data, sampled data, statistics and aggregated statistics.
  • For the Excel output.
    • Excel not required on the host station.
    • Each configured data export generates its own file.
    • Each export creates one workbook with one or more worksheets each containing either Trend or Log data.
    • Additional worksheets generated if the optional statistics and/or aggregated statistics are enabled.
    • Option to use Excel workbook Templates to format the output.
  • For the Csv output.
    • Each export can optionally generates a sub-folder within the chosen output path. Each configured Log Page or Trend Page generates a corresponding file within that sub-folder.
    • Additional files generated if the optional statistics and/or aggregated statistics are enabled.
  • Fixed or variable export period.
    • Fixed - selectable from current or previous, hour, day, week, month or year.
    • Variable - by program or by variable (time and date value in register).
  • Export can be executed on-demand (manually by a user) or automatically as a background task (programmatically by SCADA Basic), on event, cyclically or according to a schedule.
  • Generation and configuration possible with run-time licenses.
  • Can be used to export data from Proprietary and HDS archive units.
  • User access managed by PcVue's User Accounts system.

Licensing

The Data Export module is licensed. If the Data Export is not license-enabled, it operates as if in trial mode.

Limitations

The following are the maximum limits for Data Export.

  • Number of exports that can be configured - 100.
  • Number of log pages per export - 10.
  • Number of trend pages per export - 10.
  • Number of trends per trend page - 100.

Operation in demonstration mode

The following operational restrictions apply when using Data Export in demonstration mode.

  • Configuration is fully functional.
  • Manual export is functional but restricted.
    • The output data values generated by an export is randomized.
    • The word "Demo" appears in the output, appended to the variable names and descriptions.
  • Automatic mode (using SCADA Basic) is disabled and attempts to use it will only produce a message in the Event Log.
