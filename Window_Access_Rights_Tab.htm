Access Rights Tab

The Access Rights tab contains properties that determine how the user can change the appearance of a mimic at run-time. Show picture

Tab options and command buttons

  • Level - The access level that the user must have in order to open the mimic.
  • Allow zoom – If selected the user is allowed to zoom the mimic between the specified Minimum and Maximum zoom levels.
  • Allow layer selection – If not selected, the user cannot change the layers that are displayed at run time and the layer tool in the run-time context menu is hidden.
  • Warning box – If selected a warning dialog is displayed if the user attempts to change the zoom or layers display when it is not permitted.
  • Warning beep – If selected a warning sound is produced if the user attempts to change the zoom or layers display when it is not permitted
