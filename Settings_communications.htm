Communication Settings

The communications settings are displayed from the Communications entry in the Application Explorer Main settings dialog or from the Settings task when the Communications folder is selected in the Application Explorer configuration tree. The communications settings affect the operation of Data Acquisition, Network applications, and PcVue server interfaces. Show picture

  • Data acquisition - See the topic Data acquisition settings
  • Networking - See the topic Networking settings
  • Servers - See the topic Servers settings
  • Regulation - See the topic Data Flow Regulation Settings.
