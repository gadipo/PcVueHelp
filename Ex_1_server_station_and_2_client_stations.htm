Architecture Examples - 1 Server Station and 2 Client Station

Project requirements

  • Station1 communicates with equipment and produces the value of variables.
  • Station2 consumes the value of all variables and displays mimics.
  • Station3 consumes the value of all variables and displays mimics.

The project configuration is identical on all stations.

License requirements

  • Station1 requires a server license.
  • Station2 requires a client license.
  • Station3 requires a client license.

Network configuration

  • The default network configuration is used with all stations configured as both client and server and with all links between all stations made.
  • Two station lists are required.
    A server list LSERVER containing the name Station1.
    A client list LCLIENT containing the names Station2 and Station3.

Variables tree configuration

  • All user defined variables are attached to the server list LSERVER and the client list LCLIENT.
