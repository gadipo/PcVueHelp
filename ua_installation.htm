OPC UA Gateway Installation

Prerequisites

  • You have an account with administrator privileges on the UA Gateway host computer.

Installation

The gateway is supplied as a self-installing executable file, and a license file named OEM.ini. The license file contains the license and the default configuration settings necessary for the gateway to work correctly with PcVue. The OEM license file must be in the same folder as the executable. Running the executable starts the installation process.

The gateway supplied with PcVue installation media will only work with the PcVue OPC Client and Server. It will not work with any other OPC client or server.

The gateway automatically starts, using default settings, after installation but there are two further configuration steps to complete before it is fully functional.

  • Use the Administration Tool to confirm settings including the user account, endpoints, and certificates. See the topic OPC gateway configuration.
  • Use the Configuration Tool to add and configure OPC UA servers, as well as the OPC DA server. See the topic Adding an OPC UA server to the gateway.

The configuration tools can be opened by right clicking the OPC UA gateway icon in the system tray or from the Windows' Start menu.
