Creating a PcVue Life Indicator

 PcVue Life Indicator allows you to configure a variable, the value of which is periodically changed so that a PLC or other equipment can check if PcVue is running.

Explanation of fields

  • Reference name - A 40 character string that identifies the life indicator. The name is used when information about the operation of the life indicator appears in the trace window.
  • Name of list of producers - The name of the list of producers if used on a multi-station application.
  • Comment - A free format comment of up to 128 characters.
  • Active - Must be set to 1 for the life indicator to be active, else 0.
  • Name of the variable to change - the name of the variable whose value is periodically changed. The variable can be a bit or a register and must have the control attribute set in its configuration.
  • Type of change - how the variable value is changed at each interval.
    • 0 - A bit variable is set to 0.
    • 1 - A bit variable is set to 1.
    • 2 - A bit variable is toggled.
    • If a register variable is used the value is incremented by 1 each time, and this field is not used.
  • Monitor period - the period, in seconds, at which the variable is changed.

Example 1

Toggles the bit variable WATCHDOG.BIT every 10 seconds.

TREATMENT,WATCHDOGSV,WD_EQT1,0,,,PcVue watchdog,,,,1,WATCHDOG.BIT,2,10

Example 2

Increments the register variable WATCHDOG.MV every 60 seconds.

TREATMENT,WATCHDOGSV,WD_EQT1,0,,,PcVue watchdog,,,,1,WATCHDOG.MV,,60
