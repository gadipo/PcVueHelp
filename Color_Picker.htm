The Color Picker Dialog

Do not confuse the Color Picker dialog with the Color Picker and Color Dropper tools.

Colors in the Palette are changed using the Color Picker dialog. To display the Color Picker dialog, double-click on the color in the Palette that you want to change. Show picture

There are three color styles available.

  • Solid - A single color.
  • Blinking - A blinking color contains two color selections. The affected part of the drawing element will alternate between the two colors at a pre-defined rate.
  • Gradient - A gradient color contains between two and eight colors. The affected part of a drawing element will be shaded starting with one color and gradually changing to the other colors across the drawing.

The Color Picker dialog is also displayed when you double click on a color box in any Drawing Properties dialog.

A color is defined by the three components of hue, saturation, and brightness.

  • Hue describes the color's shade or tint. It is measured on a circular spectrum running from red to green to blue and returning to red. Hue is selected on the color circle by the angular position of the color pick dialog with respect to the center of the circle.
  • Saturation describes the hue's purity. A color with a saturation of 255 is bright and vivid, and a color with a saturation of 0 is a shade of gray. Saturation is selected by the distance of the color pick dialog from the center of the color circle.
  • Brightness describes the color's brightness. A color with brightness of 0 is black.
