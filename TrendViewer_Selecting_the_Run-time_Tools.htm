Selecting the Trend Viewer's Run-time Features

The features available to the User at run-time are selected from the Execution tab. Show picture



General

  • Historic data request - Enables the Historic button on the tool bar at run-time.
  • *Toolbar position – Allows the User to change the location of the toolbar.
  • *Toolbar modification – Allows the User to select the tools that appear on the toolbar.
  • Right click popup – Enables display of the pop-up menu at run-time that is used to select the cursor mode.
  • Save runtime properties – Automatically saves any changes made to the Trend Viewer, during run-time, when the mimic is closed.
  • Lock legend position - Disables movement of the split bar that allows the trace and legend areas to be resized at run-time.
  • Remove a curve from the Legend Tab - Allows a variable to be removed from the Trend Viewer by right clicking on its entry in the Legend.
  • Include variables when restoring configuration - In run mode, when selecting the Restore tool, the variables displayed on the trend revert to those selected at design time.

* Normally these options would be disabled when the project has been completed to prevent the end-user from changing the toolbar.

The Trace Toolbox

The Trace Toolbox is included only for backwards compatibility. The functionality it provides has been superseded by that provided by the Legend.

The Trace Toolbox provides additional information about each of the traces in the Trend and allows the User to modify the traces. The Trace Toolbox and each of its properties may be enabled or disabled as required. Note that any changes made at run-time using the Trace Toolbox are temporary and are lost when the Trend display is closed.

  • Auto open trace toolbox - If selected the Trace Toolbox is automatically opened when a trend is displayed (when the mimic containing it is opened).
  • Enable trace toolbox - Enables the User to open the Trace Toolbox.
  • Visibility – Enables individual traces to be temporarily hidden. This can be useful when one trace is obscuring another.
  • Color – Allows the color of each trace to be changed.
  • Scale – Allows the scale for each trace to be displayed or hidden.
  • Min/Max – Allows the range for each trace to be changed.
  • Enable advanced properties – Allows access to the toolbox advanced properties, all of which may be individually enabled and disabled.
  • Select variable – Allows the variable for each trace to be changed.
  • Trace width – Allows the width of each trace to be changed.
  • Trace style – Allows the plotting style for each trace to be changed.
