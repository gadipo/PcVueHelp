Starting PcVue for the First Time

After installation, PcVue is started by clicking on its icon in the Start Menu or on the desktop. The first thing that appears is the loading window (the Event Viewer) that displays status and error messages while the various software managers are being loaded. Show picture

Once PcVue is running, the Event Viewer is hidden by the main window. It can be moved in front of the main window at any time (providing the Event Viewer is not minimized) using function key F7.

The first time you run PcVue, the Event Viewer opens in English. This start-up default language can be changed by modifying the Languages settings.

Modes

If you try to start the software without a hardware protection key, a dialog will ask you to insert one. Show picture

  • To run PcVue in demonstration/evaluation mode, select Start in demonstration mode.
  • To run the mode appropriate to the hardware protection key, insert the key and select Try again.
  • To cancel startup, select Exit.

For details of versions of PcVue, see the Installation book.

The Event Viewer will display some more messages about loading and creating variables.

Creating or selecting a project

Starting and running a project on a networked disk drive is not supported.

You can create a new project or select an existing one as follows.

  1. Select the menu Configure.Project.Select Project to open the Select Project dialog.
  2. Either select an existing project from the list, or type in a new project name and select OK. Show picture

  3. Before a new project is created, you are asked to confirm the action. Select OK.

The complete path for the project must not exceed 255 characters.

If you select a project created with an earlier version of PcVue, which uses an HDS database, you may have to migrate the database prior to starting the project. See the topic Overview of database migration for more information.
