Selecting the Output and Input Format

If you have chosen to display the register value, you may select the output (display) format using the following command buttons next to the Format field.

Selects either the auto or the custom formats.

Allows the format to be supplied by a variable in the server.

Show picture

If you have selected a time and date style output format then you can also select a format for entering the value. The conversion to the raw value in the register is handled automatically by the software.

You can only use custom input formats when you have a time and date style output format selected.
