About Formulae

A formula is the name given to the mechanism that calculates the value of a variable using an algorithm, and the value of a number of other variables.

The formula mechanism supports a number of industry standard algorithms, plus two that are User configurable. The available algorithms are:

Flow Calculation 1

F = c1*v1*(v2+c2)/(v3+c3)

Flow Calculation 2

F = c1*v1*sqrt((v2+c2)/(v3+c3))

Flow Calculation 3

F = c1*sqrt(v1*(v2+c2)/(v3+c3)*(1-c5*v1/c6max))

Integral

C = C+(v1*Period/3600)

Standard Operator 1

R = (op1) v1 (op2) v2 (op3) v3 (op4) v4

Standard Operator 2

R = R+((op1) v1 (op2) v2 (op3) v3 (op4) v4)

Where possible, formulae should be given preference over calculating a value with a SCADA BASIC program, as formulae are considerably more efficient.
