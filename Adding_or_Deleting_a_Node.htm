Adding or Deleting a Node

It is sometimes necessary to add or delete a node from a polyline, polygon, Bézier curve or Bézier shape. For example, you may have made an error when drawing the shape.

To add or delete a node, first select the drawing element and then right-click on either the node to be deleted, or a node adjacent to where the new node is to be added. A context menu will be displayed. Selecting the minus command will result in the node being deleted. Selecting the plus command will add a new node half way between the selected node and the next node in the shape.
