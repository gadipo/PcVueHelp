Overview of Images

The HMI supports the use of images in the following formats.

  • BMP
  • GIF and animated GIF
  • JPEG
  • PNG
  • WMF
  • EMF

It also supports multimedia files in AVI format.

The use of images can considerably enhance the appearance of your application. However, large images or those with a large number of colors can consume significant system resources when they are rendered.

Before an image can be used in the HMI, you must move it to the correct folder using the Windows Explorer or the Libraries Manager (Application Explorer). The image must reside in either:

  • The Bitmap folder of your project.
  • The Bitmap folder of any Libraries exposed to the project.

An image is inserted as a reference. That is, the image file name is inserted in the mimic definition file. This means that, even when it has been inserted into a mimic, Microsoft Paint or any other drawing package may still edit an image. Any changes made will be visible the next time the mimic is opened.

There are many variations and options for the encoding of GIF, animated GIF, JPEG, PNG, WMF and EMF images. If an image does not render correctly it is likely that PcVue does not support the particular encoding variation of that image. Consider changing the encoding using a suitable 3rd party graphic editor before trying again.

How to insert an image

  1. Click on the Image button in the Draw toolbar. A dialog will be displayed containing a list of available images and an area in which they may be previewed.
  2. Select the Library in which the image is located.
  3. Select the name of the required image from the list. If necessary, a lexical filter may be entered to reduce the number of images displayed in the list.
  4. Insert the image by either:
    1. Clicking the Insert button. The image will be placed at the top left hand corner of the mimic. Drag the image to the desired location in the mimic.
    2. Click on the name of the image in the list and drag it to the required location in the mimic.
    3. Click on the thumbnail of the selected image and drag it to the required location in the mimic.

About the Insert Image dialog

The Insert Image dialog displays a list of images from the selected library. A thumbnail of the selected image is displayed at the top right corner.

The images list can be displayed in several formats selected by the tools above the list.

Format Description
Detail Displays the available images as a list of one per line including the size of the image.
List Displays the available images as a tabular list. Only the image name is displayed.
Small icon Displays the available images with a small icon representing each image.
Medium icon Displays the available images with a medium thumbnail representing each image.
Large icon Displays the available images with a large thumbnail representing each image.
Refresh Used to refresh the list in the event that the content of the library has been modified while PcVue is running.

If you choose to display the images in as thumbnails there may be a small delay while the thumbnails are generated
