The Chat View

The Chat view is displayed by tapping on a chat in the Chat List, or after creating a new chat. Show picture

Messages from the local user are displayed on the right, messages from other users on the left.

The chat settings and filter can be changed from the menu tool at the top right corner if the chat user has the correct privileges.

How to send a message

New messages are added by typing in the text box at the bottom of the chat view and clicking the Send button. Note that the send button only appears when there is a message to send, replacing the Record button.

How to change the chat settings

Changing the chat setting requires Owner privileges. The Settings view is displayed by tapping the Settings tool and contains two tabs, Settings and User. Show picture

The following settings are available.

  • In the Settings tab
    • Chat title - The title of the chat, which appears in the chats list.
    • Appearance - Select the chat color. A color is allocated automatically when a chat is created but may be changed later.
    • Storage time of messages - The message storage time. Messages are deleted automatically after this period, unless Unlimited is selected. Note that this setting is applied retrospectively to messages that have already been sent.
  • In the Users tab
    • Users - Invite and remove users, and / or change their permissions.
    • Quit the chat room - Leave the chat. A chat must have at least one owner, so if the original owner wishes to leave the chat, another user must be given owner privileges.

Tap the Done button once the changes are ready.

How to change the message filter

The Message Filter dialog is displayed by tapping the Filter tool and contains two tabs Date Time Range and Users Filter. Show picture

  • Date Time Range - Display only messages that were sent between the start and end dates.
  • Users filter - Display only messages from the selected users.
