Configuring a Function Key

  1. Creating a function key and configuring the general properties.
    1. Select the Function Keys folder in the Application Explorer configuration tree and, from the task list, select Add function key. A dialog to configure the new function key will open. Show picture
    2. Enter a description of the function key. The description is optional and is only displayed in the Application Explorer.
    3. Select or enter the name of a bit to enable or disable the function key. The bit must be set to 1 for enable. The bit is optional.
    4. Tick the Enable box. It must be ticked for the function key to be enabled.
  2. Select the Trigger tab and configure the trigger.
    1. Select the key using the drop down list box.
    2. Select any modifiers (Shift and / or Control) to be used in conjunction with the key.
    3. Select the key event on which the action is to be triggered (Up or Down).
  3. Select the Action tab and Select the action type from either Standard Action, Function to Execute or Send Animation.
  4. If triggering a Standard action then:
    1. Select the action using the drop down list box.
  5. If triggering a Execute a function then:
    1. Enter the name of the SCADA Basic program that is to run. The ellipsis button adjacent to the Program name field displays a list of available programs. The program must exist or you will not be able to create the function key.
    2. Enter a variable branch for use with the program. The branch is optional.
    3. Enter the name of the function, within the program, to run. If you do not specify a function, the Main function will be executed.
    4. Enter a comma'separated list of arguments for the function. The arguments can be retrieved in the function using the GETARG instruction. The list of arguments is optional.
  6. If triggering a Send animation then:
    1. Enter the name of the mimic that contains the animation. The ellipsis button adjacent to the Mimic field displays a list of available mimics. The mimic must exist or you will not be able to create the function key.
    2. If the mimic requires a variable branch, then select or enter it.
    3. Enter the name of the animation. The animation name can be found using the Graphic Explorer when the mimic is open.
  7. Click OK to confirm the configuration and close the dialog.

 PcVue does not warn you about, or prevent you from, configuring more than one action on the same key / modifier / event combination. If you do that, all configured actions are triggered when the key is used.
