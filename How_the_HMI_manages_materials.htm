How the HMI Handles Materials

When rendering the surface of a mesh, the HMI uses all the material properties from the 3D model file. However, only the following properties are available from the 3D Properties dialog.

  • Diffuse color
  • Specular color
  • Emissive color
  • Specular level
  • Opacity

Materials that are not indexed have a temporary index created by the HMI so the material is available for selection, and the properties for editing, from the 3D Properties dialog. The index name for non-indexed materials is produced as follows.

  1. The HMI creates an index named 'Material #x' where x is a number from 0 to the number of different non-indexed materials inside a model file.
  2. Meshes with the same material properties share the same indexed material.
