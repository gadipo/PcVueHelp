The Link Open Animation

The Link Open animation may be applied to any drawing element although normally it will be applied to a text string with the text used as an indication to the user. Show picture

The name of the window to be opened is selected using the drop-down list box.

You can use substitution strings in the mimic name to make the animation context sensitive. See the topic Using substitution strings in an animation.

Opening a Window With a Branch

When a Link Open animation is configured an optional branch may be supplied which is then used with the window when it is opened. Subsequently any reference to variables within that window is relative to that branch. Used in conjunction with the structured server this technique can be used to reduce the number of windows required and the subsequent development time.

Several copies of a window may be opened at the same time as long as they have different branches.

Using Relative or Absolute Opening Positions

You specify the position at which the window is opened using the X Position and Y Position properties.

If you select the Relative property, the position in which the window is opened is determined by the position properties plus the location that the window was originally saved in. For example, if the window was saved at position 50, 70 and a relative position of 30, 10 is supplied, the window will be opened at 80, 80.

If you select the Absolute property, the window is always opened at the position specified in the animation. The position in which the window was originally saved is ignored.
