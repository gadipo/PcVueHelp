POW

See also

Power function.

WebVue support - Yes.

Syntax

DblVal = POW(x, y);

Return type: DOUBLE.

Argument

Meaning

x andy

Any numeric type.

Execution

Return:

x to the power of y.

Example

SUB Main()

DIM dblResult as double;

DIM dblValue1 as double;

DIM dblValue2 as double;

dblValue1 = 2.5;

dblValue2= 2;

dblResult = POW(dblValue1,dblValue2);

PRINT("Result of 2.5^(2) = ",dblResult);

'Display "Result of 2.5^(2) = 6.25"

END SUB
