Configuring an Alarm Viewer, Log Viewer, Trend Viewer or Grid toolbar
 

 
 
 
 
 
 
 
 
 
 
 
 
 
Configuring an Alarm Viewer, Log Viewer, Trend Viewer or Grid Toolbar
 
You can allow the User to change both the position and contents of the Alarm Viewer, Log Viewer, Trend Viewer or Grid toolbar at run-time. 
 
 
To enable the User to change the toolbar position, enable the property - Toolbar position.
 
To enable the User to change the toolbar contents, enable the property - Toolbar modification.
 
 

 
The Developer can permanently change the toolbar by enabling the toolbar modification properties, changing the toolbar at run-time and then disabling the toolbar modification properties in design mode before the User is allowed access.
 
 
How to change the toolbar position at run-time
 
 
In Run mode, right click on the toolbar. The toolbar context menu is displayed. Show picture
 
Select the new toolbar position from either of the tools representing left, top, right or bottom.
 
 

 

 
 

 
 
How to customize the toolbar
 
In Run mode, the contents of the toolbar are changed using the Customize Toolbar dialog that is opened from the tool in the toolbar context menu. Show picture
 
Using the dialog, you can add, remove and change the position of the available tools.
 

 

 
 

 
 
How to add or remove a tool from the Toolbar
 
 

 
Select the tool to be added in the Available toolbar buttons list.
 
 

 
Select the Add button. The tool is added at the end
 of the Toolbar Buttons list.
 
 
 
Or click on the tool
 to be added in the Available toolbar buttons list, then drag and drop it in the
 desired position in the Current Toolbar Buttons list.
 
With the exception of the Separator, as tools are added
 they are removed from the Available toolbar buttons list. The Separator
 can be inserted repeatedly.
 
Removing a tool is the reverse process.
 
How to change the position of a tool in the Toolbar
 
 

 
Select the tool to be moved in the Current Toolbar
 Buttons list.
 
 

 
Click the Move up or Move down button as required.
 
 
 
Or click and drag the tool to its new position list During the drag process a small black arrow will be displayed indicating the new position.

Configuring the Line Format for an Alarm or Log Viewer
 
The active part of an Alarm or Log Viewer is the display list that contains one or more lines each displaying one item of logged data.
 
The number of lines that are available depends on the size of the viewer and the font chosen. The font used is selected using the Line font button. The color in which the text of each line is displayed is configured from the Preferences menu.
 
The display list is arranged in up to eight columns. The information displayed in each column is configured from the Display tab using special format characters. You can also enter a caption (title) for each column that will appear in the list header. Any columns that do not have format characters, and do not have columns with format characters to the right of them, are not displayed. Show picture

Configuring the column widths
 
The width of each column is configured at run-time by clicking and dragging the column boundaries in the header. After you have changed the column widths, the mimic containing the viewer must be saved or the changes will be lost when the mimic is closed. To change the column width the property Size of Columns, in the Execution tab, must be ticked. To prevent the User from changing the column widths later, Size of Columns must be un-ticked when the Viewer configuration is finished.

Overview of Graphic controls and Viewers
 

 
 
 
 
 
 
 
 
 
 
 
Overview of Graphic controls and Viewers
 
Graphic controls and Viewers are higher-level graphic objects that can be inserted in mimics.
 
 
 
 
 

 Alarm viewer
 
 The Alarm Viewer is a control that displays 
 a list of the current state of alarms.
 
 

 Log viewer
 
 The Log Viewer is a control that displays 
 events and actions that have been archived through a Log List.
 
 

 Trend viewer
 
 The Trend Viewer is a control that produces 
 a display of up to 100 register or bit variables plotted against time.
 
 

 Grid control
 
 The Grid control provides a tabular display of text cells.
 
 

 Sql grid control
 
 The Sql grid control enables the display data using the results of Sql queries.
 
 

 Messaging control
 
 The Messaging Control provides access for a user to the Instant Messaging service.
 
 

 Data Export Wizard
 
 The Data Export Wizard allows you export data directly from the Trend Viewer or the Grid Control to an Excel workbook, Csv file or the clipboard.
 
 

 Chart control
 
 
 
The Chart (XY Plot) is a control that plots the historical values of one to eight variables (Y-Axis) against one other (X-Axis).
 
 
 

 
 
Pie Chart control
 
 
 
 
The Pie Chart displays one or more values as a pie, doughnut or pyramid style chart.
 
 
 

 Map control
 
 The Map control displays interactive geographic maps, with an overlay of animated symbols representing real world objects.
 
 

 Form controls
 
 The form controls include a combo-box, a list-box, a check-box-list, an option-button-list, a tree view and a text box.
 
 

 Web browser control
 
 The Web browser control allows you to embed a standard web browser in a mimic to render web content.
 
 

 BIM control
 
 The BIM control displays, and allows the user to interact with, a 3D BIM model defined by a .Ifc file.
 
 

 ONVIF control
 
 The ONVIF control displays a video stream from an ONVIF compatible camera. It includes PTZ control.
 
 

 Video control
 
 The Video control displays a general-purpose audio-video stream from either a network stream or a file.
 
 

 ActiveX controls
 
 Mimics are ActiveX containers and offer you the possibility to insert and use third party ActiveX controls.
 
 
 
 
The tools to insert any of the controls or viewers in a mimic are found in the Insert menu.
 
You insert a control or a viewer in a mimic and apply a basic configuration to it in the same manner as for other drawing elements. However, the specialized features are also configurable, as appropriate to each kind of control.
