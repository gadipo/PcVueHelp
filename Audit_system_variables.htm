Audit System Variables Naming Rules

The audit system variables are provided to animate mimics, create trends and generally facilitate troubleshooting. The audit variable names all start with System.Audit and are constructed using the following branches.

<Source>

Indicating the data acquisition source selected from the following.

  • All
  • Internal
  • External
  • Equipment
  • DDE
  • OPCClient
  • OPCUA
  • LON
  • BACnet
  • IEC61850
  • IEC104
  • SNMP

<Data type>

  • DA - Data acquisition
  • DC - Data command

<Counter type>

  • Flow
  • Instance
  • ElapsedTime

<Manager>

The manager concerned.

  • UserInterface
  • Alarm
  • Historical
  • Lan
  • OPCServer
  • VarManager Toolkit<X> with X in the range of 1 to 8.
