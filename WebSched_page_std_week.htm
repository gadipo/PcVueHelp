Standard Week Page

The Standard Week page is displayed using the Standard Week tab. It displays the standard week for the schedule. Show picture

The Standard Week page displays the time intervals of the selected schedule not taking into account the configuration of any Exception Periods and Effective Period (BACnet only). Users with configuration rights can edit the intervals that make up the Standard Week. See the topic - Editing a schedule.

Standard Week page main features

  1. The identity of the schedule to which the standard week applies.
  2. The default output of the schedule - that is the output when there is no active interval.
  3. Edit toolbar. Edit tools for the Schedule Control. See the topic - Editing a schedule.
  4. Schedule Control. Provides the main display and edit features for the schedule. See the topics - Schedule Control main display features and Editing a schedule.
  5. Edit mode tools. See the topic - Editing a schedule.
  6. Schedule Control zoom tools. See the topic - Schedule Control main display features.
