Using the Alarm Viewer to Invoke an Alarm's Associated Action

If you configured one or more associated actions when you created an alarm, you may access them from the Alarm Viewer by clicking on the alarm with the mouse. A list of associated actions will appear adjacent to the mouse pointer. To select the action click on it with the left mouse button. Show picture



If the context menu has been enabled, and associated actions have been configured for the right mouse button, the associated actions appear as part of the context menu.
