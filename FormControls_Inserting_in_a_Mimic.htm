Inserting a Form Control in a Mimic

You insert a form control in a mimic as follows.

  1. Insert the form control.
    1. Open the mimic in which the form control is to be inserted and select Design mode.
    2. From the Insert menu, select one of the form controls: Combo-box, List-box, Check-box list, Option-button list, Tree View, or Text-box. An empty rectangular drawing element appears in the mimic to represent the form control and its properties dialog opens. Show picture

  2. Configuring the form controls appearance.
    1. Use the Border and Appearance properties to configure the style of the form controls' border. These are almost identical to those used when configuring a solid drawing element such as a rectangle.
    2. Use the Position and Size properties to fine-tune the form control's position and size.
    3. Use the Font and colors properties to change the background color and the font used for any text displayed by the form control. Note that you must first tick the corresponding boxes to enable these properties.

You can specify more properties to control the graphical element visibility. From the Aspect tab, behind the down arrow at the bottom of the Properties dialog box:

  • Specify zoom visibility bounds so that the graphic control is only visible for predefined zoom levels.

  • Specify layers so that the graphic control is only visible if one of the selected layers is displayed.

These are even more useful when combined with user permissions and decluttering templates.

How to configure the default icons

The default icons are selected from the Aspect tab. At the time of writing, only the Tree-view control supports icons. Show picture

To select the icons you first choose the root number and child level and then the Collapsed node icon or Expanded node icon button as appropriate. The Icons Selector dialog is displayed from where the icons are selected.
See the topic The icons selector for information on how to use the Icons Selector.

The icons you choose will be used as a default for all node instances. For example, in the picture above, the blue arrows would be used by default for all instances of Root 1, Level 1 nodes. To select an icon for just the root number, leave the Child level blank.

The button Reset default icons deletes all the default icon configuration.

The icons, for each node instance, can be changed from their default using the Content tab. See Configuring the content of a Tree-view for more information.
