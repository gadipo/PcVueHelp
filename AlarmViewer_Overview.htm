Alarm Viewer Overview

The Alarm Viewer is a control that displays a list of the current state of alarms. Any number of Alarm Viewers may be embedded in a window, the only limitation being the amount of physical space available.

  • Alarm Viewers may be embedded in any mimic.
  • Unlimited number of Alarm Viewers in each project.
  • Size of alarm list configurable from 1 line to maximum allowed by screen resolution.
  • The alarms to be displayed may be filtered by Domain and Nature, level and status. For example, all alarms on and not acknowledged.
  • The format of the alarm text 100% configurable.
  • Toolbar and context menu to select which alarm transitions are displayed, acknowledge alarms, mask alarms and change the mode of operation.
  • Filter tool to select the Domain, Nature and level of the alarms that are displayed.
  • Can be scripted by SCADA Basic for more advanced features.
  • Can have a context mimic. For details, see the topic Developing the HMI.Working with Mimics.Context Mimics.Opening a Context Mimic Using the Right-hand Mouse Button.
