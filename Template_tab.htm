The Template Tab

In the Window Properties dialog for any mimic based on a template, you can select which of the properties of the template the mimic will inherit. You can also change the choice of template on which the mimic is based or even remove it altogether. Show picture

What the animation detection priority does

Consider the case where a mimic and the template on which it is based both have animated controls in the same position. Show picture

In Run mode, the outcome will depend on the priority setting.

  • Priority to template commands - The control in the template takes precedence.
  • Priority to Mimic Commands - The control in the mimic takes precedence.
