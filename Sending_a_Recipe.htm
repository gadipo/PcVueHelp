Sending a Recipe

A recipe is a list of variables and corresponding values. For more information, on how recipes are configured see the book on Recipes in Configuration. Recipes are sent at run-time using the Send Recipe animation that may be attached to any drawing element. Options allow the recipe to be sent directly, with confirmation, with selection and with modification. Show picture

If you are sending a specific recipe, you must enter its reference number or title in the Send Recipe animation tab.

If you are allowing the user to select the recipe at run-time then these fields may be left blank.

Selecting the run-time behavior

The send mode properties define what happens when the user clicks on the control zone at run-time.

  • Direct - Send the recipe without giving the user the option to confirm or cancel. The number or title of the recipe must be entered in the animation tab.
  • Confirmation - Send the recipe displaying a dialog giving the user the option to confirm or cancel. The dialog displays the recipe number and title.
  • With beep - Sound the PC's speaker when the recipe is sent.
  • Selection - Displays a box allowing selection of the recipe before it is sent. If a family is entered in the animation tab, at run-time the user may only select recipes from that family.
  • Modification - Displays the recipe variables and values allowing them to be modified before they are sent.

Using a variable branch

If you enter a variable branch in the Branch field of the Send Recipe tab, any variable names in the recipe not preceded by the @ character will be prefixed with the branch.

Using Send Recipe at run-time

The Send Recipe animation has options that affect its operation at run time.

The ability of a user to send or modify recipes at run time is affected by access rights.

  • Sending a Recipe with Selection - When the Selection property is enabled, a dialog allowing a recipe to be chosen is displayed when the recipe control zone is selected. The dialog displays a list of recipes, filtered by family, sorted into numeric order. The display list is made up of pages each capable of displaying 512 recipes. If you have more than 512 recipes, you can use the Page property to select another page. A recipe is selected for display or to be sent by double-clicking on its entry in the display list. Show picture
  • Sending a Recipe with Modification. If the Modification option is selected then the recipe configuration is displayed when the recipe control zone is selected. The recipe variable list may be modified by changing the value associated with each variable and by adding and deleting variables. The recipe may then be sent and/or saved.

The size of the file that contains a Recipe's configuration is limited to 32 KB.
