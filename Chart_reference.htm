Configuring the Reference Series

This topic is for the features configured by the Chart's Reference Series tab. Show picture

Programmable ranges

The Reference Series tab allows the configuration of two additional series that can be used as references. The properties that are available are a sub-set of those available in the Series tab.

Unlike a normal series, a reference must be populated programmatically using either SCADA Basic or VBA to provide the co-ordinates. Each set of co-ordinates comprises of an X value, and a minimum Y value and a maximum Y value. This is used to draw an area in the Spline Area style, similar to a normal series, but with both upper and lower bounds defined. Show picture

The advanced property, Sort series according to X-axis, has no meaning in this context,

Programmable lines

The Programmable Lines are similar to Programmable Ranges except that you only enter start and end co-ordinates and a straight line is drawn. Show picture

The advanced properties, Sort series according to X-axis, Spline coef, and Transparency have no meaning in this context,

Example of SCADA Basic program to populate a programmable series

Sub MakeRange()
 
 
 
    Dim cWindow as Str, cBranch As Str, cIdentity as Str;
 
    Dim iX as Integer, iYmin As Integer, iYmax As Integer;
 
 
 
'-------- Get context. ID of chart is passed as an argument
 
    cWindow = Getarg("WINDOW");
 
    cBranch = Getarg("WBRANCH");
 
    cIdentity = Getarg("ARG1");
 
 
 
'-------- Clear any existing range
 
    Chart("RANGE_CLEAR", cWindow, cBranch, cIdentity, 1);
 
    Chart("RANGE_CLEAR", cWindow, cBranch, cIdentity, 2);
 
    
 
'-------- Populate range 1 and 2
 
    iX = 0; iYmin = 0; iYmax = 100;
 
    Chart("RANGE_ADDPOINT", cWindow, cBranch, cIdentity, 1, iX, iYmin, iYmax);
 
    Chart("RANGE_ADDPOINT", cWindow, cBranch, cIdentity, 2, iX, iYmin - 40, iYmax + 40);
 
    iX = 100; iYmin = 300; iYmax = 400;
 
    Chart("RANGE_ADDPOINT", cWindow, cBranch, cIdentity, 1, iX, iYmin, iYmax);
 
    Chart("RANGE_ADDPOINT", cWindow, cBranch, cIdentity, 2, iX, iYmin - 40, iYmax + 40);
 
    iX = 200; iYmin = 400; iYmax = 500;
 
    Chart("RANGE_ADDPOINT", cWindow, cBranch, cIdentity, 1, iX, iYmin, iYmax);
 
    Chart("RANGE_ADDPOINT", cWindow, cBranch, cIdentity, 2, iX, iYmin - 40, iYmax + 40);
 
 
 
End Sub
