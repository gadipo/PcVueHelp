How to configure Domains and Natures

 PcVue offers the possibility to associate metadata to configuration elements in your application. Domains and Natures are 2 of these metadata.

Domains and Natures are general-purpose categories or grouping criteria. A domain can represent a geographical or physical area within the process being monitored, and a nature can be an indication of the type of information. In practice, a domain and nature can be used to represent any logical group.

Each variable can be assigned to a domain and/or a nature. The domain and nature of a variable can be used in a number of ways:

  • To select which alarms are displayed in an Alarm Viewer.

  • To filter data that is being recorded as part of the archives or sent to a printer.

  • To select which data is displayed in a Log Viewer.

  • To select which variables are displayed in the Variable selector dialog.

The mechanism for configuring a domain or a nature is the same. This explanation uses domain configuration as an example:

  1. Open the Application Explorer and go to Domains under the General node.

  2. Select the Domains node and click Add a domain from the toolbar. A dialog to configure the new domain opens. Show picture

    You can configure up to 512 domains and 512 natures.

  3. Enter the name of a new domain and an optional description. Domain names may contain up to 100 alphanumeric characters.

  4. Click Apply to save the changes, then OK to close the dialog box.
