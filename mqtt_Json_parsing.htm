Json parsing reference

Json parsing in PcVue relies on the Expression engine and its JsonGetValue() function.
This topic includes typical examples of Jsonpayloads parsing useful in the context of the MQTT Add-on.

Refer to the Expression Engine book in the Application Explorer help for more information about the Expression engine and the JsonGetValue() function.

Pay attention to the fact that the Json syntax is case sensitive.

Example 1 - Basic parsing and filtering

For this example, let's assume we receive the following Json stream on a topic:

{
 
    "sensorID":"A8F5L9B3Z5",
 
    "location":"Room7",
 
    "floor":1,
 
    "installed":2016,
 
    "batterylevel":37,
 
    "connected":true,
 
    "metrics":[
 
    {
 
    "value":39,
 
    "timestamp":"2008-04-22T06:12:40Z",
 
    "quality":1
 
    }
 
 ]
 
}

How to extract the value of location without any filtering?

  • Filtering expression: None
  • Extraction expression: JsonGetValue(value,'location') -> "Room7"

How to extract the value of value in the array called metrics if sensorID is equal to A8F5L9B3Z5?

  • Filtering expression: StrEqualNoCase(JsonGetValue(value,'sensorID'),A8F5L9B3Z5)
  • Extraction expression: JsonGetValue(value,'metrics[0].value') -> 39

Example 2 - Advanced parsing in Json arrays

Json streams can contain fixed-size arrays, but they can also contain a varying number of values in dynamic size arrays. The Json parser and expression engine allows you to retrieve elements from either a predefined fixed index in the array and from a dynamic index if necessary. This feature can also be used for filtering.

For this example, let's assume we receive the following Json stream on a topic:

{
 
    "sensorID":"A8F5L9B3Z5",
 
    "location":"Room7",
 
    "floor":1,
 
    "installed":2016,
 
    "batterylevel":37,
 
    "connected":true,
 
    "metrics":[
 
    {
 
    "value":26,
 
    "timestamp":"2020-01-13T08:10:00.123Z",
 
    "quality":0
 
    },
 
    {
 
    "value":27,
 
    "timestamp":"2020-01-13T08:20:00.123Z",
 
    "quality":0
 
    },
 
    {
 
    "value":28,
 
    "timestamp":"2020-01-13T08:30:00.123Z",
 
    "quality":0
 
    }
 
   ]
 
}

You can retrieve a specific array element by specifying its index, for example the second one:

[Extractors]
 
ExtractValue = JsonGetValue(value, 'metrics[1].value')
 
[Timestamps]
 
ExtractTimestamp = JsonGetValue(value, 'metrics[1].timestamp')
 
[Qualities]
 
ExtractQuality = JsonGetValue(value, 'metrics[1].quality')

The variable will be set with the 2nd element of the array:

  • Value = 27
  • Timestamp = 13/01/2020 08:20:00.123
  • Quality = 0

You can retrieve all elements of the array by specifying the index X:

[Extractors]
 
ExtractValue = JsonGetValue(value, 'metrics[X].value')
 
[Timestamps]
 
ExtractTimestamp = JsonGetValue(value, 'metrics[X].timestamp')
 
[Qualities]
 
ExtractQuality = JsonGetValue(value, 'metrics[X].quality')

The variable will be set with all values received in the array.

Please note that for any given variable mapping, the choice of a fixed index or dynamic one must match across the parameters you define.

This combination of parameters is correct and will work properly:

[Extractors]
 
ExtractValue = JsonGetValue(value, 'metrics[1].value')
 
[Timestamps]
 
ExtractTimestamp = JsonGetValue(value, 'metrics[1].timestamp')
 
[Qualities]
 
ExtractQuality = JsonGetValue(value, 'metrics[1].quality')

These combinations of parameters are inconsistent and will not work properly:

[Extractors]
 
ExtractValue = JsonGetValue(value, 'metrics[X].value')
 
[Timestamps]
 
ExtractTimestamp = JsonGetValue(value, 'metrics[1].timestamp')
 
[Qualities]
 
ExtractQuality = JsonGetValue(value, 'metrics[1].quality')

or

[Extractors]
 
ExtractValue = JsonGetValue(value, 'metrics[1].value')
 
[Timestamps]
 
ExtractTimestamp = JsonGetValue(value, 'metrics[X].timestamp')
 
[Qualities]
 
ExtractQuality = JsonGetValue(value, 'metrics[1].quality')
