Yokogawa Ip

This driver supports communication over Ip with Yokogawa PLCS, including the FA-M3 series.

Communication is via the Ethernet interface module (F3SP05, F3SP21, F3SP25 and F3SP35).

Supported functions

The functions implemented by the driver are:

  • Read/ Write multiple bits (1 to 256 bits)
  • Read/ Write multiple words (1 to 64 words)

Prerequisites

To establish the connection, you have to configure the Ethernet module using switch settings before configuring an equipment.

Switch Description Position
01 Data code (binary) ON
02 Write protect (conducts writing from the remote node when the PC CPU is running) OFF
03 – 06 Always set OFF OFF
07 TCP connection on timeout error (Closes the line on timeout error) OFF
08 Loopback test OFF

To set the equipment IP address, configure the hexadecimal switches. For example:

  IP Address
Hexadecimal 85 8C 0B 39
Decimal 133 140 11 57

Addressing for X and Y frames

The slot number is composed of a unit number and a slot location: imm where:

  • i : unit number = 0 : main unit ; = 1 to 7 : sub-units
  • mm : slot location (01-13) (from the right end of the power supply module to the right end of the base module)
  • Starting address = Terminal number (1-64)

Example:

To access the Terminal number 12 of the X device on the main unit:

  • Slot number = 02 (i=0, mm=2)
  • Starting address = 12

Addressing for W and L frames

FA-link module number = FA link module number (0-7)

Starting address = link relay Number (1-1024)

Example:

To access the FA-link module 1 at the address 12:

  • FA-link module number = 1
  • Starting address = 12
