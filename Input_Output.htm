Input-Output Instructions

Instruction

Action
EMAIL Manages simple email functions to send messages, with or without attachments.

FTP

Copies a file to or from an FTP server.

FTPS

Copies a file to or from an FTP server using FTPS.

HARDCOPY

Initiates a hard copy of the desktop client screen on the Windows default printer or clears its print jobs.

KEY

Program the keyboard function keys on the desktop client.

LPRINT

Send a message to a printer.

MULTIMEDIA

Provides access to the Window's Media Control Interface for the desktop client.

PRINTER

Manage the operation of a printer.
SQL_COMMAND

Sends Sql commands using a pre-configured Sql connection.
SQL_CONNECTION Manage PcVue runtime objects used to handle Sql connections.
SQL_QUERY Used to execute a specific Sql Query (read or write)

SVKEY

Read the contents of the protection key on multi-station systems.

SVSQL
