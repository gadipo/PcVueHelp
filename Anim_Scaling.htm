Scaling

When using the scaling animation the value of a register is represented by changing the size of a drawing. You can scale a drawing in both the X and Y axes. Scaling may be applied to all drawing elements, although there are some limitations when using it with text. Show picture

The scaling for the drawing is defined by specifying minimum and maximum values for the register together with corresponding scale ratios. You can specify different scaling for the X and Y dimensions if required. At runtime, the drawing is scaled about the anchor point.

When scaling text the property Do Not Auto Size must be selected.
