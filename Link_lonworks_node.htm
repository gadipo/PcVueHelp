Linking a Variable to a LonWorks Variable

Before you can link one of PcVue's variables to a LonWorks variable the LonWorks network and one or more nodes must have been added to the LonWorks folder in Data acquisition. For further information, see the help book on Data Acquisition.

LonWorks variables are frequently (but not always) structured. That is the same variable can contain a number of different fields. For example, a variable of type SNVT_switch contains two fields, value and state.

How to link one of PcVue's variables to a LonWorks variable

  1. Display the variable's Source tab.
    1. In the Application Explorer expand the variables tree and select the variable.
    2. Select the Properties task. The Variable Properties dialog opens.
    3. Select the Source tab.
  2. Select the LNS network, Node, Variable and Field.
    1. Using the Source list box select a LonWorks network. A LonWorks network is referenced using the Alias that was entered when it was configured in PcVue. Once you have selected a LonWorks network as the source the LNS tab appears in the dialog. Show picture
    2. Using the Node list box select the LonWorks node. After you select the node there will be a brief pause whilst PcVue interrogates the LonDB for a list of variables available for that node.
    3. Select the LonWorks variable using the Variable drop down list box. When it has been selected, the type of variable is displayed in the read only field, Type.
    4. If you have selected a structured variable, use the Field drop down list box to select the field to which you are linking. Alternatively, you can use an expression as the Field property. See the topic How to use an expression with a structured LonWorks variable for details.
  3. Select OK to confirm the configuration and close the dialog.
