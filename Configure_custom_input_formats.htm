How to Configure Custom Input Formats

Custom input formats may be used when a user enters the value of a variable. To enter a custom input format use the following procedure.

  1. Apply a Send - register animation to a text drawing element. The drawing element dialog is displayed with a new Send - register tab.
  2. Select or enter the name of the register to be displayed.
  3. Select the down arrow button next to the Format field.
  4. Select the Custom option. The Custom Display Format dialog is displayed.
  5. Enter the output format and select the received input units. You can only use custom input formats when using time and date formatting.
  6. Select the Use option button and enter the required input format. You can test the format by entering a relevant value in the Sample field.
  7. Click the OK button to confirm the format.
