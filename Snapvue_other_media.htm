Sending and Receiving Other Media

In addition to sending plain text messages, you can also send attachments in the form of a file, or a voice message. Note that you can send either a text message, an attachment or a voice message. You cannot send, for example, an attachment and accompany it with a text message. To do that you would have to send two separate messages.

Sending attachments

Attachment are sent using the button at the lower left corner of the chat view. The button opens a browse view from where the attachment is selected. After it has been selected, and the browse window closed, the file is sent using the Send button. Show picture

Received attachments are displayed in the chats view in a similar manner to a text message. If the attachment it is a recognized image format, tapping it will display it, otherwise it will be downloaded.

Sending and receiving voice messages

Voice messages are sent using the button at the lower right corner of the chat view. The button opens a voice recorder widget from where the user records the message. After it has been recorded, and the voice recorder widget closed, the file is sent using the Send button. Show picture

Received voice messages are displayed in the chat view in a similar manner to a text message. Tapping a voice message will play it through the device's audio channel.
