Advanced Image Properties

The following advanced image properties are all found in the Picture tab of the image dialog.

To illustrate this topic, the author has used an image of the color palette, not the color palette itself.

EMF+ rendering

The EMF+ rendering property is available for WMF and EMF image formats only.

Selecting EMF+ rendering instructs the HMI to use the EMF+ format for rendering the image. The main effect of this is to support image anti-aliasing. Show picture

Background blend

The Background blend property is available for PNG image format only.

Selecting the Background blend property causes the HMI to blend (add) the image RGB values to those of the background. For example, if the image color was blue (0, 0, 255) and the background color was green (0, 255, 0) the color rendered would be cyan (0, 255, 255). Show picture

The color swab adjacent to the Background blend property in the Picture tab is only to preview the effect that any particular background color will have on the blended image. It has no effect on the image rendered in the mimic.

Opacity

The Opacity property is available for PNG image format only.

Changing the Opacity property allows the background color to be seen through the image. The range of values for Opacity is 0 (totally transparent) to 255 (opaque and the default). Show picture
