Starting the WebScheduler

The WebScheduler can be started either from the Start Menu shortcut created by the PcVue installation, or by typing in the Url into a Web Browser.

The following syntaxes are used for the Url.

  • https://<Host computer name>/WebScheduler - Where <Host computer name> is the name of the PC hosting IIS.
  • https://<Host computer IP>/WebScheduler - Where <Host computer IP> is the IP address of the PC hosting IIS.
  • https://LocalHost/WebScheduler - When accessing the WebScheduler on the same PC on which IIS is hosted.
