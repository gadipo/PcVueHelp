The Drawing Context Menu

The drawing context menu gives quick access to the most frequently used commands and to a drawing element Properties dialog. The menu is displayed by clicking anywhere in a selected drawing element (or group of elements) with the right mouse button. The window in which the drawing element resides must be in Edit mode.

 

Cut the drawing element and place it on the Windows Clipboard.

Copy the drawing element to the Windows Clipboard.

Delete the drawing element.

Create a drawing group from the selected elements.

Restore a drawing group to its individual elements.

Lock the drawing element.

Unlock the drawing element.

Arrange the selected drawing elements.

Only applicable to objects in 3D windows.

Create a symbol using the selected drawing elements.

Enable the drawing element so that it may be used as a VBA control.

Enable use of a VBA script inside a symbol.

Display the scripting language editor.

Animate the selected drawing element.

List the selected object's properties.

Display the Drawing Properties dialog.

Import/export the texts of a symbol. See the Help topic How to Import/Export a File's Texts, in the Text Management book.
