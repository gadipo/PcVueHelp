Loop Through Grid control Example

If any cells are selected, this script loops through them so as to display each cell in turn.

Private Sub Shape1_Click()

If AIGrid1.FirstSelectedCell Then

    MsgBox AIGrid1.SelectedCellRow & AIGrid1.SelectedCellCol

    While AIGrid1.NextSelectedCell

        MsgBox AIGrid1.SelectedCellRow & AIGrid1.SelectedCellCol

    Wend

Else

    MsgBox "No selection"

End If

End Sub
