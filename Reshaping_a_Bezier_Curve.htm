Reshaping a Bézier Curve

The nodes, which define the shape of a Bézier curve, may be moved in the same way as for a polyline or polygon. In addition, each node has a tangent and associated handle which may be used to reshape the curve. The tangents are anchored at one end to each node and are parallel to the curve where they join it. By moving the tangent handles, you adjust the angle at which the curve intersects with each node and its radius.
