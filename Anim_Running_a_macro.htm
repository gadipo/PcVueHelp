Running a Macro

The Macro animation allows up to 5 instructions from the SCADA Basic language to be executed directly from a control zone without having to create a program. Show picture

Instructions

The use of spaces within the instructions of a script is not allowed.

The use of nested function calls is not allowed.

Using a macro to emulate a function key

The Macro animation supports a special instruction that may be used to produce the same action as one of the standard function keys. For example, you could create a control zone that allows you to display the Report Toolbox (using F12). For a list of the standard function keys, see the topic Key Codes in the Reference Topics book of the SCADA Basic help.

STANDARDACTION,KeyNum

where KeyNum is the function key number.
