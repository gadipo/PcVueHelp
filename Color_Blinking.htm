Selecting a Blinking Color

Some drawing elements support blinking colors. A blinking color contains two color selections. At Run time, the affected part of the drawing element will alternate between them at a pre-defined rate. Show picture

How to select a blinking color

  1. Select the Fill Mode as Blinking.
  2. Click the Primary Color and use the Color Picker dialog to select it.
  3. Click the Alternate Color and use the Color Picker dialog to select it.
  4. Select which of the blinking periods to use. There are 16 configurable blink rates.
  5. Click OK to apply the color.

The blinking rates are configured from the main menu: Show picture

  1. Select Configure.Preferences.Blinking to open the Blinking Color and Visibility dialog. The colored panel blinks to show the rate of blinking.
  2. Move each slider to set the period for which its color is to be displayed.
  3. Click OK to apply the settings.

In the Blinking Color and Visibility dialog, the colors that show the blink rates do not affect the colors used in an animation.
