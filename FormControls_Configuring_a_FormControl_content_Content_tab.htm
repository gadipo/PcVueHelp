Configuring a Form Control's Contents

The list of items that you see in a Form Control at run-time is configured using the Content tab. This provides the option of storing the list contents locally (as part of the form control's configuration in the mimic file) or in a named text file.

  • The Content tabs of the Check-box list, Option-button list, Combo-box and List box are almost identical and are dealt with here in a single Help topic.
  • The Tree-view and Text-box are different and have dedicated topics.
