Selecting a Gradient Color

A gradient color contains two or more colors. The affected part of a drawing element will be shaded starting with one color gradually changing to the other colors across the drawing. The type of gradient may be selected from either spot or linear. The following shows some examples of the use of gradient colors.

How to select a color with a spot gradient

  1. Select the Fill Mode as gradient.
  2. Select the Type as spot.
  3. Click the one of the small color boxes on the ends of the gradient control bar above the color circle and select its color using the Color Picker dialog in the normal way.
  4. Repeat for the other color box.
  5. Select the position at which the color shading starts by dragging the color boxes along the gradient control.
  6. Select the center for the color spot by entering X and Y co-ordinates. These are entered as 0% to 100% with 0, 0 representing the top left of the drawing when it is filled.
  7. Click OK to apply the color.

How to select a color with a linear gradient

  1. Select the Fill mode as Gradient.
  2. Select the Type as linear.
  3. Click the one of the small color boxes on the gradient control above the color circle and select its color using the Color Picker dialog in the normal way.
  4. Repeat for the other color box.
  5. Select the position at which the color shading starts by dragging the color boxes along the gradient control.
  6. Select the angle for the color gradient.
  7. Click OK to apply the color.

How to add additional colors when using a gradient

By default, a gradient uses two colors. However, if necessary, it is possible to add additional colors up to a maximum of eight.

  1. Select the Fill mode as gradient.
  2. Click and drag the new color box icon to the left of the gradient control onto the gradient control.
  3. Select the color for the new color box in the normal way.
  4. Select the position at which the color shading starts by dragging the color box along the gradient control.

How to remove additional colors

  1. Click the color box to be removed and drag it over the new color box icon.

Performance Considerations

The use of gradient colors is very attractive but, when using them, it is important to note that the speed at which they are rendered will depend on the choice of graphic adapter and the color configuration.

  • Step -Make the Step parameter are large as possible. The drawing performance depends directly on the size of the Step. For example, a step of 4 will render twice as fast as a step of 2.
  • Linear gradient - T the angles 0, 90, 180 or 270 are most efficient.
  • Radial gradient - M more time consuming than linear, particularly if the X and Y co-ordinates are not close to the object's center (50%, 50%)
  • Number of colors - This has no impact on the drawing performance.
