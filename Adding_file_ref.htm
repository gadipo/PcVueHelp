Adding a file reference to a template

This topic explains how to add a File reference element to a template using the Application Architect.

For an explanation of the use of File references, see the help for the Application Explorer.

How to add a File reference to a template

  1. Navigate to a template's Configuration elements folder.
    1. In the right pane of the Application Architect, select the Templates tab.
    2. Using the templates configuration tree select the template to which the file reference will be added.
    3. Using the secondary navigation tree select the Configuration elements folder.
  2. Add and configure the File reference.
    1. Select Add a file reference. A new file reference with a default name is created.
    2. Edit the name to one appropriate to your application. You can only edit the name directly by clicking on it in the configuration tree.
    3. Select the File type and configure the file reference's properties. Show picture
