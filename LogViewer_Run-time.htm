Configuring the Log Viewer's Run-time Behavior

The run-time behavior is selected using the following properties found in the Execution tab. Show picture



  • Comment Modification - If selected, the User is allowed to add a comment of up to 40 characters to each log entry.
  • Filtering - If selected the User can display the Log List dialog and change the log list.
  • List Choice - If selected the User can select the log list that the viewer is attached to.
  • Size of Columns - If selected, the User can change the width of columns at run-time by clicking and dragging with on the header.
  • Toolbar Position - If selected the User can change the toolbar position at run-time.
  • Toolbar Modification - If selected, the User can change the toolbar configuration at run-time.
  • Context - If selected, when the User clicks on a log line at run time the context window for Log Viewers is opened.
  • Enable sorting - If selected, the list of events is sorted according to the contents of a particular column.
  • Sort modification - If selected the User can change the sort column and direction.
  • Restrict navigation to selected time range - if selected, the User can select a time range with the Calendar tool, but will not be prevented from navigating outside this time frame when using scroll bars (in particular the page up and page down buttons).
  • Printing - If selected, the User can print selected log information, on the specified printer, directly from the Log Viewer. The print format can be customized using the Format property and substitution strings.
  • Display Buffer - The maximum number of lines that can be recorded in the display buffer. The display buffer has a maximum size of 6 MB which should be taken into account when configuring this property. One character is one byte.

If you want to change the column width or modify the toolbar, but do not want the user to do so, enable the properties and make the changes at run-time and then disable the properties again. The changes to the display will be saved.

Changing the way that the list is sorted

To permanently sort the Log Viewer by a particular column and direction:

  1. In design mode, tick both the Enable Sorting and Sort modification properties.
  2. Switch to run mode and click the header of the column to be used for sorting. Clicking more than once will toggle between ascending and descending.
  3. Switch back to design mode and un-tick Sort modification.
  4. Save the changes.

To allow the User to change the column and direction for the sort:

  1. In design mode, tick both the Enable Sorting and Sort modification properties.
  2. Save the changes.
