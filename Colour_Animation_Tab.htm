The 3D Color Animation Tab

Each type of 3D Color animation tab is very similar to those used in 2D animation, the main difference being the addition of a combo-box to select the materials to which the animation is applied, and a list box to select which color properties are shown in the tab.

How to use the Materials combo-box

When configuring a color animation you can select one or more materials. The Materials combo-box has a non-standard behavior to allow multi-selection capability. Show picture

  • The combo-box displays all selected materials with each material separated by a semicolon.
  • The drop-down list is not automatically closed when you click on an item. Instead, the item's selection state is reversed. In that way, you can select (or de-select) several items before closing the list.
  • To close the drop-down list, click once outside of it.

The materials available for selection depend on the level at which the color animation is being applied.

Level

Materials available for selection

Symbol or group

No individual materials are available for individual selection. The combo-box contains one item, All Child Object Materials, which allows the animation to be applied to all materials in all models.

Model

All materials available in the model.

Frame without a mesh

No individual materials available for individual selection. The combo-box contains one item, All Child Object Materials, which allows the animation to be applied to all materials in all sub-frames.

Frame with a mesh

All materials available in the frame, plus an additional item, All Child Object Materials, which allows the animation to be applied to all materials in all sub-frames.

How to select the color properties displayed in the tab

A single color animation can change all properties provided by a material. In order to keep the size of the Color Animation tab manageable, the properties displayed at any one time are selected by a list box. Using the list box, you can choose to display:

  • Diffuse color and opacity
  • Specular color and level
  • Emissive color

As a design aid, once you have selected and changed properties in any one of the categories, its entry in the list box is displayed in bold text. Show picture
