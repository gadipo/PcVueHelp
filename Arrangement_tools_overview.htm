Overview of the 3D Arrangement Tools

This book describes the tools in the Arrange menu and toolbar that you can use on 3D models.

The arrange tools only become active when using an orthographic camera view and when least one model is selected (for some tools, multiple models must be selected). An orthographic camera view is one where the camera is parallel to one of the world axis (top, bottom, front, rear, left or right view).

For details of how to use the arrangement tools in general, see the book 'The Arrange Toolbar' in the main Help for the HMI.

You cannot use the following arrangement tools with 3D objects.

  • Flip tools (horizontally or vertically).
  • Boolean operator tools (union, intersection and extraction of shapes; plus reversing the order of polygon points).
