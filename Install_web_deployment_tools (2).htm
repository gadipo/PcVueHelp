Installing the Web Deployment Tools

 PcVue web deployment tools are required on the computer that has the role of web server, where the IIS Web server is installed. Depending on the chosen architecture, this can be on the same or different computer to PcVue itself. The main component of the web deployment tools is the Web Deployment Console.

When either the PcVue installation, or the Web Deployment tools installation is started, it first checks for the presence of the prerequisites. See the topic Installing PcVue.

A wizard guides you through the installation process, prompting for information where necessary. At each step, after configuring any options click the OK or Next button to proceed. Clicking the Cancel button at any time will abort the installation.

  1. Welcome to the InstallShield Wizard - This is the first step of the installation itself. Click Next to continue. Show picture
  2. License agreement - Before you can continue with the installation you should read the License Agreement and select I accept... or I do not accept... as appropriate. Click Next to continue only if you accept the agreement. Show picture
  3. Destination Folder - Suggests a destination folder for the installation. To install the software elsewhere, use the Change button and select another path. Show picture
  4. Ready to Install the Program - Click Install to proceed with the installation. Show picture
  5. Installing status: A progress bar indicates progress with installing each component, then a dialog confirms that the installation is complete. Click on Finish.
