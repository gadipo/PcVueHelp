StartExport Method

       Applies to

Starts exporting records from the database.

Syntax

object.StartExport (StartTime As Date, EndTime As Date, [Path As String], [FileNameExtension As String = "dat"], [Format As HDSMgtExportFormat = hdsmgtExportFormatText], [ColumnHeaders As Boolean = True], [Delimiter As String = ";"], [NullValueSubstitution As String])

Part

Description

StartTime

The earliest date-time of records eligible for export.

EndTime

The latest date-time of records eligible for export.

Path

Path for export files.

Format

Export format - values:

  Text: hdsmgtExportFormatText   

  ADTG: hdsmgtExportFormatADTG

  XML: hdsmgtExportFormatXML

For text files:

  FileNameExtension

Suffix for database file names.

  ColumnHeaders

Whether to include column headers.

  Delimiter

 

  NullValueSubstitution

The substitution string to be used for a null value.
