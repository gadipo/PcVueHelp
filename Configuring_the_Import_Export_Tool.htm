Configuring the Import/Export Tool

Before any import or export takes place, you must configure the path for the export/import files and the languages that are to be used.

If you use the Import/Export Tool on a single item (mimic, mimic template or symbol), the dialog is not displayed and the process will use the settings you make here.

If you use the Import/Export Tool on an entire library, the dialog is displayed and you can change the settings at the time of use.

How to configure the import/export file paths

  1. Open the Translation Import/Export Tool from the command Display.Translation import/export on the menu. Show picture

  2. Select or enter a file name for each of the types of file that you wish to use.
  3. The tool to the right of each file name field opens a standard Windows Explorer dialog to aid file selection. If no path is specified, the location of the files defaults to the project's TP folder.
  4. Enable each of the types of file (mimic, mimic template and symbol) that are required by ticking the appropriate tick box.

  5. Using the drop-down list boxes, select the base and alternative languages.

  6. Tick the box Display Confirmation Message... if you want a message to appear when the process ends.
  7. Click the Save button to save the configuration.

You can use the same file for all three import/export types (mimics, mimic templates and symbols) if required.

If you do not require the Import/Export Tool to operate for either the base language or the alternative language you can select the entry NONE at the top of the language drop-down list.

Processing status and message

Each import/export process displays a scrolling list of its actions in the Status pane. You can also request a message to confirm its completion.

  • Tick the option Display confirmation message... to display a message when the process is complete.
