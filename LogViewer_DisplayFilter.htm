Configuring the Log Viewer Filter

The Filter tab is used to select the source and filter the logged data that will appear in the list. Show picture



The source of events for a Log Display is a single Log List, selected using the List name drop down list box. When you select a Log List its properties are automatically copied into the filter properties of the Log Display. You can then edit the properties to select a sub-set of the information as required. For example, you can select a particular Domain or Nature or de-select one or more of the event types.

Configuring the Log Viewer filter when using the Only Log Alarms On/Off Property

If you select the property, Only log alarm on/off changes in log filters, from Configure.Project.Settings.Archives the operation and appearance of the filter tab is modified. Show picture
