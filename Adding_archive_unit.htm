Adding an Archive Unit to a Template

This topic explains how to add an archive unit to a template using the Application Architect. There are five types of archive unit.

  • Database (HDS)
  • Proprietary
  • ODBC
  • Free
  • VCR

For an explanation of their use, see the help for the Application Explorer.

How to add an archive unit to a template

  1. Navigate to a template's Configuration elements folder.
    1. In the right pane of the Application Architect, select the Templates tab.
    2. Using the templates configuration tree select the template to which the associated label will be added.
    3. Using the secondary navigation tree select the Configuration elements folder.
  2. Add and configure the Archive Unit.
    1. Select Add an archive unit and then the archive unit type. A new archive unit with a default name is created.
    2. Edit the name to one appropriate to your application. You can only edit the name directly by clicking on it in the configuration tree.
    3. Configure the archive unit's properties. The properties that are available will depend on the type of archive unit. Show picture
