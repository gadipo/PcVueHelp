IEC 61850 status variables

In the following system variable names:

  • <Network> is the name of a configured IEC 61850 network,
  • <Device> is the name of a configured IEC 61850 device,
  • <DataGroup> is the name of a configured IEC 61850 dataset, data report, data group object or setting group.

SYSTEM.61850.XON

R

Bit

Manual

Flow regulation status.
0 Flow regulation activated. Loss of exchanges possible.
1 Flow regulation not activated.

SYSTEM.61850.LOSTTRANS

R

Register

Automatic

Counter for the number of lost IEC 61850 exchanges when flow regulation was last activated.

SYSTEM.61850.<Network>.ON

R

Bit

Automatic

Network activation indication where <Network> is the network name.
1 The network is activated.
0 The network is not activated.

SYSTEM.61850.<Network>.STATUS

R

Register

Automatic

Network status where <Network> is the network name.

The table below describes possible values.

SYSTEM.61850.<Network>.AVAILABLE

R

Bit

Automatic

Network availability where <Network> is the network name.

0 Healthy
1 Failure - See the STATUS variable for more information.

SYSTEM.61850.<Network>.<Device>.ON

R

Bit

Automatic

Device activation indication where <Network> is the network name and <Device> is the device name.
1 The device is activated.
0 The device is not activated.

SYSTEM.61850.<Network>.<Device>.STATUS

R

Register

Automatic

Device status indication where <Network> is the network name and <Device> is the device name.

The table below describes possible values.

SYSTEM.61850.<Network>.<Device>.AVAILABLE

R

Bit

Automatic

Device availability indication where <Network> is the network name and <Device> is the device name.

0 Healthy
1 Failure - See the STATUS variable for more information.

SYSTEM.61850.<Network>.<Device>.<DataGroup>.ON

R

Bit

Automatic

Datagroup activation indication where <Network> is the network name, <Device> is the device name and <DataGroup> is the datagroup name.
1 The datagroup is activated.
0 The datagroup is not activated.

SYSTEM.61850.<Network>.<Device>.<DataGroup>.STATUS

R

Register

Automatic

Datagroup status indication where <Network> is the network name, <Device> is the device name and <DataGroup> is the datagroup name.

The table below describes possible values.

<DataGroup> here represents a dataset, a data report, a data group, or a setting group.
