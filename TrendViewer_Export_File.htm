Export Wizard Step 6 - Export to File

In the final step of the Data Export Wizard, you select the export file type as either an Excel Workbook (.xlsx) or a Comma Separated Variable (.csv). If you choose CSV, you can also select the delimiter used for columns. Show picture

The folder selector is disabled for users without Desktop right permissions. To ensure secure operations, configure user accounts with appropriate rights.

Clicking Finish will open the standard Windows' Save As dialog from where you can change or confirm the folder and file name. By default, the TP folder of the current project is selected. The file name is calculated using the following formula, plus a file extension appropriate to the file type.

<Trend Viewer ID>_<Data Type>_<Year><Month><Hour><Minute>

As a final step you are given the opportunity to open the file with the application associated with the selected file extension. Show picture
