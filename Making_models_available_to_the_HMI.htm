Making 3D Models Available to the HMI

When the HMI is required to insert a 3D model into a 3D mimic, it searches for the file by name in the 3D folder.

To make a particular model available:

  1. Copy it, or the set of models that includes it, from the project's folder \Shared Libraries\3DModels (or an external source) to the 3D folder of the project.
  2. Restart the HMI to make the 3D models available in the HMI.

If you are starting from scratch, or from an understanding of 2D techniques, please refer to the 3D Mimics chapter.
