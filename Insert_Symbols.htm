How to insert a Symbol into a Window

Before you can insert a symbol into a window, the window must be in Design mode. There are then a number of methods by which a Symbol may be inserted into a window.

  • Using the Insert.Symbol command.
  • Using the Insert.Symbols from instance command.
  • By dragging and dropping the symbol from another window.
  • By using the standard Windows commands for Copy (Ctrl+C) and Paste (Ctrl+V) to copy the symbol either to the same window or a different window.
  • By using the Copy tool from the Standard toolbar to copy the symbol either to the same window or to a different window.
  • By dragging and dropping it from the Library Browser.

How to insert a symbol using the Insert.Symbol command

  1. Select the Insert.Symbol command. A box containing a list of Symbols, referenced by name, will be displayed. A lexical filter may be applied to reduce the number of symbols displayed in the list.
  2. If using multiple Libraries select the relevant Library.
  3. Select the symbol name from the scrollable list. A visual preview of the symbol is displayed adjacent to the list.
  4. If the symbol uses relative variables then enter a valid branch.
  5. Insert the symbol into the window by selecting the OK button.

How to insert a symbol using the Insert.Symbols from instance command

  1. Select the Insert.Symbols from instance command. A dialog box containing the list of instances and the symbols they contain will be displayed. A lexical filter may be applied to reduce the number of instances displayed.
  2. In the left pane, select the instance that contains the symbol you want to insert.
  3. In the right pane, select the symbol by ticking its name from the list and click the Insert button. Double clicking a symbol in the visual preview also inserts it in the current window. The branch corresponding to the instance is added automatically. By using multi-selection, you can insert multiple symbols.
  4. Repeat steps 2 and 3 if you need to insert other symbols.
  5. Click the Close button to close the Instances selector.

The toolbar allows you to filter the list of instances displayed:

  • By entering a text-based filter on the instance name (Starts with, Contains, Ends with or a regular expression).
  • By selecting the name of a template, in which case only instances of this template will be displayed.

How to drag and drop a symbol from another window

  1. Using the mouse click on the symbol in the source window with the left mouse button. Hold down the mouse button and drag the symbol to its new window - release the mouse button when it is in the correct position.
  2. When the mouse button is released, a dialog will be displayed showing the names of any variables and allowing a branch to be entered. Enter a branch (if required) and select the OK button. The symbol has now been moved to the new window.

How to copy a symbol from another window

  1. Using the mouse click on the symbol in the source window with the left mouse button. Hold down the mouse button and the Ctrl key on the keyboard. Drag the copy of the symbol to its new window - release the mouse button when it is in the correct position.
  2. When the mouse button is released, a dialog will be displayed showing the names of any variables and allowing a branch to be entered. Enter a branch (if required) and select the OK button. The symbol has now been moved to the new window.

How to paste a symbol using Ctrl+C and Ctrl+V

  1. Using the mouse select the symbol in the source window by clicking with the left mouse button. Use Ctrl+C to copy the symbol.
  2. Move the mouse pointer to destination window and click with the left button to select the window. Use Ctrl+V to paste the symbol into the window.
