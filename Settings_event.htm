Event Actions Settings

The event actions settings are displayed from the Settings task when the Events folder is selected in the Application Explorer's configuration tree. These settings affect the general operation of the event handling within PcVue. Show picture

  • Networking
    • Execute action only on active servers - If selected, event actions produced by a server association are only executed on the active servers of the association.
    • Execute action on both active and passive servers - If selected, event actions produced by a server association are executed on all servers of the association.
