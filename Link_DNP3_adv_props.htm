Advanced Properties Specific to DNP3 Variables

The Advanced tab of the variable's properties dialog contains a number of properties only available when a DNP3 source has been selected.

Additional properties common to all DNP3 variables

  • Time tagging - The method used to generate the timestamp for the variable.
    • Default - Use the default method as configured in the DNP3 general settings.
    • Time tagged - Use the time tag from the device.
    • Not time tagged - Do not use the time tag from the device and use PcVue's timestamp instead.
