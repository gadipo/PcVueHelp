VBA hints and tips

The following hints and tips are based on the feedback we revceived. They do not concern programming techniques but more the way in which you use VBA in PcVue.

Stopping a VBA Program

To stop a VBA program that cannot be stopped in any other way (for example, if it is in an infinite loop) you can press the keys Ctrl+Break.

Enabling events

You must have the Dim WithEvents statement in a code module to enable an object's events to be intercepted by VBA procedures.

Events are fired only if the EnableEvents property of the underlying object is set to True. This happens automatically once the first event script has been associated with the object. For instance, you must set the EnableEvents property of a symbol to True to allow it to respond to events. For more details see the topics Dim WithEvents - Coding VBA Events dynamically and Embedding VBA scripts in symbols.

In Design mode, events are not executed.

The 'Me' keyword

'Me' refers to the current object - the specific instance of the class where the code is executing. For an example, see the Microsoft VBA topics 'Me <keyword>' and 'Invalid use of Me keyword'.

Access to Variables

You must subscribe a variable before you can use it in VBA, unless it is used in an animation in a mimic that is currently open and therefore already subscribed.

The [...] syntax provides an additional shortcut to access a variable directly. For example, [Main.Branch1.B1] is the equivalent of: Variables("Main.Branch1.B1").

Diagnostic tools

The HMI is always open when you are developing a VBA program. You can use animations in a mimic to monitor the values and properties of variables dynamically. You can also keep open the Properties List (for a selected element in the HMI) for reference as you develop the code. The contents of the Properties List change dynamically as you select HMI objects and alter their properties.

For diagnostic tools in the standard Microsoft VBA environment, see the Microsoft Help topic 'Ambiguous Selection' (Code/Module Window, Immediate Window/Pane, Locals Window/Pane, Object Browser and Watch Window/Pane).
