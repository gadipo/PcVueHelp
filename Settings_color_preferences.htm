Color Preferences

The Color Preferences dialog is opened from the menu using the command Configure.Preferences.Colors. Show picture

The Color Preferences dialog contains five tabs that are used to select the colors for certain objects and features in PcVue's HMI. Some of the color preferences are used as is - that is the color defined in the preferences is the color that is used. Others are used as a default that can optionally be changed in each instance of the object. Changing a color in the Color Preferences dialog automatically changes the colors in the HMI's objects where it is used as is.

Each color preference defines a pair of colors, one for the background and the other for the text (or border depending on the use). Non-indexed or indexed colors can be used.

Alarm on text tab

Used to color alarms in the alarm viewer and log viewer, and the text drawing element when animated using color-alarm on text.

  • In an alarm viewer or log viewer, the background color is only used if the Advanced color mode property is selected. The text color is always used to color the text of alarms. The colors are used as is.
  • In a text drawing element, both the background and text colors are used. The colors are used as a default.

Alarm tab

Used as the default colors whenever a shape is animated using a color-alarm animation. For a solid shape, rectangle, polygon, bézier shape ..., both background and border colors are used. For a line, polyline or bézier curve only the border color is used.

Bit tab

Used to color bit transitions in the log viewer, and a shape drawing element when using a color-bit animation.

  • In a log viewer, the text color is always used to color the text of bit transitions. The background color is unused. The colors are used as is.
  • For a solid shape, rectangle, polygon, bézier shape ..., both background and text colors are used. For a line, polyline or bézier curve only the border color is used. The colors are used as a default.

Action tab

Used to color user actions in the log viewer. The colors are used as is. Only the text colors are used.

Thresholds tab

Used to color the cells of the grid, when used in historical mode, when the register values exceed their thresholds. The colors are used as a default.

Navigator

Used to color the borders of the navigator, run-time navigator and mimic navigator.

Trends

Used to color the traces in a trend viewer. The colors are used as a default.

Chart Control

Used to provide the default palette colors for the Pie Chart.
