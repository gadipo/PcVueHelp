Using Variables as Alarm Counters

Certain variable names are recognized as having special significance. These variables are automatically updated to provide information about the system operation that would otherwise be unavailable to the User. Included in this category are a range of register variables that may be used to count and display the number of alarms by state, level, Domain and Nature.

These variables are not created automatically, they must be created as part of the project configuration by the Developer if they are to be used. The variables and the information that they contain are as follows.

Variable names of SYSTEM.<TYPE>.<PRIORITY> format

The value of TYPE determines which alarm state is counted.

Type

Meaning

0 or ACKOFF

Alarm off.

1 or NOACKON

Alarm on and not acknowledged.

2 or NOACKOFF

Alarm off and not acknowledged.

3 or ACKON

Alarm on and acknowledged.

4 or NS

Alarm invalid (not Set).

5 or USERMASK

Alarms masked by the operator.

The value of PRIORITY determines which alarm level is counted. The range is 0 to 29. Alternatively, ALL may be used to count alarms of all priorities.

Variable names of SYSTEM.<TYPE>.<PRIORITY>.<DOM_NAT> format

System alarm counters by type, priority, Domain and Nature.

DOM_NAT is a string that determines which Domain and Nature is to be taken into consideration. The character used to separate the Domain and Nature is normally an underscore but may be changed using the Project.General Operation box. The wild character * may be used to signify all Domains or all Natures.

Do not create both versions of an alarm counter register, for example SYSTEM.3.8 and SYSTEM.ACKON.8.

Examples

SYSTEM.3.8 or SYSTEM.ACKON.8

A register variable that counts all alarms of level 8 that are On and Not Acknowledged.

SYSTEM.1.ALL.ZONE1_ELEC or SYSTEM.NOACKON.ALL.ZONE1_ELEC

A register variable that counts all alarms On and Not A acknowledged in Domain ZONE1 and nature ELEC.

SYSTEM.3.15.*_CONTROL or SYSTEM.ACKON.15.*_CONTROL

A register variable that counts all level 15 alarms with the CONTROL nature that are On and Acknowledged.
