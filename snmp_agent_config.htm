Installation and Configuration

The SNMP Agent Add-on of PcVue comprises of two dll files that are installed in the \Bin folder as part of the installation. The files are named:

  • svmgrSNMPAgent.dll
  • svSNMPAgent.dll

The file named svmgrSNMPAgent.dll was developed using the SV Manager Toolkit. In order for PcVue to load the Add-on at startup, a text file USRMGR.DAT must also be added to the BIN folder. The file must contain the following lines:

[USRMGR\svmgragent]
 
DLL=svmgrSNMPAgent.dll

SNMP Agent Add-on configuration file

The Add-on is configured using a file called SNMPAgent.ini which must be created in the PER folder of the PcVue project. The file has the following structure:

[TaskParameters]
 
AttributeNumber = 8
 
SnapshotPeriod = 400
 
BrowsingLevel = 10
 
NbTrapFormat = 1
 
 
 
[Generic1]
 
Params = Oid#TimeStamp#Description#AlarmLevel#TextAttr03#TextAttr12
  • [TaskParameters]
    • AttributeNumber - A variable's text extended attribute that is used to identify the corresponding variable as being exposed via SNMP. A number from 3 to 16. Defaults to 16.
    • SnapshotPeriod - The period in milliseconds at which the task refreshes the shared memory and hence the variable values that are exposed to SNMP. A number representing the number of milliseconds. Default to 500 (in milliseconds).
    • BrowsingLevel - The variable browsing level that identifies it as being exposed via SNMP. A number from 1 to 29. If omitted a default of 99 is used making no variable eligible based on the browsing level criteria.
    • NbTrapFormat - The number of generic trap formats than can be used in the application. A number. If omitted a default of 0 is used (no generic trap).
    • AssociationName - Send traps only from the active station of the association. If omitted a default of 0 is used (all stations in the association send traps).
  • [GenericX] - Where X is the trap format number.
    • Params - The list of parameters attached to the generic trap. The maximum number of parameters cannot exceed 18. Any parameters above 18 are discarded. See below for a table of the supported parameters, and examples.

Generic trap parameters

Key OID suffix Description
Oid 1 Oid of the variable that triggered the trap
Value 2 Value of the variable that triggered the trap
TimeStamp 3 Timestamp at which the trap occurred
Format dd/mm/yyyy hh:mm:ss.llll
Description 4 Description of the trigger variable*
AlarmLevel 5 Alarm level of the trigger variable
TextAttr03 6 Text attribute 03 of the trigger variable*
TextAttr04 7 Text attribute 04 of the trigger variable*
TextAttr05 8 Text attribute 05 of the trigger variable*
TextAttr06 9 Text attribute 06 of the trigger variable*
TextAttr07 10 Text attribute 07 of the trigger variable*
TextAttr08 11 Text attribute 08 of the trigger variable*
TextAttr09 12 Text attribute 09 of the trigger variable*
TextAttr10 13 Text attribute 10 of the trigger variable*
TextAttr11 14 Text attribute 11 of the trigger variable*
TextAttr12 15 Text attribute 12 of the trigger variable*
TextAttr13 16 Text attribute 13 of the trigger variable*
