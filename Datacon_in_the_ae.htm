Managing Data Connections in the Application Explorer

Tasks applicable to Data Connections

When the Data Connections folder is selected.

Icon Task Description
Add a Sql connection Add a new Sql data connection.
Settings Display the Data connections settings dialog. See the topic Data connections settings for more information.

When a specific data connection is selected.

Icon Task Description
Start Sql connection Start the selected Sql connection
Stop Sql connection Stop the selected Sql connection
Test connection Test the selected Sql connection
Properties Display the selected data connection properties dialog.
Remove Remove the selected data connection.
Add Sql read query Add a Sql read query to the selected data connection.
Add Sql write query Add a Sql write query to the selected data connection.

When a specific Sql query is selected

Icon Task Description
Remove Remove the selected Sql query.
Properties Display the selected Sql query properties dialog.
Test read query (resp. write query) Test the selected Sql read query (resp. write query). This action executes the query and displays the results.
Trigger read query (resp. write query) Trigger the selected Sql read query (resp. write query). This action executes the query and processes the results (update variable values for example).
Trigger read query with parameters (resp. write query) Trigger the selected Sql read query (resp. write query) and propose to manually fill-in the value of the expected parameters. This action executes the query and processes the results (update variable values for example).

Behavior of cut, copy and paste

You can cut and paste or copy and paste a data connection. A new data connection is created with a default name.

You can cut and paste or copy and paste a Sql read or write query. A new Sql query is created with a default name.
