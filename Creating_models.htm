Creating 3D Models

The HMI's 3D mimics feature is an assembly tool kit that creates 3D mimics from existing models. It is not able to create 3D models from scratch. To do so you must use software designed for that purpose, obtain a suitable collection of models or employ a specialist design service.

Many software tool kits are available to create 3D models. They have similar features for modeling and rendering functions, so that basic projects for 3D model creation could be carried out on any of them.

Such toolkits are usually supplemented with a range of utility programs and add-ins. These tend to provide:

  • Format conversions (on sets of files).
  • Special effects.
  • Creation of software add-ins and scripts.
  • Calibration and optimization.

Some have software development kits (SDK) for enhancing the behaviors of 3D objects. There are many resource sites and forums for users of such applications and tools:

  • Instructional resources such as the tutorials indexed in the Digital Art Masters web site (3D Total/3D Studio Max tutorials).
  • Collections of 3D objects (DirectX files) for download.

However, most of these resources are oriented to leisure media development (for games, cinema/video and broadcast) rather than to visualization for industrial and building applications. Those relating to industrial processes tend to focus on conversion of 3D models from computer-aided design (CAD) sources whilst the building automation sources often cite interfaces to geographical information systems (GIS and Building Information Modem (BIM)).

Not all DirectX multimedia files are suitable for inclusion in the HMI of the PcVue software.
